  • Cloud provider metadata fields
  • Common fields
  • DHCPv4 fields
  • DNS fields
  • Docker fields
  • ECS fields
  • Flow Event fields
  • Host fields
  • HTTP fields
  • ICMP fields
  • Jolokia Discovery autodiscover provider fields
  • Kubernetes fields
  • Memcache fields
  • MongoDb fields
  • MySQL fields
  • NFS fields
  • PostgreSQL fields
  • Process fields
  • Raw fields
  • Redis fields
  • SIP fields
  • Thrift-RPC fields
  • Detailed TLS fields
  • Transaction Event fields
  • Measurements (Transactions) fields
