Available Docker imagesedit

Versions of the Elastic Stack, containing Elasticsearch, Kibana, and other products, are available as downloadable Docker images.

The first table contains the stack versions that shipped with the 2.7 version of Elastic Cloud Enterprise. You can also see the most recent stack packs and Docker images, which might have released after the 2.7 version of ECE, as well as the full list of available stack packs and Docker images.

Docker images included with Elastic Cloud Enterprise 2.7.2

docker.elastic.co/cloud-enterprise/elastic-cloud-enterprise:2.7.2

docker.elastic.co/cloud-assets/elasticsearch:6.8.13-0

docker.elastic.co/cloud-assets/kibana:6.8.13-0

docker.elastic.co/cloud-assets/apm:6.8.13-0

docker.elastic.co/cloud-assets/elasticsearch:7.10.1-0

docker.elastic.co/cloud-assets/kibana:7.10.1-0

docker.elastic.co/cloud-assets/apm:7.10.1-0

docker.elastic.co/enterprise-search/enterprise-search:7.10.1

Additional Elastic Stack versions are available as Docker images that you can use with ECE. For offline installations, you need to download both the Elastic Stack pack and the Docker images for the same version.

To learn more about adding the stack pack to ECE, see Manage Elastic Stack Versions.
