Enhancements and bug fixesedit

The following changes are included in this release.

Featuresedit

Show name for external trusts. Added read-only name field to external trusts in deployments API that shows the name of the referenced trust relationship.

Force social login if the organization mandates it. Adds a feature where organizations can mandate all the members to use social login

Enhancementsedit

Use Elasticsearch scoring when querying deployments. Searching deployments will now score and sort results based on the search query.

Added a name field to accounts. There is now a name field for account trust relationships.

Bug fixesedit

Stack upgrade: disallow upgrading to unreleased versions if already on latest major. Fixed a bug where it was possible to upgrade to an unreleased stack version.

Treat security realm warnings on upgrade like all other warnings. Fixed bug preventing users updating their security realm settings during an upgrade.

Fix disabled Enterprise Search not rendering in UI. Due to a bug, it was not possible to re-enable Enterprise Search after it had been disabled. This is now working again.

Usage values now add up to the total value. Fixed a bug where the usage values did not add up to the total value on the Billing Usage page.
