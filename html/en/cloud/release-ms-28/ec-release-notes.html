Release notes

This section summarizes the changes in each Elasticsearch Service release:

  • GCP regions US East 4 (N. Virginia) and Europe West 2 (London) available - October 28, 2019
  • Better plugin and bundle support, updated pricing calculator, bug fixes, and more released - October 23, 2019
  • GCP region Asia Pacific Southeast 1 (Sydney) available - September 10, 2019
  • Elasticsearch Service on Microsoft Azure available - September 03, 2019
  • Cross-cluster search, OIDC and Kerberos authentication support released - August 12, 2019
  • AWS region EU (London) available - August 13, 2019
  • GCP region Asia Pacific Northeast 1 (Tokyo) available - July 30, 2019
  • Console usability improvements and Kibana bug fix released - July 3, 2019
  • Snapshot repository support for GCS and private subscription tier available - June 5, 2019
  • Support for Elastic Stack 6.8, and 7.1, released - May 20, 2019
  • Support for index lifecycle management (ILM) and migration to hot-warm architecture available - April 19, 2019
  • Support for the Elasticsearch keystore, faster deployment creation, support for Elastic Stack 7.0, and more features released - March 20, 2019
  • Increased trial capacity, increased scaling limit, and more features released - February 13, 2019
  • Support for APM Servers, automatic multi-factor authentication, and more features released - January 24, 2019
  • Support to fine-tune snapshot retention periods, open support cases from the Elasticsearch Service console, and more features released - December 17, 2018
  • Cloud Console improvements, support to configure snapshot intervals, and more features released - November 12, 2018
  • SAML Support for Elasticsearch and Kibana released - October 29, 2018
  • Next-Generation of Elasticsearch Service released - August 1, 2018
  • Elasticsearch Service branding update - July 3, 2018
  • Minor Cloud Console updates released - June 19, 2018
  • New Cloud Console released - May 16, 2018

Also see Restrictions and known problems.
