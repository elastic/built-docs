Elasticsearch version 7.10.0edit

Also see Breaking changes in 7.10.

Breaking changesedit

Authentication
  • API key name should always be required for creation #59836 (issue: #59484)
Network
  • Set specific keepalive options by default on supported platforms #59278

Breaking Java changesedit

Machine Learning
  • Deprecate allow_no_jobs and allow_no_datafeeds in favor of allow_no_match #60601 (issue: #60642)
Mapping
  • Pass SearchLookup supplier through to fielddataBuilder #61430 (issue: #59332)

Deprecationsedit

Cluster Coordination
  • Deprecate and ignore join timeout #60872 (issue: #60873)
Machine learning
  • Renames */inference* APIs to */trained_models* #63097
