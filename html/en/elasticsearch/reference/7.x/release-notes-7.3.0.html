Elasticsearch version 7.3.0edit

Also see Breaking changes in 7.3.

Breaking changesedit

CCR
  • Do not allow modify aliases on followers #43017 (issue: #41396)
Data Frame
  • [ML][Data Frame] removing format support in date_histogram group_by #43659

Breaking Java changesedit

Mapping
  • Refactor put mapping request validation for reuse #43005 (issues: #37675, #41396)
Search
  • Refactor IndexSearcherWrapper to disallow the wrapping of IndexSearcher #43645

Features/Java High Level REST Client
