Upgrade Elastic Agentedit

This functionality is in beta and is subject to change. The design and code is less mature than official GA features and is being provided as-is with no warranties. Beta features are not subject to the support SLA of official GA features.

This process is not supported for upgrading DEB/RPM packages or Docker images.

To upgrade Elastic Agent to a new version:

  1. Download and extract the installation package for the new version.

  2. Run the install command to upgrade Elastic Agent.

    You must run this command as the root user.

    ./elastic-agent install

    You must run this command as the root user.

    ./elastic-agent install

    Open a PowerShell prompt as an Administrator (right-click the PowerShell icon and select Run As Administrator).

    From the PowerShell prompt, change to the directory where you installed Elastic Agent, and run:

    .\elastic-agent.exe install

    Follow the prompts to confirm that you want to upgrade Elastic Agent. The installation program detects the existing installation and begins upgrading to the new version.

To confirm that Elastic Agent is upgraded, go to the Agents tab in Fleet.

If you run into problems, see Troubleshoot common problems.
