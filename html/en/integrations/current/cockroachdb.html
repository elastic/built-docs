Elastic

This integration collects metrics from CockroachDB. It includes the following datasets for receiving logs:

  • status datastream: consists of status metrics

Compatibility

edit

The CockroachDB integration is compatible with any CockroachDB version exposing metrics in Prometheus format.

status

edit
