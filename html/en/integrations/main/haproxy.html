This integration periodically fetches logs and metrics from HAProxy servers.

Compatibility

edit

The log dataset was tested with logs from HAProxy 1.8, 1.9 and 2.0, 2.6 running on a Debian. It is not available on Windows. The integration supports the default log patterns below:

  • Default log format
  • TCP log format
  • HTTP log format
  • HTTPS log format
  • Error log format

The info and stat datasets were tested with tested with HAProxy versions from 1.6, 1.7, 1.8 to 2.0.

Troubleshooting

edit

If source.address is shown conflicted under metrics-* data view, then this issue can be solved by reindexing the stat data stream indices.
