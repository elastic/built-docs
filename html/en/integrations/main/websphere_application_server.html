What’s this?

Elastic

This Elastic integration is used to collect the following metrics from IBM WebSphere Application Server:

  • JDBC metrics
  • Servlet metrics
  • Session Manager metrics
  • ThreadPool metrics

This integration uses Prometheus to collect above metrics.

To open Prometheus endpoint read following instructions.

Compatibility

edit

This integration has been tested against WebSphere Application Server traditional version 9.0.5.17.

Troubleshooting

edit

If host.ip is shown conflicted under metrics-* data view, then this issue can be solved by reindexing the JDBC, Servlet, Session Manager and ThreadPool data stream’s indices.
