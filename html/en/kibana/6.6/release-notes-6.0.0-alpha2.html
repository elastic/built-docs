Kibana 6.0.0-alpha2edit

Breaking Changesedit

Core
  • Remove legacy pre-4.2 configurations #12013
  • NODE_ENV no longer has an impact on Kibana #12010
  • Only support 64 bit operating systems #11941
Watcher
  • The watch _status field has been renamed to status, as underscores in field names will not be allowed.

New Featuresedit

Core
  • Getting started page #11805
Dashboard
  • Clone dashboard from view mode #10925
Reporting
  • Enabled export as CSV.
Security
  • Added a View Dashboards Only setting to the roles section under Security management. If a user is assigned a role that has this setting enabled, when they access Kibana it presents Dashboard Only mode.
Visualize
  • Region map #10937
  • Gauge chart #10336

Enhancementsedit

Core
  • Add new UI setting to control the amount of items in listing pages #11674
Discover
  • Apply filters to the event context view #11466
Machine Learning
  • Added a Datefeed Preview tab to expanded rows in the jobs list under Job Management. This displays the JSON response from the preview endpoint.
Timelion
  • Support multiline Timelion queries #11972
Time series visual builder
  • Help text for writing painless scripts #11936
  • Toggle for automatically applying changes to visualization #11460
Timepicker
  • Improve accessibility of the datepicker #11753
Watcher
  • Watcher indices no longer use multiple types.

Bug Fixesedit

Graph
  • Reinstated _xpack/graph/_explore as the correct graph endpoint. _xpack/_graph/_explore is deprecated and will be removed in v7.0.
Timelion
  • Timelion tutorial now persists until you close it #11962
Time series visual builder
  • No longer trigger error in certain circumstances when using using percentiles #11772
Visualize
  • Disable save button if visualization is dirty #11576
