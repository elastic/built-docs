Kibana 6.7.0edit

Known issuesedit

Monitoring
  • If you have a version 7.0 monitoring cluster, a version 6.7 production cluster and Kibana version 6.7.0 or 6.7.1, you cannot see the monitoring data in Kibana. #34357

Breaking changesedit

Canvas
  • Removes custom WebPack bundles to remove duplicated dependencies. Canvas plugins must now use the Kibana platform. See the README.md for more information. #30123
