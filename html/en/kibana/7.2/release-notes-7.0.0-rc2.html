Kibana 7.0.0-rc2edit

Known issuesedit

Monitoring
  • If you have a version 7.0 monitoring cluster, a version 6.7 production cluster and Kibana version 6.7.0 or 6.7.1, you cannot see the monitoring data in Kibana. #34357
