Kibana 8.19.6

edit

The 8.19.6 release includes the following known issues, enhancements, and fixes.

Known issues

edit
Alerts aren’t generated for rules with alert flapping off and an alert delay higher than 1.

Details

Alerts aren’t generated for rules that have Alert flapping detection turned off and the alert delay set to a value higher than 1.

Workaround
Set the alert delay value to 1 or turn on Alert flapping detection.

Enhancements

edit
Elastic Security solution
For the Elastic Security 8.19.6 release information, refer to Elastic Security Solution Release Notes.
Kibana security
  • Enhances error message by explaining why the role is considered malformed (#239098).

Fixes

edit
Alerting and cases
  • Updates nodemailer to 7.0.9 (#238816).
  • Isolates the configuration parameters for the Tines connector to the server side (#236863).
Dashboards and Visualizations
  • Fixes bug that prevented you from deleting unsaved dashboard changes after you had enabled the timeRestore setting and selected a time and date using the date picker (#239992).
  • Fixes error that occured when you selected a blank value in the options list (#239791).
  • Fixes a bug in Lens that incorrectly assigned unsaved, ad-hoc dataview references (#239431).
  • Fixes bug that broke the filter functionalty in table rows when any column is used as a formula in Lens (#239222).
  • Fixes metric color assignment when breakdown and a max dimension are defined in Lens (#238901).
  • Ensures adhoc dataviews from ES|QL charts aren’t being filtered out in the KQL search bar (#238731).
  • Fixes bug where where an error in the visualization could cause the entire Visualize Listing page to error. This improves the error handling to make it easier to identify which visualization is causing problems (#238355).
  • Fixes bug that prevented transforming of panels in sections from URL state (#237382).
Data ingestion and Fleet
  • Fixes bug that occured during an integration policy upgrade that prevented the new package global variables from being included and stale variable references from being removed (#238542).
  • Fixes error when deleting orphaned integration policies and allows you to force delete orphaned integration policies using Fleet APIs (#237875).
  • Fixes an issue where fields were resetting automatically when editing ingest pipeline settings (#237509).
Discover
  • Ensures that the combined filters are considered when selecting the correct fields for the query while the courier:ignoreFilterIfFieldNotInIndex advanced setting is enabled (#238945).
  • Clears time field sorting after going from Classic to ES|QL mode (#235338).
Elastic Observability solution
  • Removes unecessary _source from APM query (#239205).
  • Uses word boundaries to only replace the metric name as an identifier and not as part of other text (#238849).
  • Fixes error when Observability AI Assistant is disabled (#238811).
  • Uses runWithCache for Fleet bulk operations (#238326).
  • Disables max attempts for private locations sync task (#237784).
  • Fixes recover alert while monitor is down (#237479).
  • Fixes the AI Assistant button tooltip by closing the tooltip when you are not hovering over the button (#237202).
Kibana platform
  • Fixes files management flyout crashing (#237588).
Management
  • Fixes a bug where creating a new component template with @custom suffix in its name would update mappings of all unrelated data streams and a message would display asking to rollover conflicting ones (#237952).
Search
  • Fixes an issue in the RAG Playground where invalid fields displayed red styling but no error message. Error text is now shown to help users identify and correct form issues (#238284).
