Task Manager health APIedit

Retrieve the health status of the Kibana Task Manager.

Requestedit

GET <kibana host>:<port>/api/task_manager/_health

Response codeedit

200
Indicates a successful call.

Exampleedit

Retrieve the health status of the Kibana Task Manager:

$ curl -X GET api/task_manager/_health
