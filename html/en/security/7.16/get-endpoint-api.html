Get endpointedit

Retrieves metadata about a single host running Endpoint Security.

Console supports Elasticsearch APIs only. Console doesn’t allow interactions with Kibana APIs. You must use curl or another HTTP tool instead. For more information, refer to Run Elasticsearch API requests.

Request URLedit

GET <kibana host>:<port>/api/endpoint/metadata/<endpointID>

Example requestsedit

Retrieves an endpoint with an endpointID value of ed518850-681a-4d60-bb98-e22640cae2a8:

GET /api/endpoint/metadata/ed518850-681a-4d60-bb98-e22640cae2a8

Response codeedit

200
Indicates a successful call.
