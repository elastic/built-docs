Index template scriptedit

This code creates a new index template for temporarily storing existing detection alerts when you update .siem-signals-* index mappings. You need to update the index mappings to visualize process relationships after upgrading to Elastic Stack release 7.9.0 or 7.9.1 from a previous minor release (7.8.x, 7.7.x, and so on).

Click here to scroll to the bottom of the page and use the builtin functions to paste the code into the Kibana dev console. You can click on the settings icon to update Kibana’s URL.
