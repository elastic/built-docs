Generated Documents

edit

The APM Server produces documents with a specific structure. The defined structure can be used to create an index in Elasticsearch. See the examples in this section for different document types.

Examples

edit

In the following you find example documents sent to Elasticsearch from APM Server:

  • Transaction document
  • Trace document
  • Error document
