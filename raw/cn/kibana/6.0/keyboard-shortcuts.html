键盘快捷键edit

控制台配备了一套非常方便的键盘快捷键，使其工作效率更高。下面是一段概述：

一般编辑edit

Ctrl/Cmd + I
当前请求自动缩进。
Ctrl + Space
打开自动补全 （即使没有打字也可以）。
Ctrl/Cmd + Enter
提交请求。
Ctrl/Cmd + Up/Down
跳转到上一个/下一个请求的开始或结束。
Ctrl/Cmd + Alt + L
折叠或展开当前代码块。
Ctrl/Cmd + Option + 0
折叠除当前代码块之外的所有代码块，通过添加 shift 来展开。

自动补全可见时edit

Down arrow
光标切换到自动补全菜单，使用方向键选择下一个选项。
Enter/Tab
在自动补全菜单中选择当前或最上面的选项。
Esc
关闭自动补全菜单。
