Contributingedit

The Go APM Agent is open source and we love to receive contributions from our community — you!

There are many ways to contribute, from writing tutorials or blog posts, improving the documentation, submitting bug reports and feature requests or writing code.

You can get in touch with us through Discuss. Feedback and ideas are always welcome.

Reporting bugs and contributing codeedit

To report bugs, please create an Issue on the apm-agent-go GitHub repository. For tips on contributing code fixes or enhancements, please see the contributing guide in the code repository.
