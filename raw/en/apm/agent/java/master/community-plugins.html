Community Pluginsedit

This page lists some of the plugins developed by the open source community.

Have a question about developing a community plugin? You can post questions and discuss issues in the Java discussion forum.

Have you created a plugin that’s not listed? Add the name and description of your plugin to the source document for Community Plugins and open a pull request in the Java agent GitHub repository to get your change merged.

Want to contribute? Contributing to the Elastic APM Java agent.

Elastic provides no warranty or support for community-sourced plugins.
