Introductionedit

Welcome to the APM Node.js Agent documentation.

The Elastic APM Node.js agent sends performance metrics and errors to the APM Server. It has built-in support for the most popular frameworks and routers, as well as a simple API which allows you to instrument any application. The agent is only one of multiple components you need to get started with APM. Please also have a look at the documentation for

  • APM Server
  • Elasticsearch
