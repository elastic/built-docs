Introductionedit

Welcome to the APM Real User Monitoring (RUM) JavaScript Agent documentation.

The Elastic APM RUM JavaScript agent sends performance metrics and errors to the APM Server. The agent is only one of multiple components you need to get started with APM. Please also have a look at the documentation for

  • APM Server

    • APM JavaScript agent is compatible with APM Server v6.4+.
    • enable frontend endpoints in the apm-server configuration.
  • Elasticsearch
