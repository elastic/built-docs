Instrument and configure podsedit

This functionality is in technical preview and may be changed or removed in a future release. Elastic will apply best effort to fix any issues, but features in technical preview are not subject to the support SLA of official GA features.

To instrument and configure your application pods, complete the following steps:

  1. Add the helm repository to Helm
  2. Configure the webhook with a Helm values file
  3. Install the webhook with Helm
  4. Add a pod template annotation to each pod you want to auto-instrument
  5. Watch data flow into the Elastic Stack
