This documentation refers to the API of the standalone (legacy) APM Server. This method of running APM Server will be deprecated and removed in a future release. Please consider upgrading to Fleet and the APM integration. If you’ve already upgraded, see API.

The APM Server exposes endpoints for:

  • Events intake
  • Sourcemap upload
  • Agent configuration
  • Server information
