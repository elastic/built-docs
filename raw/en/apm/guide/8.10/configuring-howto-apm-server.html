Configure

edit

How you configure the APM Server depends on your deployment method.

  • APM Server binary users need to edit the apm-server.yml configuration file. The location of the file varies by platform. To locate the file, see Installation layout.
  • Fleet-managed users configure the APM Server directly in Kibana. Each configuration page describes the specific location.
  • Elastic cloud users should see Add APM user settings for information on how to configure Elastic APM.
