Troubleshootingedit

This documentation refers to the standalone (legacy) method of running APM Server. This method of running APM Server will be deprecated and removed in a future release. Please consider upgrading to the Elastic APM integration. If you’ve already upgraded, see Troubleshooting.

If you run into trouble, there are three places you can look for help.
