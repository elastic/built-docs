OpenTelemetry API/SDK with Elastic APM agentsedit

Use the OpenTelemetry API/SDKs with Elastic APM agents. Supported Elastic APM agents translate OpenTelemetry API calls to Elastic APM API calls. This allows you to reuse your existing instrumentation to create Elastic APM transactions and spans.

If you’d like to use OpenTelemetry to send data directly to the APM server instead, see OpenTelemetry native support.

See the relevant Elastic APM agent documentation to get started:

  • Java
  • .NET (link coming soon)
  • Node.js
  • Python
