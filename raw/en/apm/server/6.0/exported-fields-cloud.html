Cloud provider metadata fieldsedit

Metadata from cloud providers added by the add_cloud_metadata processor.

meta.cloud.provideredit

example: ec2

Name of the cloud provider. Possible values are ec2, gce, or digitalocean.

meta.cloud.instance_idedit

Instance ID of the host machine.

meta.cloud.instance_nameedit

Instance name of the host machine.

meta.cloud.machine_typeedit

example: t2.medium

Machine type of the host machine.

meta.cloud.availability_zoneedit

example: us-east-1c

Availability zone in which this host is running.

meta.cloud.project_idedit

example: project-x

Name of the project in Google Cloud.

meta.cloud.regionedit

Region in which this host is running.
