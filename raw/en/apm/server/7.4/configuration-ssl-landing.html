SSL/TLS settingsedit

SSL/TLS is available for:

  • APM Server inputs (APM Agents)
  • APM Server outputs that support SSL, like Elasticsearch, Logstash, or Kafka.

Additional information on getting started with SSL/TLS is available in Securing APM Server.
