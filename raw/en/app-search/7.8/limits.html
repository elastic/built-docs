Limitsedit

For performance and historical reasons, App Search has default limitations on some objects and API calls.

There are two types of limits:

  • Engine Level: Limits which apply to one Engine.
  • Query Level: Limitations on API request building.

Many limitations are configurable via the enterprise-search.yml configuration file. Those are noted with the version where they became configurable.
