System module

edit

This module collects and parses logs created by system logging server of common Unix/Linux based distributions.

Compatibility

edit

This module was tested with logs from OSes like Ubuntu 12.04, Centos 7, and macOS Sierra.

This module is not available for Windows.

Dashboard

edit

This module comes with a sample dashboard showing syslog data.

Syslog fileset settings

edit

var.paths

edit

An array of paths where to look for the log files. If left empty, Filebeat will choose the paths depending on your operating systems.
