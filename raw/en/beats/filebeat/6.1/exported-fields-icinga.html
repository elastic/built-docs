Icinga fieldsedit

Icinga Module

icinga fieldsedit

debug fieldsedit

Contains fields for the Icinga debug logs.

icinga.debug.facilityedit

type: keyword

Specifies what component of Icinga logged the message.

icinga.debug.severityedit

type: keyword

Possible values are "debug", "notice", "information", "warning" or "critical".

icinga.debug.messageedit

type: text

The logged message.
