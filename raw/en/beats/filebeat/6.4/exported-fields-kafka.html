log fields

Kafka log lines.

kafka.log.timestamp

The timestamp from the log line.

kafka.log.level

example: WARN

The log level.

kafka.log.message

type: text

The logged message.

kafka.log.component

type: keyword

Component the log is coming from.

kafka.log.class

type: text

Java class the log is coming from.
