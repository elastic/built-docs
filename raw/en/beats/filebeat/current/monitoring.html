Monitoring Filebeatedit

You can use the Elastic Stack monitoring features to gain insight into the health of Filebeat agents running in your environment.

To monitor Filebeat, make sure monitoring is enabled on your Elasticsearch cluster, then configure the method used to collect Filebeat metrics. You can use one of following methods:

  • Internal collection
  • Metricbeat collection

To learn about monitoring in general, see Monitoring the Elastic Stack.
