Docker fieldsedit

Docker stats collected from Docker.

docker.container.id

type: keyword

Unique container id.

docker.container.image

type: keyword

Name of the image the container was built on.

docker.container.name

type: keyword

Container name.

docker.container.labels

type: object

Image labels.
