Exported fieldsedit

This document describes the fields that are exported by Functionbeat. They are grouped in the following categories:

  • Beat fields
  • Cloud provider metadata fields
  • Docker fields
  • ECS fields
  • Functionbeat fields
  • Host fields
  • Jolokia Discovery autodiscover provider fields
  • Kubernetes fields
  • Process fields
