Configure the output

edit

You configure Functionbeat to write to a specific output by setting options in the Outputs section of the functionbeat.yml config file. Only a single output may be defined.

The following topics describe how to configure each supported output. If you’ve secured the Elastic Stack, also read Secure for more about security-related configuration options.

  • Elasticsearch Service
  • Elasticsearch
  • Logstash
  • Console
