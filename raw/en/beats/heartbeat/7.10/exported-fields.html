Exported fieldsedit

This document describes the fields that are exported by Heartbeat. They are grouped in the following categories:

  • Beat fields
  • Cloud provider metadata fields
  • Common heartbeat monitor fields
  • Docker fields
  • ECS fields
  • Host fields
  • HTTP monitor fields
  • ICMP fields
  • Jolokia Discovery autodiscover provider fields
  • Kubernetes fields
  • Process fields
  • Host lookup fields
  • SOCKS5 proxy fields
  • Monitor summary fields
  • TCP layer fields
  • TLS encryption layer fields
