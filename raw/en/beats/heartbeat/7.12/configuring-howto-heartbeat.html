Configure Heartbeatedit

To get started quickly, read Quick start: installation and configuration.

To configure Heartbeat, edit the configuration file. The default configuration file is called heartbeat.yml. The location of the file varies by platform. To locate the file, see Directory layout.

There’s also a full example configuration file called heartbeat.reference.yml that shows all non-deprecated options.

See the Config File Format for more about the structure of the config file.
