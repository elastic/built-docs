Configure the outputedit

You configure Journalbeat to write to a specific output by setting options in the output section of the journalbeat.yml config file. Only a single output may be defined.

The following topics describe how to configure each supported output:

  • Elasticsearch
  • Logstash
  • Kafka
  • Redis
  • File
  • Console
  • Cloud

If you’ve secured the Elastic Stack, also read Securing Journalbeat for more about security-related configuration options.
