Monitoring Journalbeatedit

You can use the Elastic Stack monitoring features to gain insight into the health of Journalbeat agents running in your environment.

To monitor Journalbeat, make sure monitoring is enabled on your Elasticsearch cluster, then configure the method used to collect Journalbeat metrics. You can use one of following methods:

  • Internal collection - Internal collectors send monitoring data directly to your monitoring cluster.
  • Legacy internal collection (deprecated) - Legacy internal collectors send monitoring data to your production cluster.

To learn about monitoring in general, see Monitoring the Elastic Stack.
