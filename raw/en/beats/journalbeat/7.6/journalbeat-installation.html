Step 1: Install Journalbeat

edit

Step 1: Install Journalbeatedit

Before you begin: If you haven’t installed the Elastic Stack, do that now. See Get started with the Elastic Stack.

To download and install Journalbeat, use the commands that work with your system.

If you use Apt or Yum, you can install Journalbeat from our repositories to update to the newest version more easily.

See our download page for other installation options, such as 32-bit images.

deb:

curl -L -O https://artifacts.elastic.co/downloads/beats/journalbeat/journalbeat-7.6.2-amd64.deb
sudo dpkg -i journalbeat-7.6.2-amd64.deb

rpm:

curl -L -O https://artifacts.elastic.co/downloads/beats/journalbeat/journalbeat-7.6.2-x86_64.rpm
sudo rpm -vi journalbeat-7.6.2-x86_64.rpm

linux:

curl -L -O https://artifacts.elastic.co/downloads/beats/journalbeat/journalbeat-7.6.2-linux-x86_64.tar.gz
tar xzvf journalbeat-7.6.2-linux-x86_64.tar.gz

docker:

See Running on Docker for deploying Docker containers.
