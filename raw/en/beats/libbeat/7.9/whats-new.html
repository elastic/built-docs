What’s new in Beats 7.9edit

Here are the highlights of what’s new and improved in 7.9. For detailed information about this release, check out the release notes and breaking changes.

Elastic Logging Plugin for Docker now supports docker logs commandedit

Starting with version 7.9, the Elastic Logging Plugin for Docker fully supports the docker logs command. The plugin maintains a local copy of logs that can be read without a connection to Elasticsearch. The plugin mounts the /var/lib/docker directory on the host to write logs to /var/log/containers. For more information, see the Elastic Logging Plugin for Docker docs.
