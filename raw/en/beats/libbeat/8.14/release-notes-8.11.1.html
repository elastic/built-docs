Beats version 8.11.1edit

View commits

Due to a memory leak issue, Windows users running Elastic Agent are recommended to avoid upgrading to this release and instead use the 8.11.2 release in which the issue is resolved. If you’ve already upgraded to version 8.11.0 or 8.11.1, we recommend upgrading to 8.11.2 as soon as possible.

Addededit

Affecting all Beats

  • Setting environmental variable ELASTIC_NETINFO:false will disable the netinfo.enabled option of add_host_metadata processor.
