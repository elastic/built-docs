Elastic Logging Plugin overviewedit

This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features.

The Elastic Logging Plugin is a Docker plugin that sends container logs to the Elastic Stack, where you can search, analyze, and visualize the data in real time.

The Elastic Logging Plugin is built on top of the Beats platform and supports many of the features and outputs supported by the Beats shippers.

Beat users: see Known problems and limitations.
