Nginx Module

edit

This module periodically fetches metrics from Nginx servers.

Compatibility

edit

The Nginx metricsets were tested with Nginx 1.9 and are expected to work with all version >= 1.9.

Example Configuration

edit

The Nginx module supports the standard configuration options that are described in Modules Configuration. Here is an example configuration:

metricbeat.modules:
- module: nginx
  metricsets: ["stubstatus"]
  enabled: true
  period: 10s

  # Nginx hosts
  hosts: ["http://127.0.0.1"]
