Apache Moduleedit

This module periodically fetches metrics from Apache HTTPD servers.

Compatibilityedit

The Apache metricsets were tested with Apache 2.4.20 and are expected to work with all versions >= 2.2.31 and >= 2.4.16.

Example Configurationedit

The Apache module supports the standard configuration options that are described in Modules. Here is an example configuration:

metricbeat.modules:
- module: apache
  metricsets: ["status"]
  enabled: true
  period: 10s

  # Apache hosts
  hosts: ["http://127.0.0.1"]
