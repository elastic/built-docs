System Core Metricset

edit

The System core metricset provides load statistics for each CPU core.

This metricset is available on:

  • Darwin
  • FreeBSD
  • Linux
  • OpenBSD
  • Windows

Fields

edit

For a description of each field in the metricset, see the exported fields section.
