Kibana moduleedit

This functionality is experimental and may be changed or removed completely in a future release. Elastic will take a best effort approach to fix any issues, but experimental features are not subject to the support SLA of official GA features.

The Kibana module only tracks the high-level metrics. To monitor more Kibana metrics, use our X-Pack monitoring which is available under a free basic license.

Example configurationedit

The Kibana module supports the standard configuration options that are described in Specify which modules to run. Here is an example configuration:

metricbeat.modules:
- module: kibana
  metricsets: ["status"]
  period: 10s
  hosts: ["localhost:5601"]
