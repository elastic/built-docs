Dropwizard moduleedit

This is the Dropwizard module. The default metricset is collector.

Compabilityedit

The Dropwizard module is tested with dropwizard metrics 3.1.0.

Example configurationedit

The Dropwizard module supports the standard configuration options that are described in Specify which modules to run. Here is an example configuration:

metricbeat.modules:
- module: dropwizard
  metricsets: ["collector"]
  period: 10s
  hosts: ["localhost:8080"]
  metrics_path: /metrics/metrics
  namespace: example
  enabled: true

This module supports TLS connections when using ssl config field, as described in Specify SSL settings. It also supports the options described in Standard HTTP config options.

Metricsetsedit

The following metricsets are available:

  • collector
