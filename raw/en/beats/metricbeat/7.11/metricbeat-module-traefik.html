Traefik moduleedit

This module periodically fetches metrics from a Traefik instance. The Traefik instance must be configured to expose it’s HTTP API.

Compatibilityedit

The Traefik metricsets were tested with Traefik 1.6.

Example configurationedit

The Traefik module supports the standard configuration options that are described in Modules. Here is an example configuration:

metricbeat.modules:
- module: traefik
  metricsets: ["health"]
  period: 10s
  hosts: ["localhost:8080"]

This module supports TLS connections when using ssl config field, as described in SSL. It also supports the options described in Standard HTTP config options.

Metricsetsedit

The following metricsets are available:

  • health
