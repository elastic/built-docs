Elasticsearch ccr metricsetedit

This is the ccr metricset of the Elasticsearch module. It interrogates the Cross Cluster Replication Stats API endpoint to fetch information about shards in the Elasticsearch cluster that are participating in cross-cluster replication.

Fieldsedit

For a description of each field in the metricset, see the exported fields section.
