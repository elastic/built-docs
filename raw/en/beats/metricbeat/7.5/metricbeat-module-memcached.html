Memcached moduleedit

This is the Memcached module. These metricsets were tested with Memcached version 1.4.35.

The default metricset is stats.

Compatibilityedit

The memcached module is tested with memcached 1.4.35.

Example configurationedit

The Memcached module supports the standard configuration options that are described in Specify which modules to run. Here is an example configuration:

metricbeat.modules:
- module: memcached
  metricsets: ["stats"]
  period: 10s
  hosts: ["localhost:11211"]
  enabled: true

Metricsetsedit

The following metricsets are available:

  • stats
