couchdb module

edit

This is the couchdb module.

The default metricset is server.

Compatibility

edit

The Couchdb module is tested with Couchdb 1.7.

Example configuration

edit

The couchdb module supports the standard configuration options that are described in Modules. Here is an example configuration:

metricbeat.modules:
- module: couchdb
  metricsets: ["server"]
  period: 10s
  hosts: ["localhost:5984"]
