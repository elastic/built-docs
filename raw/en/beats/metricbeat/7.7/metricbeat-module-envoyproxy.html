Envoyproxy moduleedit

This is the envoyproxy module.

The default metricset is server.

Compatibilityedit

The envoyproxy module is tested with Envoy 1.7.0 and 1.12.0.

Example configurationedit

The Envoyproxy module supports the standard configuration options that are described in Modules. Here is an example configuration:

metricbeat.modules:
- module: envoyproxy
  metricsets: ["server"]
  period: 10s
  hosts: ["localhost:9901"]

Metricsetsedit

The following metricsets are available:

  • server
