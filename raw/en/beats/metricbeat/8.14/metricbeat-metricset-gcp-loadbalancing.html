Google Cloud Platform loadbalancing metricsetedit

Load Balancing metricset fetches metrics from Load Balancing in Google Cloud Platform.

The loadbalancing metricset contains all metrics exported from the Stackdriver API. The field names are aligned to Beats naming conventions with minor modifications to their GCP metrics name counterpart.

Fields

For a description of each field in the metricset, see the exported fields section.
