Google Cloud Platform storage metricsetedit

Storage metricset fetches metrics from Storage in Google Cloud Platform.

The storage metricset contains all metrics exported from the GCP Storage Monitoring API. The field names are aligned to Beats naming conventions with minor modifications to their GCP metrics name counterpart.

You can specify a single region to fetch metrics like us-central1. Be aware that GCP Storage does not use zones so us-central1-a will return nothing. If no region is specified, metrics are returned from all buckets.

Fields

For a description of each field in the metricset, see the exported fields section.

Here is an example document generated by this metricset:
