MongoDB dbstats metricsetedit

This is the dbstats metricset of the MongoDB module.

It requires the following privileges, which is covered by the clusterMonitor role:

  • listDatabases action on cluster resource
  • for each of the databases, also need dbStats action on the database resource

This is a default metricset. If the host module is unconfigured, this metricset is enabled by default.

Fields

For a description of each field in the metricset, see the exported fields section.
