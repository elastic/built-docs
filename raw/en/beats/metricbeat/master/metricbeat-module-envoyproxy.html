envoyproxy moduleedit

This is the envoyproxy module.

The default metricset is server.

Compatibilityedit

The envoyproxy module is tested with Envoy 1.7.0.

Example configurationedit

The envoyproxy module supports the standard configuration options that are described in Specify which modules to run. Here is an example configuration:

metricbeat.modules:
- module: envoyproxy
  metricsets: ["server"]
  period: 10s
  hosts: ["localhost:9901"]

Metricsetsedit

The following metricsets are available:

  • server
