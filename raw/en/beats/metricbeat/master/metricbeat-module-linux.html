Refer to the documentation for a detailed comparison of Beats and Elastic Agent.

The Linux module reports on metrics exclusive to the Linux kernel and GNU/Linux OS.

Example configuration

The Linux module supports the standard configuration options that are described in Modules. Here is an example configuration:

metricbeat.modules:
- module: linux
  period: 10s
  metricsets:
    - "pageinfo"
    - "memory"
    # - ksm
    # - conntrack
    # - iostat
    # - pressure
    # - rapl
  enabled: true
  #hostfs: /hostfs
  #rapl.use_msr_safe: false

Metricsets

The following metricsets are available:

  • conntrack
  • iostat
  • ksm
  • memory
  • pageinfo
  • pressure
  • rapl
