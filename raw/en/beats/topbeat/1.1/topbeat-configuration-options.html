Configuration Optionsedit

Before modifying configuration settings, make sure you’ve completed the configuration steps in the Getting Started.

The Topbeat configuration file uses YAML for its syntax. The configuration options are described in the following sections. After changing configuration settings, you need to restart Topbeat to pick up the changes.

  • Input
  • Elasticsearch Output
  • Logstash Output
  • Redis Output
  • File Output
  • Console Output
  • Shipper
  • Logging (Optional)
  • Run Options (Optional)
