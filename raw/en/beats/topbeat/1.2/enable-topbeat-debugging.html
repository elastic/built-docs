Debuggingedit

By default, Topbeat sends all its output to syslog. When you run Topbeat in the foreground, you can use the -e command line flag to redirect the output to standard error instead. For example:

topbeat -e

The default configuration file is topbeat.yml (the location of the file varies by platform). You can use a different configuration file by specifying the -c flag. For example:

topbeat -e -c mytopbeatconfig.yml

You can increase the verbosity of debug messages by enabling one or more debug selectors. For example, to view the published transactions, you can start Topbeat with the publish selector like this:

topbeat -e -d "publish"

If you want all the debugging output (fair warning, it’s quite a lot), you can use *, like this:

topbeat -e -d "*"
