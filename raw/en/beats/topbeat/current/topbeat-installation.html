Step 1: Installing Topbeatedit

To download and install Topbeat, use the commands that work with your system (deb for Debian/Ubuntu, rpm for Redhat/Centos/Fedora, mac for OS X, and win for Windows).

If you use Apt or Yum, you can install Topbeat from our repositories to update to the newest version more easily.

deb:

curl -L -O https://download.elastic.co/beats/topbeat/topbeat_1.3.1_amd64.deb
sudo dpkg -i topbeat_1.3.1_amd64.deb

rpm:

curl -L -O https://download.elastic.co/beats/topbeat/topbeat-1.3.1-x86_64.rpm
sudo rpm -vi topbeat-1.3.1-x86_64.rpm

mac:

curl -L -O https://download.elastic.co/beats/topbeat/topbeat-1.3.1-darwin.tgz
tar xzvf topbeat-1.3.1-darwin.tgz

win:

  1. Download the Topbeat Windows zip file.
  2. Extract the contents of the zip file into C:\Program Files.
  3. Rename the topbeat-<version>-windows directory to Topbeat.
  4. Open a PowerShell prompt as an Administrator (right-click the PowerShell icon and select Run As Administrator). If you are running Windows XP, you may need to download and install PowerShell.

  5. Run the following commands to install Topbeat as a Windows service:

    PS > cd 'C:\Program Files\Topbeat'
PS C:\Program Files\Topbeat> .\install-service-topbeat.ps1

If script execution is disabled on your system, you need to set the execution policy for the current session to allow the script to run. For example: PowerShell.exe -ExecutionPolicy UnRestricted -File .\install-service-topbeat.ps1.

Before starting Topbeat, you should look at the configuration options in the configuration file, for example C:\Program Files\Topbeat\topbeat.yml. For more information about these options, see Configuration Options.
