Configure the outputedit

You configure Winlogbeat to write to a specific output by setting options in the output section of the winlogbeat.yml config file. Only a single output may be defined.

The following topics describe how to configure each supported output:

  • Elasticsearch
  • Logstash
  • Kafka
  • Redis
  • File
  • Console
