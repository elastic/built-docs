Elastic Cloud Enterprise 1.0.1edit

New for Elastic Cloud Enterprise version 1.0.1:

  • The Elastic Stack packs shipped with Elastic Cloud Enterprise have been updated. ECE now ships with Elasticsearch and Kibana version 5.4.1 in addition to the previously shipped Elasticsearch 2.4.5 and Kibana 4.6.4. To learn more about the 5.4.1 version of the Elastic Stack, see Elasticsearch 5.4.1 and 5.3.3 released.

If you previously installed version 1.0.0, you do not need to upgrade your ECE installation. You can add the Elastic Stack pack for version 5.4.1 and then upgrade your deployments.

You should upgrade the admin-console-elasticsearch and logging-and-metrics clusters used by ECE and any existing 5.4.0 clusters that you might have created. There is currently no supported method for deleting Elastic Stack packs, but we recommend that you do not create any new 5.4.0 clusters.
