Elastic Cloud Enterprise 2.1.1edit

Elastic Cloud Enterprise version 2.1.1 includes the following enhancement:

  • Improved Elastic Cloud Enterprise stability. Various fixes to the API and ZooKeeper have improved the stability of Elastic Cloud Enterprise.

Bug Fixesedit

Elastic Cloud Enterprise version 2.1.1 includes the following bug fixes:

  • apm.system_settings and kibana.system_settings are now ignored. When changes are made to the Kibana and APM values in the top-level system_settings, the changes are now ignored to avoid configuration errors.
  • Fixed documentation links. When you click links in the UI or Elastic Cloud Enterprise documentation, the links now take you to the correct location.

Release date: March, 26 2019
