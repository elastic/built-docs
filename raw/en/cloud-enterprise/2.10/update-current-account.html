Updates the current accountedit

Updates the current account.

Requestedit

PUT /api/v1/account

Request bodyedit

(AccountUpdateRequest) (required) the current account

Responsesedit

200

(AccountResponse) Account updated successfully

404

(BasicFailedReply) Account not found. (code: accounts.not_found)

Headers

x-cloud-error-codes (string; allowed values: [accounts.not_found])
The error codes associated with the response

Request exampleedit

curl -XPUT https://{{hostname}}/api/v1/account \
-H "Authorization: ApiKey $ECE_API_KEY" \
-H 'Content-Type: application/json' \
-d '
{
   "trust" : {
      "trust_all" : true
   }
}
'
