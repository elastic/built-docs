ApmSettingsedit

The settings for the APM Server.

Propertiesedit

metadata (ClusterMetadataSettings)
The top-level configuration settings for the Elasticsearch cluster.

Exampleedit

{
   "metadata" : {
      "contact_email" : "string",
      "hidden" : true,
      "name" : "string",
      "organization_id" : "string",
      "owner_id" : "string",
      "resources" : {
         "cpu" : {
            "boost" : true,
            "hard_limit" : true
         }
      },
      "subscription_level" : "string",
      "system_owned" : true
   }
}
