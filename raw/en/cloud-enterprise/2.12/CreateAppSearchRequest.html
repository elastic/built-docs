CreateAppSearchRequestedit

The request body for creating an App Search.

Propertiesedit

name (string)
The human readable name for the App Search cluster (default: takes the name of its Elasticsearch cluster)
plan (AppSearchPlan, required)
The plan for the App Search cluster.
