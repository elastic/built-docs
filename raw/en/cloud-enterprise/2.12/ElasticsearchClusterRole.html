ElasticsearchClusterRole

edit

The authorization information for an Elasticsearch cluster user.

Properties

edit
roles (array[string], required)
The list of roles for this user
username (string, required)
The username

Example

edit

   "roles" : [
      "string"
   ],
   "username" : "string"
}
