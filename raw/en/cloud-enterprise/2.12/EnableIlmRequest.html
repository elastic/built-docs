EnableIlmRequestedit

Request sent to enable ILM on a deployment.

Propertiesedit

index_patterns (array[IndexPattern], required)
A locally-unique user-specified id for Kibana

Exampleedit

{
   "index_patterns" : [
      {
         "index_pattern" : "string",
         "node_attributes" : {
            "some_property" : "string"
         },
         "policy_name" : "string"
      }
   ]
}
