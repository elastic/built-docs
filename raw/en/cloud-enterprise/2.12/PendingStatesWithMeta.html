PendingStatesWithMeta

edit

Persisted pending states for a role along with metadata

Properties

edit
meta (Metadata, required)
The metadata
value (PendingStates, required)
The pending stages

Example

edit
{
   "meta" : {
      "created_time" : "2019-01-01T00:00:00Z",
      "modified_time" : "2019-01-01T00:00:00Z",
      "version" : "string"
   },
   "value" : {
      "runner_ids_to_pending_state" : {
         "some_property" : {
            "value" : true
         }
      }
   }
}
