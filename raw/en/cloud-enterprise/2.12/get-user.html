Fetch a single useredit

Fetches a single user.

Requestedit

GET /api/v1/users/{user_name}

Path parametersedit

Name Type Required Description

user_name

string

Y

Identifier for the user

Responsesedit

200

(User) User successfully fetched

404

(BasicFailedReply) User not found. (code: user.not_found)

Headers

x-cloud-error-codes (string; allowed values: [user.not_found])
The error codes associated with the response

Request exampleedit

curl -XGET https://{{hostname}}/api/v1/users/{user_name} \
-H "Authorization: ApiKey $ECE_API_KEY"
