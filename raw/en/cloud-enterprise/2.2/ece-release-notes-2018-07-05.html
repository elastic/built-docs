Elastic Stack 6.3.1 released

Elastic Cloud Enterprise now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 6.3.1.

For more information about the features in Elasticsearch and Kibana 6.3.1, see:

  • Kibana 6.3.1 Release Notes
