What’s new with the Elastic Stack

As soon as the latest Elastic Stack version is released, we make it available to you.

From the Cloud UI, you can upgrade your existing deployments to the latest versions of the Elastic Stack. To learn more about the best practices to use when upgrading, refer to Upgrade Versions.
