Refresh authentication token

edit

Issues a new authentication token.

Request

edit

POST /api/v1/users/auth/_refresh

Responses

edit
200

(TokenResponse) The token refreshed successfully and was returned in the body of the response.

401

(BasicFailedReply) The authentication token is invalid or expired. (code: root.unauthorized)

501

(BasicFailedReply) You need to configure the authentication cluster. (code: authc.no_authentication_cluster)

502

(BasicFailedReply) The authentication cluster failed to process the request. The response body contains details about the error. (code: authc.authentication_cluster_error)

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request example

edit
curl -XPOST {{hostname}}/api/v1/users/auth/_refresh \
-u $CLOUD_USER:$CLOUD_KEY
