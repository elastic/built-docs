Extensionedit

An API extension. It represents clusters' plugins and bundles

Propertiesedit

extension_type (string, required)
The extension type.
id (string, required)
The identifier of the extension.
name (string, required)
The name of the extension.
url (string, required)
The extension URL.
version (string)
Elasticsearch version.
