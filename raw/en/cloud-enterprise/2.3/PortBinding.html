PortBindingedit

Value used to bind to a port on the host.

Propertiesedit

host_ip (string)

IP to bind to on the host. I.e {@code 0.0.0.0}

Example: "0.0.0.0"

host_port (string, required)
Port as observed by the host.

Exampleedit

{
   "host_ip" : "string",
   "host_port" : "string"
}
