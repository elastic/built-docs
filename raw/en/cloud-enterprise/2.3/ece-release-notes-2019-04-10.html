Elastic Stack 7.0.0 released

Elastic Cloud Enterprise now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 7.0.0.

For more information about the features in Elasticsearch and Kibana 7.0.0, see:

  • Elasticsearch 7.0.0 Released
  • Elasticsearch 7.0.0 Release Notes
  • Kibana 7.0.0 Release Notes

Service release: April 10, 2019
