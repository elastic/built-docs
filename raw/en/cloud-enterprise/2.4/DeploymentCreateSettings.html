DeploymentCreateSettingsedit

Additional configuration for the new deployment object.

Propertiesedit

ip_filtering_settings (IpFilteringSettings)
The set of rulesets applies to this deployment.

Exampleedit

{
   "ip_filtering_settings" : {
      "rulesets" : [
         "string"
      ]
   }
}
