MetadataItemsedit

The container object for metadata items.

Propertiesedit

items (array[MetadataItem], required)
A collection of arbitrary metadata items associated with a resource.

Exampleedit

{
   "items" : [
      {
         "key" : "string",
         "value" : "string"
      }
   ]
}
