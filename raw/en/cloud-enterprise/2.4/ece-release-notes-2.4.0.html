Elastic Cloud Enterprise 2.4.0

New for Elastic Cloud Enterprise 2.4.0:

  • Updated proxy layer defaults. In previous versions, the Elastic Cloud Enterprise proxy layer permitted negotiation with a client using TLS 1.0 and an intermediate level cipher suite. The proxy settings are now updated to default to minimum TLS 1.2 and a modern cipher suite. To revert to the legacy behavior see Configuring the TLS version.
