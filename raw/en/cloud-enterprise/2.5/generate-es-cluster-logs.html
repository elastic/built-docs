edit

Generate logsedit

DEPRECATED (Scheduled to be removed in the next major version): Retrieves the log file for the Elasticsearch cluster.

Requestedit

GET /api/v1/clusters/elasticsearch/{cluster_id}/support/_generate-logs

Path parametersedit

Name Type Required Description

cluster_id

string

Y

The Elasticsearch cluster identifier.

Query parametersedit

Name Type Required Description

date

string

Y

The log retrieval start date in standard format. For example, YYYY-MM-DD [THH [:mm] ].

Responsesedit

200

(string as byte) The support diagnostic bundle was successfully generated

404

(BasicFailedReply) The cluster specified by {cluster_id} cannot be found (code: 'clusters.cluster_not_found')

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request exampleedit

curl -XGET https://{{hostname}}/api/v1/clusters/elasticsearch/{cluster_id}/support/_generate-logs \
-H "Authorization: ApiKey $ECE_API_KEY"
