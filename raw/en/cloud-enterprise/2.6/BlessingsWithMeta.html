BlessingsWithMeta

edit

The blessings

Example

edit

   "meta" : {
      "created_time" : "2019-01-01T00:00:00Z",

   },
   "value" : {
      "runner_ids_to_blessing" : {
         "some_property" : {
            "value" : true
         }
      }
   }
