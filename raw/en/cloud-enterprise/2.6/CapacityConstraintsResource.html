CapacityConstraintsResourceedit

Specifies the maximum and minimum capacity constraints.

Propertiesedit

max (integer as int64, required)
Max capacity
min (integer as int64, required)
Min capacity

Exampleedit

{
   "max" : 0,
   "min" : 0
}
