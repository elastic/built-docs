CoordinatorsSummaryedit

Summarized information about coordinators.

Propertiesedit

coordinators (array[CoordinatorSummary], required)
Summarized information on each coordinator

Exampleedit

{
   "coordinators" : [
      {
         "attributes" : {
            "some_property" : "string"
         },
         "client_port" : 0,
         "election_port" : 0,
         "leader_port" : 0,
         "name" : "string",
         "public_hostname" : "string"
      }
   ]
}
