MoveClustersDetails

edit

MoveClustersDetails

edit

Information about the Elasticsearch clusters, multiple Kibana instances, and multiple APM Servers that are moved off of the allocator.

Properties

edit
apm_clusters (array[MoveApmClusterDetails], required)
Detailed information about the Apm clusters being moved.
appsearch_clusters (array[MoveAppSearchDetails], required)
Detailed information about the App Search clusters being moved.
elasticsearch_clusters (array[MoveElasticsearchClusterDetails], required)
Detailed information about the Elasticsearch clusters being moved.
enterprise_search_clusters (array[MoveEnterpriseSearchDetails], required)
Detailed information about the Elastic Enterprise Search clusters being moved.
kibana_clusters (array[MoveKibanaClusterDetails], required)
Detailed information about the Kibana clusters being moved.
