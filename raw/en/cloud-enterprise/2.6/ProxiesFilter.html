ProxiesFilteredit

Selects proxies by matching a proxy metadata item.

Propertiesedit

key (string, required)
Key of the proxy metadata item to be matched
value (string, required)
Value of the proxy metadata item to be matched

Exampleedit

{
   "key" : "string",
   "value" : "string"
}
