TrustSettingsedit

Configuration of trust with other clusters.

Propertiesedit

accounts (array[AccountTrustRelationship], required)
The list of trust relationships with different accounts

Exampleedit

{
   "accounts" : [
      {
         "account_id" : "string",
         "trust_all" : true,
         "trust_whitelist" : [
            "string"
         ]
      }
   ]
}
