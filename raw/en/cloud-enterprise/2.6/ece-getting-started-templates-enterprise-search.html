Enterprise Search templateedit

This template provides the capabilities of Elastic Enterprise Search, including:

  • App Search - deliver a pre-tuned search experience in your applications
  • Workplace Search - deliver a fast and relevant search experience across your organization

To enable Enterprise Search on Elastic Cloud Enterprise, see Enable Enterprise Search.

For full Enterprise Search documentation, see the Enterprise Search Reference.

Included in this template:

  • Elasticsearch data nodes: Start at 1 GB memory x 1 availability zone. Use the data.default instance configuration.
  • Enterprise Search nodes: Start at 2 GB memory x 1 availability zone. Use the enterprise.search instance configuration.
  • Kibana: Starts at 1 GB memory x 1 availability zone. Uses the kibana instance configuration.
