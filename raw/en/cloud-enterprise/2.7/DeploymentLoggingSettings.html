DeploymentLoggingSettingsedit

The logging settings for a deployment

Propertiesedit

destination (AbsoluteRefId, required)
The destination deployment that this deployment's logs will be sent to

Exampleedit

{
   "destination" : {
      "deployment_id" : "string",
      "ref_id" : "string"
   }
}
