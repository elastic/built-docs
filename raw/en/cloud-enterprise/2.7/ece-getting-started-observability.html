Elastic Observabilityedit

This solution provides all of the data ingestion capabilities to know your infrastructure, including:

  • APM - Monitor software services and applications in real time.
  • Logs - Ingest and view a variety of logs to identify problems.
  • Metrics - Collect data from multiple sources and set custom alerts.

For the rest of documentation, see the Observability Guide.
