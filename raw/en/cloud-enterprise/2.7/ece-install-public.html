Install ECE on a public cloudedit

You can deploy ECE on any of the following cloud providers:

  • Google Cloud Platform (GCP)
  • Microsoft Azure

with one of the following operating systems:

  • Ubuntu 16.04 LTS (Xenial Xerus)
  • Red Hat Enterprise Linux (RHEL) 7 and CentOS 7
  • SUSE Linux Enterprise Server (SLES) 12

You can also use our Amazon Machine Images (AMI) that are preconfigured to install Elastic Cloud Enterprise and can be deployed from the AWS Management Console.
