List security realm configurationsedit

Retrieves a list of security realm configurations.

Requestedit

GET /api/v1/platform/configuration/security/realms

Responsesedit

200

(SecurityRealmInfoList) The security realm configurations were successfully returned

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request exampleedit

curl -XGET https://{{hostname}}/api/v1/platform/configuration/security/realms \
-H "Authorization: ApiKey $ECE_API_KEY"
