ContainerSetSummaryedit

Summarized information about a container set

Propertiesedit

container_set_id (string, required)
The identifier for this container set
containers_created_count (integer as int32, required)
Number of containers created in this container set
containers_expected_running_count (integer as int32, required)
Expected number of running containers in this container set
containers_running_count (integer as int32, required)
Number of containers running in this container set
containers_started_count (integer as int32, required)
Number of containers started in this container set
healthy (boolean, required)
Whether the container set is healthy
hidden (boolean, required)
True if the container set is hidden

Exampleedit

{
   "container_set_id" : "string",
   "containers_created_count" : 0,
   "containers_expected_running_count" : 0,
   "containers_running_count" : 0,
   "containers_started_count" : 0,
   "healthy" : true,
   "hidden" : true
}
