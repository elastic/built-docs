DeploymentMetricsSettingsedit

The metrics settings for a deployment

Propertiesedit

destination (AbsoluteRefId, required)
The destination deployment that this deployment's metrics will be sent to

Exampleedit

{
   "destination" : {
      "deployment_id" : "string",
      "ref_id" : "string"
   }
}
