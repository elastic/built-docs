Extensionsedit

A collection of extensions

Propertiesedit

extensions (array[Extension], required)
The list of extensions.

Exampleedit

{
   "extensions" : [
      {
         "deployments" : [
            "string"
         ],
         "description" : "string",
         "download_url" : "string",
         "extension_type" : "string",
         "file_metadata" : {
            "last_modified_date" : "2019-01-01T00:00:00Z",
            "size" : 0,
            "url" : "https://example.net"
         },
         "id" : "string",
         "name" : "string",
         "url" : "string",
         "version" : "string"
      }
   ]
}
