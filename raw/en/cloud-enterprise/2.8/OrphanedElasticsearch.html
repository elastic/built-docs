OrphanedElasticsearchedit

Details about an orphaned Elasticsearch resources.

Propertiesedit

dependents (array[ElasticsearchDependant], required)
List of orphaned dependent resources
id (string, required)
The id of the orphaned resource

Exampleedit

{
   "dependents" : [
      {
         "id" : "string",
         "kind" : "string"
      }
   ],
   "id" : "string"
}
