Elastic Enterprise Searchedit

This solution provides the capabilities of Elastic Enterprise Search, including:

  • App Search - delivering a pre-tuned search experience in your applications.
  • Workplace Search - delivering a fast and relevant search experience across your organization.

To enable Enterprise Search on Elastic Cloud Enterprise, see Enable Enterprise Search.

For the rest of the Enterprise Search documentation, see the Enterprise Search Reference.

Included in this solution:

  • Elasticsearch data nodes: Defaults to 1 GB memory x 1 availability zone. Use the data.default instance configuration.
  • Enterprise Search nodes: Defaults to 2 GB memory x 1 availability zone. Use the enterprise.search instance configuration.
  • Kibana: Defaults to 1 GB memory x 1 availability zone. Uses the kibana instance configuration.
