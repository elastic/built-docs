Elastic Cloud Enterprise 2.6.1edit

This version includes fixes for the following bugs and known issues:

  • Parsing rules in IP Tables defined in multiport syntax using x:x is now working.
  • Settings xpack.security.audit.logfile.events.ignore_filters.* is now accepting arrays of strings as described in the Logfile audit events ignore policies section.

To upgrade your Elastic Cloud Enterprise installation to this latest version, see Upgrade your installation.

Release date: September 3rd, 2020
