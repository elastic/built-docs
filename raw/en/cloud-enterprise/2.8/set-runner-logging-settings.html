Set runner logging settingsedit

Set the logging settings for this runner instance.

Requestedit

PUT /api/v1/platform/infrastructure/runners/{runner_id}/logging_settings

Path parametersedit

Name Type Required Description

runner_id

string

Y

The identifier for the runner

Request bodyedit

(LoggingSettings) (required) The new logging settings for the runner instance

Responsesedit

200

(LoggingSettings) The updated logging settings for the runner instance

Headers

x-cloud-resource-created (string)
The date-time when the resource was created (ISO format relative to UTC)
x-cloud-resource-last-modified (string)
The date-time when the resource was last modified (ISO format relative to UTC)
x-cloud-resource-version (string)
The resource version, which is used to avoid update conflicts with concurrent operations
404

(BasicFailedReply) The logging settings for this runner were not found. (code: runners.logging_settings.not_found)

Headers

x-cloud-error-codes (string; allowed values: [runners.logging_settings.not_found])
The error codes associated with the response

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request exampleedit

curl -XPUT https://{{hostname}}/api/v1/platform/infrastructure/runners/{runner_id}/logging_settings \
-H "Authorization: ApiKey $ECE_API_KEY" \
-d '
{
   "logging_levels" : {
      "some_property" : "string"
   }
}
'
