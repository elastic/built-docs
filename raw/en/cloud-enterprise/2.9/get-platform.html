Get platform info

edit

Retrieves information about the active platform.

Request

edit

GET /api/v1/platform

Responses

edit
200

(PlatformInfo) The platform information is retrieved.

Request example

edit
curl -XGET https://{{hostname}}/api/v1/platform \
-H "Authorization: ApiKey $ECE_API_KEY"
