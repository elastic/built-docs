SecurityRealmInfoListedit

A list of Elasticsearch security realms.

Propertiesedit

realms (array[SecurityRealmInfo], required)
The list of security realms

Exampleedit

{
   "realms" : [
      {
         "enabled" : true,
         "id" : "string",
         "name" : "string",
         "order" : 0,
         "type" : "string",
         "urls" : [
            "string"
         ]
      }
   ]
}
