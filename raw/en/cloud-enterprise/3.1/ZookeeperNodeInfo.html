ZookeeperNodeInfoedit

Information about a ZooKeeper node.

Propertiesedit

id (string, required)
Identifier of this ZooKeeper node
state (string; allowed values: [connected, suspended, reconnected, lost, read_only], required)
State of the ZooKeeper node

Exampleedit

{
   "id" : "string",
   "state" : "string"
}
