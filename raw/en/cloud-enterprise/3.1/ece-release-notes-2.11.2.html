Elastic Cloud Enterprise 2.11.2edit

The following change is included in this release.

Bug fixesedit

Add remediation scripts for platform failure due to expired certificates. This update adds emergency remediation scripts that are required to recover an Elastic Cloud Enterprise platform that has failed due to certificate expiry.

This bug fix does not include any changes to the ECE platform or components, so if you have a current ECE installation there is no need to upgrade to version 2.11.2.
