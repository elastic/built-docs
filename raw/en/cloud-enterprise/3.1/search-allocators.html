Search allocatorsedit

Retrieves the information for all the allocators in the platform that match a specified query.

Requestedit

POST /api/v1/platform/infrastructure/allocators/_search

Request bodyedit

(SearchRequest) The optional search request to execute. If not supplied then all allocators are matched

Responsesedit

200

(AllocatorOverview)

An overview of allocators that matched the given search query
