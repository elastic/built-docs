ContainersEntryOptionsAuth

edit

ZooKeeper Authentication information.

Properties

edit
auth (string, required)
Plain-text auth. For the "digest" scheme
scheme (string, required)

Scheme identifier, usually "digest"

Example: "digest"

Example

edit
{
   "auth" : "string",
   "scheme" : "string"
}
