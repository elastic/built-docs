Access the API using Elastic Cloud Controledit

The ecctl command line interface offers a convenient method to perform operations available through the Elastic Cloud Enterprise API by wrapping requests into memorable commands. Information for installing and using the tool are available in the Elastic Cloud Control documentation, which includes several usage examples to help you get started.
