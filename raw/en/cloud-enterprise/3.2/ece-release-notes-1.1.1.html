Elastic Cloud Enterprise 1.1.1

edit

New for Elastic Cloud Enterprise version 1.1.1:

  • Better control over timeouts when installing ECE over slow internet connections. We added a new configuration parameter --timeout-factor, which allows you to control timeout values during installation. If specified, increases all timeout values by multiplying with this number.

  • Bug fixes:

    • ECE 1.1.1 corrects an issue where upgrading from 1.0.2 to 1.1.0 and subsequently restarting allocators or adding more capacity could lead to authentication errors in Kibana. If you freshly installed 1.1.0 without upgrading your existing Elastic Cloud Enterprise, this bug does not affect your deployment.
    • Correctly handle an emergency vacate of an instance when a plan change has been aborted.
