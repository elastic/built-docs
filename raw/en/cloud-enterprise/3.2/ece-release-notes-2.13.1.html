Elastic Cloud Enterprise 2.13.1edit

The following change is included in this release.

Bug fixesedit

Update default stack releases. This fix updates the ECE default Elasticsearch version for system clusters to 7.16.3, together with an associated, internally-used snapshot tool. The new stack version upgrades Apache Log4j from version 2.17.0 to version 2.17.1, since the former was reported to have a security vulnerability.
