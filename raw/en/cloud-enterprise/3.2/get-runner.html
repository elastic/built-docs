Get runneredit

Retrieves a runner by id.

Requestedit

GET /api/v1/platform/infrastructure/runners/{runner_id}

Path parametersedit

Name Type Required Description

runner_id

string

Y

The identifier for the runner

Responsesedit

200

(RunnerInfo)

The information for the runner specified by {runner_id}.

404

(BasicFailedReply)

Unable to find the {runner_id} specified runner. Edit your request, then try again. (code: runners.runner_not_found)

Headers

x-cloud-error-codes (string; allowed values: [runners.runner_not_found])
The error codes associated with the response

Request exampleedit

curl -XGET https://{{hostname}}/api/v1/platform/infrastructure/runners/{runner_id} \
-H "Authorization: ApiKey $ECE_API_KEY"
