API referenceedit

This documentation applies to the Elastic Cloud Enterprise API only. If you are using Elasticsearch Service, check the Elastic Cloud API information instead.

Browse our OpenAPI-sourced RESTful API reference content to find out what endpoints we make available with Elastic Cloud Enterprise.

In some of our API section headings we use the acronym CRUD, which means that the related endpoints create, read, update, and delete information.
