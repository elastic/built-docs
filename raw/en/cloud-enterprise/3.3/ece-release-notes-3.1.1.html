Elastic Cloud Enterprise 3.1.1edit

The following change is included in this release.

Bug fixesedit

Add CLIENT_FORWARDER_OBSERVERS_TAG to runner and upgrader containers. This fixes an unresolved ECE environment variable that could prevent services from connecting to ZooKeeper. The bug could cause an upgrade from ECE version 3.0.0 to 3.1.0 to fail during upgrade of the system deployments.
