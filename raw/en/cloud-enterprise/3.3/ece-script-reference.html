Script referenceedit

The elastic-cloud-enterprise.sh script can perform several actions to install Elastic Cloud Enterprise or to modify your installation:

  • elastic-cloud-enterprise.sh install - To install Elastic Cloud Enterprise on a host, either to start a new installation or to add a host to an existing installation. The script defaults to this action, if no other action is specified.
  • elastic-cloud-enterprise.sh upgrade - To upgrade your existing Elastic Cloud Enterprise installation to a new version.
  • elastic-cloud-enterprise.sh reset-adminconsole-password - To reset the password of an administration console user that you use to log into the Cloud UI.
  • elastic-cloud-enterprise.sh add-stack-version - To download and add a new Elastic Stack version from Elastic and make it available on your Elastic Cloud Enterprise installation.
