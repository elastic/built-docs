Get instance types

edit

Retrieves a list of all instance types.

Request

edit

GET /api/v1/stack/instance-types

Responses

edit
200

(array[InstanceTypeResource])

List of node types

Request example

edit
curl -XGET https://{{hostname}}/api/v1/stack/instance-types \
-H "Authorization: ApiKey $ECE_API_KEY"
