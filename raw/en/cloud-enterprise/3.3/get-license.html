Get licenseedit

Retrieves the license.

Requestedit

GET /api/v1/platform/license

Responsesedit

200

(LicenseObject)

The information for the license.

404

(BasicFailedReply)

The license cannot be found. (code: license.license_not_found)

Headers

x-cloud-error-codes (string; allowed values: [license.license_not_found])
The error codes associated with the response

Request exampleedit

curl -XGET https://{{hostname}}/api/v1/platform/license \
-H "Authorization: ApiKey $ECE_API_KEY"
