Elastic Cloud Enterprise 3.4.1edit

The following changes are included in this release.

Enhancementsedit

Bump goboring version to 1.18.5. Update Proxy go version to 1.18.5. x509 certificates signed with the SHA-1 hash function are rejected by default. This does not apply to self-signed root certificates. Practical attacks against SHA-1 have been demonstrated since 2017 and publicly trusted Certificate Authorities have not issued SHA-1 certificates since 2015.
