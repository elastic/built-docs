BlessingsWithMetaedit

Persisted Blessings along with metadata

Propertiesedit

meta (Metadata, required)
The metadata
value (Blessings, required)
The blessings

Exampleedit

{
   "meta" : {
      "created_time" : "2019-01-01T00:00:00Z",
      "modified_time" : "2019-01-01T00:00:00Z",
      "version" : "string"
   },
   "value" : {
      "runner_ids_to_blessing" : {
         "some_property" : {
            "value" : true
         }
      }
   }
}
