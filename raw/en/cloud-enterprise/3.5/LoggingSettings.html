LoggingSettingsedit

Information about logging settings.

Propertiesedit

logging_levels (map[string,string], required)
Overridden logging levels. Keys are logging levels. Allowable values are OFF, ERROR, WARN, INFO, DEBUG, TRACE, ALL

Exampleedit

{
   "logging_levels" : {
      "some_property" : "string"
   }
}
