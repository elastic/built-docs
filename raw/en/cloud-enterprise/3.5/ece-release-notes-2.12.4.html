Elastic Cloud Enterprise 2.12.4edit

Enhancementsedit

Several tools bundled with Elastic Cloud Enterprise used a log4j library version which contained recently discovered security vulnerabilities. These vulnerabilities could not be exploited in Elastic Cloud Enterprise because users have no direct control over these tools. Nevertheless, to avoid automated security scanners generating false positives, we have updated the impacted tools to the latest version that uses a log4j library version which fixes the vulnerabilities.

Bug fixesedit

Fixes a bug which prevented Enterprise Search nodes from being added to deployments based on version 7.16.x of the Elastic Stack.
