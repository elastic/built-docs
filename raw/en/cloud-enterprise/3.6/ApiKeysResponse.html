ApiKeysResponseedit

The response model for the API keys.

Propertiesedit

keys (array[ApiKeyResponse], required)
The list of API keys.

Exampleedit

{
   "keys" : [
      {
         "creation_date" : "2019-01-01T00:00:00Z",
         "description" : "string",
         "expiration_date" : "2019-01-01T00:00:00Z",
         "id" : "string",
         "key" : "string",
         "organization_id" : "string",
         "user_id" : "string"
      }
   ]
}
