KeystoreContentsedit

The contents of the Elasticsearch keystore.

Propertiesedit

secrets (map[string,KeystoreSecret], required)
List of secrets

Exampleedit

{
   "secrets" : {
      "some_property" : {
         "as_file" : true,
         "value" : {}
      }
   }
}
