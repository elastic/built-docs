UserApiKeyedit

The model to specify a user and their API key in a delete request.

Propertiesedit

api_key_id (string, required)
The API key ID.
user_id (string, required)
The user ID.

Exampleedit

{
   "api_key_id" : "string",
   "user_id" : "string"
}
