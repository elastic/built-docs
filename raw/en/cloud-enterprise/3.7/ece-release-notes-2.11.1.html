Elastic Cloud Enterprise 2.11.1edit

The following change is included in this release.

Bug fixesedit

ECE certificate rotation doesn’t properly re-sign stack component certificates. If a customer upgraded from an Elastic Cloud Enterprise version prior to 2.6 to any version after 2.6, the initial certificate rotation process would not trigger rotation on stack components even if required.

This bug fix does not include any changes to the ECE platform or components, so if you have a current ECE installation there is no need to upgrade to version 2.11.1.
