Elastic Cloud Enterprise 2.12.1edit

The following change is included in this release.

Bug fixesedit

Upgrade to ECE 2.12.0 sets organization_id as platform for non-system=owned clusters Fixed a bug so that non-system-owned clusters upgraded to 2.12.0 now keep the organization_id the same as the account_id. This bug caused errors when attempting to run plans that triggered the creation of new cluster instances in non-system-owned clusters. There were SSL-related errors appearing in the ECE log files.
