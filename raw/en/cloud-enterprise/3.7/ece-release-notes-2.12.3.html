Elastic Cloud Enterprise 2.12.3edit

The following changes are included in this release.

Bug fixesedit

Bump timeouts and add logging. The ECE installer --timeout-factor CLI argument has been improved so that it applies to more potential timeout scenarios that can cause ECE upgrades to fail on larger installations. Logging has also been improved so that future issues of this type can be identified in both the upgrade.log and monitor.log files.
