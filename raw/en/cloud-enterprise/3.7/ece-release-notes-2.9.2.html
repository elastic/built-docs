Elastic Cloud Enterprise 2.9.2edit

This release of Elastic Cloud Enterprise contains two bug fixes.

Bug fixesedit

Fix searchable snapshot clean-up process. Fixes a bug where seachable snapshots, used to support cold data tier indices, could be deleted unintentionally due to some problematic purging logic.

Don’t include transient plan settings on system cluster plan changes. Fixes a bug where certain activities that invoked a plan on system clusters, such as automatic system cluster upgrades or configuring identity providers, would cause the previous plan’s transient section to be included. This is problematic when the previous plan included settings for moving nodes to new allocators or adding dangerous settings such as skip_data_migration and override_failsafe. In ECE 2.10 these plans are always run with an empty transient plan section.
