List Extensionsedit

Retrieves all of the available extensions.

Requestedit

GET /api/v1/deployments/extensions

Responsesedit

200

(Extensions)

The extensions that are available

Request exampleedit

curl -XGET https://{{hostname}}/api/v1/deployments/extensions \
-H "Authorization: ApiKey $ECE_API_KEY"
