Promote a coordinator candidate

edit

Promotes a coordinator candidate.

Request

edit

POST /api/v1/platform/infrastructure/coordinators/candidates/{coordinator_candidate_id}/_promote

Path parameters

edit
Name Type Required Description

coordinator_candidate_id

string

Y

The identifier for the coordinator candidate

Responses

edit
202

(CoordinatorCandidateInfo)

Accepted promote of coordinator candidate.

404

(BasicFailedReply)

Unable to find coordinator candidate {coordinator_id}. Edit your request, then try again. (code: coordinators.candidate_not_found)

Headers

x-cloud-error-codes (string; allowed values: [coordinators.candidate_not_found])
The error codes associated with the response

Request example

edit
curl -XPOST https://{{hostname}}/api/v1/platform/infrastructure/coordinators/candidates/{coordinator_candidate_id}/_promote \
-H "Authorization: ApiKey $ECE_API_KEY"
