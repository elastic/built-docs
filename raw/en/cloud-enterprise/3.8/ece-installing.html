Installing Elastic Cloud Enterprise

edit

Before you start, make sure you identify your deployment scenario and prepare your hosts.

You can get ECE up and running using the official bash script on a public cloud or on your own premises. Alternatively, you can install ECE with the Ansible playbook. The ECE Ansible playbook is a community project, supported by Elastic, aimed at installing ECE at scale.

Once you have installed ECE, check some final post-installation steps to get ready for production.

This outline pertains to troubleshooting on the container engine level. The following outline is structured according to Docker as the most common engine but is also valid for Podman, replacing out commands as needed.

In these pages we frequently refer to Docker, as its currently the most common container engine, but these instructions are generally valid for Podman as well, with podman replacing docker in commands as appropriate.
