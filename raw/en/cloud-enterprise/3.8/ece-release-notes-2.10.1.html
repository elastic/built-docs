The following changes are included in this release.

Bug fixes

edit

Only call the health API when platform version allows it. Fixes a bug in the Elastic Cloud Enterprise upgrader: When upgrading from ECE 2.5.0 (any version earlier than 2.6.0) to 2.10.0, the upgrade would fail with errors saying that the Adminconsole health API could not be accessed.
