Elastic Cloud Enterprise 3.7.2

edit

The following changes are included in this release.

Features

edit

Enabled SELinux for RHEL/Rocky Linux On Rocky Linux/RHEL 8 and 9, ECE now works with SELinux enabled (by specifying --selinux during installation).

FirewallD support in ECE ECE Installations of Rocky Linux/RHEL 8 and 9 which comply with Elastic’s recommended network settings now work with FirewallD enabled.

Enhancements

edit

Updated Zookeeper version. Internal service upgrade: Zookeeper has been upgraded from version 3.8.0 to 3.8.3, to include security vulnarability fixes .

Improved rolling plan reliability. Rolling plan execution success is improved on clusters with thousands of shards and/or indices by:

  • An initial evaluation of cluster health based on the reported Elasticsearch cluster health and its number of nodes.
  • Tying the master_timeout query parameter to timeout when checking the cluster health.

Added validation of ECE compatibility with OS and Docker/Podman. The ECE installation script now validates that the ECE version you’re installing or upgrading to is compatible with the operating system and Docker/Podman on your host.
