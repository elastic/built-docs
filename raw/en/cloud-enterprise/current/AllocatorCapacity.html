AllocatorCapacityedit

The total and used capacity of the allocator.

Propertiesedit

additional_master_bonus (number as float)
Additional bonus CPU shares available to Elasticsearch master instances
available_cpus (integer as int32)
Total number of CPUs available on this allocator
memory (AllocatorCapacityMemory, required)
The available and used memory of the allocator.

Exampleedit

{
   "additional_master_bonus" : 0.1,
   "available_cpus" : 0,
   "memory" : {
      "total" : 0,
      "used" : 0
   }
}
