ProxiesSummaryedit

Summarized information about proxies.

Propertiesedit

expected_proxies_count (integer as int32, required)
Expected number of proxies
healthy (boolean, required)
Whether all proxies are healthy
proxies (array[ProxySummary], required)
Summarized information about each proxy
proxies_count (integer as int32, required)
Total number of proxies

Exampleedit

{
   "expected_proxies_count" : 0,
   "healthy" : true,
   "proxies" : [
      {
         "healthy" : true,
         "proxy_id" : "string"
      }
   ],
   "proxies_count" : 0
}
