RegionInfoedit

Information about a region.

Propertiesedit

allocators (AllocatorsSummary, required)
Information about allocators
constructors (ConstructorOverview, required)
Information about constructors
container_sets_status (ContainerSetsSummary, required)
Information about container sets
coordinators (CoordinatorsSummary, required)
Information about coordinators
proxies (ProxiesSummary, required)
Information about proxies
region_id (string, required)
Identifier of this region
resources (PlatformResourcesSummary, required)
Information about resources
runners (RunnersSummary, required)
Information about runners
zookeeper_states (ZookeeperSummary, required)
Information about the Zookeeper state
