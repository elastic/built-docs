Install ECE with Ansibleedit

If you already use Ansible in your business for provisioning, configuration management, and application deployment, you can use our playbook to get up and running with Elastic Cloud Enterprise faster, on any cloud provider.

Our Ansible playbook ECE is available on GitHub: elastic/ansible-elastic-cloud-enterprise.
