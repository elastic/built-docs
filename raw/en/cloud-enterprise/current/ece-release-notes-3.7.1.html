Elastic Cloud Enterprise 3.7.1edit

The following changes are included in this release.

Featuresedit

New supported operating systems. You can now install Elastic Cloud Enterprise on Rocky Linux 8.

Bug fixedit

Admin Console broken SSO to Kibana is fixed. A bug that caused the Single Sign-On between the Admin Console and deployment Kibana instances to break is fixed in this release. The bug affected ECE installs after upgrading to ECE 3.7.0. It’s highly recommended to upgrade directly to this 3.7.1 ECE version. For details, refer to Admin Console SSO breaks in ECE 3.7.0.
