Get constructorsedit

Retrieves an overview of all constructors in an ECE installation.

Requestedit

GET /api/v1/platform/infrastructure/constructors

Responsesedit

200

(ConstructorOverview)

An overview of all constructors.

Request exampleedit

curl -XGET https://{{hostname}}/api/v1/platform/infrastructure/constructors \
-H "Authorization: ApiKey $ECE_API_KEY"
