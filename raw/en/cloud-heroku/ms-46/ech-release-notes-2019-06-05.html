Snapshot repository support for GCS and private subscription tier availableedit

Elasticsearch Add-On for Heroku now supports additional features that let you do more with our hosted offering:

  • Snapshot repository support for Google Cloud Storage (GCS). You can now configure a custom repository using your GCS account to take or restore snapshots. To learn more, see Configure a snapshot repository using GCS.
  • Private subscription tier. You can now run Elasticsearch Add-On for Heroku in a dedicated VPC with a secure connection from your environment using various peering options. For more information, see Elasticsearch Add-On for Heroku Subscriptions.

Service release: June 5, 2019 (Milestone 21)
