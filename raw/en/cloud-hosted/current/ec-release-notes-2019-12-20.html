GCP region North America Northeast 1 (Montreal) available

edit

Elastic Cloud is now available in the GCP region North America Northeast 1 (Montreal). You can select the new region when you create new deployments.

Like other recently added regions, Montreal only supports HTTPS access for additional security.

Service release: December 20, 2019
