GCP region US East 1 (South Carolina) available

edit

Elastic Cloud is now available in the GCP region US East 1 (South Carolina). You can select the new region when you create new deployments.

Like other recently added regions, South Carolina supports only HTTPS access for additional security.

Service release: April 9th, 2020
