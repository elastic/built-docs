Our Elastic Cloud API is generally available

edit

After going through an extended beta, our RESTful API for Elastic Cloud is generally available! A public API had been a common request from you and now it’s here.

Benefits of the API:

  • Create and manage your Elastic Stack deployments
  • Automate common tasks without having to go through the {console}
  • Integrate Elastic Cloud into CI/CD pipelines for automated regression testing
  • Expose Elastic Cloud internally on your company portal
  • And much more!

Love the API already but really want a command line tool? Elastic Cloud Control (ecctl), our CLI for Elastic Cloud products, now also supports the Elastic Cloud API, offering a user friendly command line interface.

We’ve got an introduction to our new API, how-to examples you can adapt for your own use case, and an extensive API reference all ready to go.

Service release: May 12, 2020
