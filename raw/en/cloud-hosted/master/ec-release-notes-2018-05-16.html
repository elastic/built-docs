New Cloud Console and bug fixes released

edit

An all-new visual console UI design is now available, based on the Elastic Cloud Enterprise user interface. As before, the new console is your window into your managed Elasticsearch clusters and Kibana instances. You can spin up new deployments, scale them out, visualize your data with Kibana, manage your Elastic Cloud account, and much more.

The new console uses the Elastic UI Framework (EUI), which is a design library with improved usability and accessibility that ensures consistency for future UI updates across all Elastic products. Visually, our new console provides high contrast, color-blind safe palettes, and the correct aria labels to improve accessibility. If you want to learn more about the reasons behind this new user interface, check The New Elastic Cloud Console.

To experience the new UI design, log in to the {console}. You can continue to use the old console until May 25, 2018. On May 25, access to the old user console will be removed.

If you are a Heroku user, the new design is not available to you, but you will continue to have access to the old console.

Service release: May 16, 2018
