GCP region Asia Pacific Southeast 1 (Sydney) available

edit

Elastic Cloud is now available in the GCP region AP (Sydney), also known as australia-southeast1. You can select the new region when you create new deployments.

Like other recently added regions, GCP (Sydney) supports only HTTPS access for additional security.

Service release: September 10, 2019
