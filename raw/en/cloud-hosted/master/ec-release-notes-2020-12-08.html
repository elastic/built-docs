AWS region Asia Pacific (Hong Kong) available

edit

Elastic Cloud is now available in the AWS region Asia Pacific (Hong Kong). You can select the new region when you create new deployments.

Like other recently added regions, Hong Kong supports only HTTPS access for additional security.

Service release: October 1, 2020
