GCP region Middle East West 1 (Tel Aviv) now available

edit

Elastic Cloud is now available in the GCP ME West 1 (Tel Aviv) region. You can select the new regions when you create new deployments.

Like other recently added regions, Tel Aviv support only HTTPS access for additional security.
