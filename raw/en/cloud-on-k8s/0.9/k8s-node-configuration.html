Node configurationedit

Any setting defined in the elasticsearch.yml configuration file can also be defined for each topology of nodes in the spec.nodes[?].config section.

spec:
  nodes:
  - nodeCount: 3
    config:
      node.master: true
      node.data: false
      node.ingest: false
      node.ml: false
      xpack.ml.enabled: true
      cluster.remote.connect: false
  - nodeCount: 10
    config:
      node.master: false
      node.data: true
      node.ingest: true
      node.ml: true
      cluster.remote.connect: false

For more information on Elasticsearch settings, see Configuring Elasticsearch.
