Check out the samplesedit

You can find a set of sample resources in the project repository.

For a full description of each CustomResourceDefinition (CRD), refer to the API Reference or view the CRD files in the project repository. You can also retrieve information about a CRD from the cluster. For example, describe the Elasticsearch CRD specification with:

kubectl describe crd elasticsearch
