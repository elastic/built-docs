Overviewedit

Built on the Kubernetes Operator pattern, Elastic Cloud on Kubernetes (ECK) extends the basic Kubernetes orchestration capabilities to support the setup and management of Elasticsearch, Kibana and APM Server on Kubernetes.

With Elastic Cloud on Kubernetes you can streamline all those critical operations, such as:

  1. Managing and monitoring multiple clusters
  2. Scaling cluster capacity up and down
  3. Changing cluster configuration
  4. Scheduling backups
  5. Securing clusters with TLS certificates
  6. Setting up hot-warm-cold architectures with availability zone awareness
