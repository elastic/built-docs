1.8.0 release highlightsedit

New and notableedit

New and notable changes in version 1.8.0 of Elastic Cloud on Kubernetes. See Elastic Cloud on Kubernetes version 1.8.0 for the full list of changes.

ECK Helm Chart graduating from beta to GAedit

The ECK operator Helm chart is now generally available. It provides an easy way to install and configure the operator along with its Custom Resource Definitions.

OpenShift 3.11 supportedit

The ECK 1.x major release series is the last to support OpenShift 3.11. Starting ECK 2.0, OpenShift 3.11 won’t be supported anymore.

Bug fixesedit

Several bugs have been fixed in this release, including Elastic Agent certificates handling and inconsistent Elasticsearch Pod PreStop hook.
