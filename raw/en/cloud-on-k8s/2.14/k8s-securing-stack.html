Secure the Elastic Stackedit

You can secure your Elastic Stack by specifying the following settings:

  • Custom HTTP certificate
  • Users and roles
  • Rotate auto-generated credentials
  • SAML Authentication

You can use Elastic Stack configuration policy to configure the following authentication methods:

  • LDAP using Elastic stack configuration policy
  • OIDC using Elastic stack configuration policy
  • JWT using Elastic Stack configuration policy
