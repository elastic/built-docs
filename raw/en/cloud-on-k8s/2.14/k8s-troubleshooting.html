If you are an existing Elastic customer with a support contract, you can create a ticket in the Elastic Support Portal and provide the following information so that we can help you.

  • Run eck-diagnostics
  • Elasticsearch diagnostics package

Alternatively, or if you do not have a support contract, and if you are unable to find a solution to your problem with the information provided in these documents, ask for help:

  • ECK Discuss forums to ask any question
  • Github issues for bugs and feature requests
