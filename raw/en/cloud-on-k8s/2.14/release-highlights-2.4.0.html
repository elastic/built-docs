2.4.0 release highlightsedit

New and notableedit

New and notable changes in version 2.4.0 of Elastic Cloud on Kubernetes. Check Elastic Cloud on Kubernetes version 2.4.0 for the full list of changes.

Introducing Helm Charts for ECK-managed resourcesedit

Starting from ECK 2.4.0, a Helm chart is available for managing Elastic Stack resources using the ECK Operator. It is available from the Elastic Helm repository and can be used to deploy Elasticsearch and Kibana:

Elastic Agent CRD is no longer experimentaledit

With the improvements made in this and multiple previous releases, Elastic Agent CRD is not considered an experimental feature anymore. Note that the APIVersion stays at v1alpha1 to avoid introducing upgrade friction to current users. Check docs for more information.
