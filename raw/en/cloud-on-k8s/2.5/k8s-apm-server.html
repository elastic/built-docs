Run APM Server on ECKedit

As of version 8.0.0, the standalone APM Server binary is deprecated and will be removed in a future release. Consider using Elastic Agent in Fleet-managed mode with ECK. Refer to the Elastic APM integration for additional details.

This section describes how to deploy, configure and access an APM Server with ECK.

  • Advanced configuration

    • Reference an existing Elasticsearch cluster
    • TLS Certificates

  • Connect to the APM Server

    • APM Server service
    • APM Server secret token
