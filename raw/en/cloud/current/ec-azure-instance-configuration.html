Elasticsearch Service Azure instance configurationsedit

Azure Ddv4, Edsv4, Fsv2, and Lsv3 virtual machines (VM) are now available for Elastic Cloud deployments in all supported Azure Cloud regions. These VMs provide additional combinations of compute, memory, and disk configurations to better fit your use-cases to optimize performance and cost.
