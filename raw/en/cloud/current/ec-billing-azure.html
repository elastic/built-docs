Azure Marketplace (deprecated)edit

Elasticsearch is now even more closely integrated with Azure.

For new accounts, Elasticsearch is now offered as a native service directly from the Azure console. This deep integration provides consolidated billing with your Azure bill, simplified data ingestion for Azure logs and metrics, support for Private Link, and more. As a result of this new integration, Elasticsearch Service can now be deployed natively, directly in the Azure portal.

You can create Elasticsearch resources in Azure right now, or learn more about the integration in our Native Azure integration documentation.

For existing accounts, nothing changes. If you already have an active Azure Marketplace account, you can continue to create new deployments and you’ll be billed for your usage just as before. You can log into your account through the Elasticsearch Service Console. To view and manage your marketplace subscriptions, visit the Azure website.
