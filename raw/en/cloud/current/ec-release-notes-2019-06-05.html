Snapshot repository support for GCS and private subscription tier availableedit

Elasticsearch Service now supports additional features that let you do more with our hosted offering:

  • Snapshot repository support for Google Cloud Storage (GCS). You can now configure a custom repository using your GCS account to take or restore snapshots. To learn more, check Configure a snapshot repository using GCS.
  • Private subscription tier. You can now run Elasticsearch Service in a dedicated VPC with a secure connection from your environment using various peering options. For more information, check Elasticsearch Service Subscriptions.

Service release: June 5, 2019 (Milestone 21)
