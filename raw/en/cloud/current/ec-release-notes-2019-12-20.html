GCP region North America Northeast 1 (Montreal) availableedit

Elasticsearch Service is now available in the GCP region North America Northeast 1 (Montreal). You can select the new region when you create new deployments.

Like other recently added regions, Montreal only supports HTTPS access for additional security. To learn more about what access different regions support, check Elasticsearch Service regions.

Service release: December 20, 2019
