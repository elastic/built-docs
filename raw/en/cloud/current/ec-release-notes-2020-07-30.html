AWS region EU West 3 (Paris) availableedit

Elasticsearch Service is now available in the AWS region EU West 3 (Paris). You can select the new region when you create new deployments.

Like other recently added regions, EU West 3 Paris supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: July 30, 2020
