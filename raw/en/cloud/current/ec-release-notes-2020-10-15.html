Azure region Australia East (New South Wales) availableedit

Elasticsearch Service is now available in the Azure region Australia East (New South Wales). You can select the new region when you create new deployments.

Like other recently added regions, New South Wales supports only HTTPS access for additional security. To learn more about what access different regions support, check Elasticsearch Service regions.

Service release: October 15, 2020
