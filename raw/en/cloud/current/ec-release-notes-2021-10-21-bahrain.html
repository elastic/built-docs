AWS region Middle East (Bahrain) availableedit

Elasticsearch Service is now available in the AWS region Middle East (Bahrain). You can select the new region when you create new deployments.

Like other recently added regions, Bahrain supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.
