Enhancements and bug fixesedit

The following changes are included in this release.

Featuresedit

xpack.apm.latestAgentVersionsUrl added to 8.8. The APM configuration setting xpack.apm.latestAgentVersionsUrl is now included in 8.8.

Enhancementsedit

Show release notes links in the upgrade modal. Links to release notes are now provided in the Upgrade Modal for the selected version.

Table view instances. Added a table view for viewing an individual deployment’s instances. You can find it on the right side of the filter bar.

Log manipulated X-Opaque-Id header for Kibana requests. Add Kibana execution context derived from X-Opaque-Id header to Proxy request logs.

Register clusters read dependency on clone. When cloning a deployment, Elasticsearch Service will now register a read dependency on the snapshot repository of the source repository to ensure searchable snapshots in the new clone continue to work even if the original cluster is deleted.

Docsedit

Add RBAC updates to ESS documentation. With RBAC (Role Based Access Control) Elastic Cloud users are now able to control access for their Elastic Cloud organizations and deployments.
