Enhancements and bug fixesedit

The following changes are included in this release.

Enhancementsedit

Disable overriding internal cloud provider keystore entries. Users are no longer allowed to override Elastic managed keystore entries.

Bug fixesedit

Return an error code and message for invalid Azure Private Link Connection IDs. An Azure Private Link Connection ID that does not parse correctly now returns a 400 error code with an "invalid input" message.

Fix validation on parsed Azure Private Link connection name. Fixed validation on truncated Azure Private Link connection names to avoid erroneously rejecting valid connection requests.

Breaking changesedit

Deployment editors can now delete deployments. We’ve made a change to deployment editor privileges. Deployment editors now have privileges to delete the deployments that they have access to.
