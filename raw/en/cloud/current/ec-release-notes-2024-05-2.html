Enhancements and bug fixes - Early June 2024edit

The following changes are included in this release.

Featuresedit

Add paging support to deployments search by the use of an opaque cursor. The Search Deployments API now returns a cursor field that can be used in the subsequent search to get the next batch of deployments.

Enhancementsedit

Add Firehose config parameter set_es_document_id. Allow users to add a set_es_document_id parameter when configuring Firehose in AWS. This allows Elasticsearch to assign each document a random ID, in order to help performance.
