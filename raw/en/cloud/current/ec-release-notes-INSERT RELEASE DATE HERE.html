Milestone 110.0 releaseedit

The following changes are included in this release.

Featuresedit

Add byok usage to feature usage. Show subscription level unmatched banner for non-compliant BYOK deployment customer

Primitive vacate byok instance with skip data migration. Allow primitive vacate (dataloss) for BYOK deployment with unhealthy instance

Enhancementsedit

Incident Action item: Stop Adminconsole when region actors fail to signal their initialization at time. - Improved robustness of API service initialization sequence.

Promote ECE release 3.7.2. Promote ECE 3.7.2 release. ECE release upgrades zookeeper from 3.8.0 → 3.8.3

[Cloud Security] Update anticipated assets rule. Update the anticipated asset rule to verify we are not overbilling by adding lt filter.

[Cloud Security] Update cnvm anticipated assets. Update the alert rule that notifies about the expected number of billable CNVM assets in the Cloud Security live environment.

feat: allow one dedicated master per zone in ess. Validate the size of dedicated masters to only allow one per zone in ESS.

Allowlist integration_assistant feature flag. Adding xpack.integration_assistant.enabled kibana feature flag to enable the Integration Assistant feature.

Make command pattern share ask timeout as deadline for traced actor. Substantial internal refactoring for improving consistency on timeout values for asynchronous tasks issued while processing an api request. This improves reliability of the API by enabling graceful degradation of service and improved efficiency during times of peak load.

[Cloud Security] Promote rules to production. After reviewing rules in QA, this pull request promote them to production env.

Delete production assets for ML to migrate to terraform. Follow up of https://github.com/elastic/cloud/pull/128683. This PR removes managed ML assets in staging environment to correspond to migration to Terraform.

[Cloud Security] update slack connector. Update slack connector id - QA env.

Delete dev and staging assets for ML to migrate to terraform. This PR removes managed ML assets in staging environment to correspond to migration to Terraform https://github.com/elastic/kibana-serverless-o11y-terraform/pull/126

[Cloud Security] update metering detection rules. Update the queried index pattern for the cloud security rules

solves: - https://github.com/elastic/security-team/issues/9478

[Cloud Security] Alerts and connector - QA. Alerts and connectors for cloud security QA usage records.

[IAM] Set .session.lifespan to 24h by default for Cloud SSO SAML realm. This sets the session.lifespan Kibana setting in the Cloud SSO realm to 24h by default.

Align operational contacts with the Cloud RBAC model. Operational email recipients have been updated: - If operational contacts are defined for the organization, then operational emails will be sent to both the operational contacts and the organization owners. - If operational contacts are not defined for the organization, then operational emails will be sent to all organization users.

Billing email recipients have been updated: - If billing contacts are defined for the organization, then billing emails will be sent to both the billing contacts and the organization owners. - If billing contacts are not defined for the organization, then billing emails will be sent to all organization users.

CP-5238 Validate ECE compatibility with OS and Docker/Podman in elastic-cloud-enterprise.sh. The Elastic Cloud Enterprise installation script validates that the ECE version you’re installing or upgrading to is compatible with the operating system and Docker/Podman on your host.

Bug fixesedit

Change dot in cloudwatch metric name to underscore. Replace dot in cloudwatch metric name with underscore.

Update proxy/route-sync to go 1.22.4. Update proxy and route-sync to go 1.22.4

Calculate average for CloudWatch metrics. This PR is to add calculation of average for both OTEL1.0 format and JSON format CloudWatch metrics.

Remove Update TrafficFilters permission for Editors. Remove Update TrafficFilters permission for Editors

Non_issuesedit

Promote ECE release 3.7.2. Promote ECE 3.7.2 release. ECE release upgrades zookeeper from 3.8.0 → 3.8.3
