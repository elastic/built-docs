Access the API using Elastic Cloud Controledit

The ecctl command line interface offers a convenient method to perform operations available through the Elasticsearch Service API by wrapping requests into memorable commands. For more information on how to install and use the tool, check the Elastic Cloud Control documentation, which includes several usage examples to help you get started.
