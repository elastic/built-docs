Access the API using the Elastic Cloud Terraform provideredit

The Elastic Cloud Terraform provider offers another convenient way for you to automate the provisioning and management of your Cloud deployments, by acting as a bridge to the Elasticsearch Service API. Check our Elastic Cloud Provider documentation for descriptions of the available provider resource types and data sources.
