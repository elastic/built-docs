User privilegesedit

Within the same organization, all members share the same set of permissions. Any user can:

  • Access all deployments within the organization
  • Generate API keys
  • Handle deployment extensions
  • Handle traffic filtering
  • Configure trust management
  • View and manage billing
  • Manage contacts
  • Invite new users
  • Remove existing users from the organization
  • Leave the organization, unless they are the last user in the organization and they have active deployments or pending bills.
