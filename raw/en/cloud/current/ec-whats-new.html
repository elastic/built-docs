What’s new with the Elastic Stackedit

As soon as the latest Elastic Stack version is released, we make it available to you.

From the Elasticsearch Service Console, you can upgrade your existing deployments to the latest versions of the Elastic Stack. To learn more about the best practices to use when upgrading, refer to Upgrade versions.

Check our Version policy to learn about when and how new Elastic Stack versions are made available.
