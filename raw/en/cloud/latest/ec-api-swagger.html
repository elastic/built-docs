API reference documentationedit

The following endpoints are available in the Elasticsearch Service RESTful API:

  • APM routes - Reset secret tokens for your APM resources.
  • Accounts - Retrieve and update the current Elasticsearch Service account.
  • App Search - Read-only - Enable read-only mode on an App Search resource.
  • Authentication - Manage your Elasticsearch Service API keys.
  • Billing Costs Analysis - Get an overview of your costs by organization ID.
  • Deployment - CRUD - Perform CRUD operations on your deployments.
  • Deployment - Commands - Manage resources in your deployments.
  • Deployment - Info - Retrieve information about resources in your deployments.
  • Deployment - Remote Clusters - Retrieve and set remote clusters.
  • Deployment Resources - CRUD - Manage an Elasticsearch keystore, restore a shutdown resource, and cancel a pending plan.
  • Deployments - Extensions - Manage the extensions for your deployment.
  • Deployments - Templates - Retrieve available deployment templates.
  • Deployments - Traffic Filter - Associations - Retrieve and set associations between your deployments and traffic filter rulesets.
  • Deployments - Traffic Filter - CRUD - Perform CRUD operations on your traffic filter rulesets.
  • Stack - Versions - CRUD - Retrieve the available Elastic Stack versions for your deployments.
  • Organizations - Manage organization invitations and memberships.
  • Definitions - Check the properties and examples for the API endpoint requests and responses.
