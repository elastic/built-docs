AWS region Pacific (Seoul) availableedit

Elasticsearch Service is now available in the AWS region Pacific (Seoul). You can select the new region when you create new deployments.

Like other recently added regions, Pacific Seoul supports only HTTPS access for additional security. To learn more about what access different regions support, check Elasticsearch Service regions.

Service release: August 11, 2020
