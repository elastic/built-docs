Azure region Southcentral US (Texas) availableedit

Elasticsearch Service is now available in the Azure region Southcentral US (Texas). You can select the new region when you create new deployments.

Like other recently added regions, Virginia supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.
