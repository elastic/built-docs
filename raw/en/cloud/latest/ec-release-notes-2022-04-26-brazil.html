Azure region Brazil South (São Paulo) now availableedit

Elasticsearch Service is now available in the Azure Brazil South (São Paulo) region. You can select the new region when you create new deployments.

Like other recently added regions, São Paulo supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.
