Azure region Canada Central (Toronto) now availableedit

Elasticsearch Service is now available in the Azure Canada Central (Toronto) region. You can select the new region when you create new deployments.

Like other recently added regions, Toronto supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.
