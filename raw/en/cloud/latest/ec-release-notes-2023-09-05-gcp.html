GCP regions Asia Southeast 2 (Indonesia) and Europe West 9 (Paris) available

edit

Elasticsearch Service is now available in these GCP regions:

  • Asia Southeast 2 (Indonesia)
  • Europe West 9 (Paris)

You can select the new regions when you create new deployments.

Like other recently added regions, these support only HTTPS access for additional security. To learn more about what access different regions support, check Elasticsearch Service regions.
