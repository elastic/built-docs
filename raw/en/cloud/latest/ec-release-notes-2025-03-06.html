Enhancements and bug fixes - March 2025

edit

The following changes are included in this release.

Enhancements

edit

Upgraded beats to 8.17.2. Upgraded metricbeat and filebeat for allocator-metricbeat and beats-runner to 8.17.2.

Bug fixes

edit

Fixed an issue with beats-runner 8.17.0 image. The beats-runner 8.17.0 image could not properly start up beats services due to improper command-line flags.
