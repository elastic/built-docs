Azure region US Central (Iowa) availableedit

Elasticsearch Service is now available in the Azure region US Central (Iowa). You can select the new region when you create new deployments.

Like other recently added regions, Iowa supports only HTTPS access for additional security. To learn more about what access different regions support, check Elasticsearch Service regions.

Service release: October 2, 2020
