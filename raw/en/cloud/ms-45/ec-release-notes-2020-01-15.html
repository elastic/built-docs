GCP region Asia Pacific South 1 (Mumbai) available

Elasticsearch Service is now available in the GCP region Asia Pacific South 1 (Mumbai). You can select the new region when you create new deployments.

Like other recently added regions, Mumbai supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: January 15, 2020
