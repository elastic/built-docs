Azure region France Central (Paris) availableedit

Elasticsearch Service is now available in the Azure region France Central (Paris). You can select the new region when you create new deployments.

Like other recently added regions, Paris supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: June 17, 2020
