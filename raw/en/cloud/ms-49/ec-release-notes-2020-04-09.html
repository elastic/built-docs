GCP region US East 1 (South Carolina) availableedit

Elasticsearch Service is now available in the GCP region US East 1 (South Carolina). You can select the new region when you create new deployments.

Like other recently added regions, South Carolina supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: April 9th, 2020
