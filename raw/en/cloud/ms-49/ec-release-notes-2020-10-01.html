AWS region Asia Pacific (Mumbai) availableedit

Elasticsearch Service is now available in the AWS region Asia Pacific (Mumbai). You can select the new region when you create new deployments.

Like other recently added regions, Mumbai supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: October 1, 2020
