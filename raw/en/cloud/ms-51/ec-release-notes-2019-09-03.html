Elasticsearch Service on Microsoft Azure availableedit

Elasticsearch Service on Elastic Cloud now supports deployments on Microsoft Azure. You can select the new cloud platform when you create new deployments. Refer to Elasticsearch Service regions to see which Azure regions are currently supported.

To find out more, take a look at Elasticsearch Service on Elastic Cloud now available on Microsoft Azure.

Service release: September 03, 2019
