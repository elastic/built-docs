AWS region Canada (Central) availableedit

Elasticsearch Service is now available in the AWS region Canada (Central). You can select the new region when you create new deployments.

Like other recently added regions, Canada Central supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: June 30, 2020
