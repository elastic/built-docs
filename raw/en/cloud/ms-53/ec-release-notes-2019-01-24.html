Support for APM Servers, automatic multi-factor authentication, and more features releasededit

Elasticsearch Service now supports additional features that let you do more with our hosted offering:

  • APM Server now available with Elasticsearch Service deployments. When you create or edit a deployment, you can stand up an APM Server and get the secret token to configure your APM Agents. The APM Server is already hooked up to work with Elasticsearch and Kibana. Learn more …​
  • Multi-factor authentication enables automatically. When you add your first MFA device, instead of having to also enable MFA to set it up, now it gets enabled for you. Learn more …​
  • Tell us more about yourself. When you decide to take the plunge and add a credit card to your Elasticsearch Service account, we want to get to know you better and we’ll be asking for a bit more information. We use this information to design our products around your needs. Help us, help you.

Service release: January 24, 2019 (Milestone 15)
