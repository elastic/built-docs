Elasticsearch Service now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 7.4.1.

For more information about the features in Elasticsearch and Kibana 7.4.1, see:

  • Elasticsearch 7.4.1 Release Notes
  • Kibana 7.4.1 Release Notes
