Elastic Stack 7.4.0 released

Elasticsearch Service now supports the latest version of the Elastic Stack, which includes Elasticsearch, Kibana and APM 7.4.0.

For more information about the features in Elasticsearch, Kibana and APM 7.4.0, see:

  • Elasticsearch 7.4.0 Release Notes
  • Kibana 7.4.0 Release Notes
  • APM 7.4.0 Release Notes

Service release: October 1, 2019
