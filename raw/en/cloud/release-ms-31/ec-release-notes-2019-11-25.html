Azure regions US West 2 (Washington) and Southeast Asia (Singapore) available

Elasticsearch Service is now available in the Azure region US West 2 (Washington) and Southeast Asia (Singapore). You can select the new region when you create new deployments.

Like other recently added regions, Washington and Singapore support only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.

Service release: November 25, 2019
