Elastic Stack 6.3.2 released

Elasticsearch Service now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 6.3.2.

For information about the features in Elasticsearch and Kibana 6.3.2, see:

  • Elasticsearch 6.3.2 Release Notes
  • Kibana 6.3.2 Release Notes
