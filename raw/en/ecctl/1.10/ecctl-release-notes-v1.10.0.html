Elastic Cloud Control (ecctl) v1.10.0edit

What’s new | Bug fixes | Breaking changes | Changelog

Welcome to the v1.10.0 release of Elastic Cloud Control. This version brings new features, some breaking changes, and bug fixes.

Elastic Cloud Control (ecctl) is Elastic’s CLI to manage Elasticsearch Service (ESS) and Elastic Cloud Enterprise (ECE).
