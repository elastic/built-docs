Elastic Cloud Control (ecctl) v1.7.1edit

Changelog

Welcome to the v1.7.1 release of Elastic Cloud Control. This version was released to fix an issue which resulted in the 1.7.0 release to be missing from the downloads. No code changes were made to ecctl itself.

Elastic Cloud Control (ecctl) is Elastic’s CLI to manage Elasticsearch Service (ESS) and Elastic Cloud Enterprise (ECE).
