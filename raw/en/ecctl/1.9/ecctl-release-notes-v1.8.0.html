Elastic Cloud Control (ecctl) v1.8.0edit

Changelog

Welcome to the v1.8.0 release of Elastic Cloud Control. This version brings new features, some breaking changes, and bug fixes.

Elastic Cloud Control (ecctl) is Elastic’s CLI to manage Elasticsearch Service (ESS) and Elastic Cloud Enterprise (ECE).
