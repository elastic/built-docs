New modules for Azure cloud monitoring

edit

With the addition of Metricbeat and Filebeat modules for Azure monitoring, Azure users can now directly monitor logs and metrics from Azure Event Hub and Azure Monitor and use prebuilt Kibana dashboards to speed up the analysis.

Heartbeat enhanced with hint-based autodiscover

edit

In 7.5, we’ve enhanced Heartbeat, as part of our Uptime solution, to include hint-based autodiscover, which is a particularly great fit for monitoring the health of Kubernetes services.
