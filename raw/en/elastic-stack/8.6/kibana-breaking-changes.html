Kibana breaking changesedit

This list summarizes the most important breaking changes in Kibana 8.6.2. For the complete list, go to Kibana breaking changes.

There are no breaking changes in Kibana 8.6.2. There are no breaking changes in Kibana 8.5.1.

Removes the console.ssl setting

Details
The console.ssl setting has been removed. For more information, refer to #123754.

Impact
Before you upgrade to 8.0.0, remove console.ssl from kibana.yml.
