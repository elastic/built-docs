--certificateedit

Summaryedit

Allows the use of a specified CA certificate file to validate the SSL certificate used by Elasticsearch.

Flagsedit

  • --certificate Path to certificate to use for SSL validation.

This flag must come before any command.

Exampleedit

Connect to a cluster at https://example.com/ via SSL and verify the certificate with the provided CA certificate file:

curator --host example.com --port 443 --use_ssl --certificate /path/to/cacert.pem <<command>> <<flags>>
