stateedit

- filtertype: state
  state: SUCCESS
  exclude: False

Empty values and commented lines will result in the default value, if any, being selected. If a setting is set, but not used by a given filtertype, it may generate an error.

This filtertype will iterate over the actionable list and match snapshots based on the value of state. They will remain in, or be removed from the actionable list based on the value of exclude.

Required settingsedit

  • state (required)

Optional settingsedit

  • exclude (default is False)
