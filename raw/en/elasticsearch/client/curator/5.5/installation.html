Installationedit

Curator can be installed in a variety of ways, depending on what meets your needs. It is important to note that Curator only requires access to a client node in the Elasticsearch cluster to work. It does not need to be installed on one of the nodes in the cluster.

If you plan on using SSL/TLS to connect to Elasticsearch, you may encounter issues with older versions of Python and outdated libraries. The binary packages have been compiled with up-to-date versions for your convenience. See the security page for more information.
