period_type

edit

This setting is only used with the period filtertype

 - filtertype: period
   period_type: absolute
   source: name
   timestring: '%Y.%m.%d'
   unit: months
   date_from: 2017.01
   date_from_format: '%Y.%m'
   date_to: 2017.01
   date_to_format: '%Y.%m'

The value for this setting must be either relative or absolute. The default value is relative.
