allocation_typeedit

This setting is used only when using the allocation action

action: allocation
description: "Apply shard allocation filtering rules to the specified indices"
options:
  key: ...
  value: ...
  allocation_type: ...
filters:
- filtertype: ...

The value of this setting must be one of require, include, or exclude.

Read more about these settings at http://www.elastic.co/guide/en/elasticsearch/reference/6.8/shard-allocation-filtering.html

The default value for this setting is require.
