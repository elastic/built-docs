Unfreeze

edit
action: unfreeze
description: "unfreeze selected indices"
options:
  continue_if_exception: False
filters:
- filtertype: ...

Empty values and commented lines will result in the default value, if any, being selected. If a setting is set, but not used by a given action, it will be ignored.

This action unfreezes the selected indices. In order to use this action, it is required to have at least a Basic license. See Subscriptions.

Optional settings

edit
  • ignore_empty_list
  • timeout_override
  • continue_if_exception
  • disable_action
