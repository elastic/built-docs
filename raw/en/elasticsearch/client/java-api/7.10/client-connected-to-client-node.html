Connecting a Client to a Coordinating Only Nodeedit

You can start locally a Coordinating Only Node and then simply create a TransportClient in your application which connects to this Coordinating Only Node.

This way, the coordinating only node will be able to load whatever plugin you need (think about discovery plugins for example).
