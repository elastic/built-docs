Supported APIsedit

The Java High Level REST Client supports the following APIs:

Single document APIs
  • Index API
  • Get API
  • Delete API
  • Update API
Multi document APIs
  • Bulk API
Search APIs
  • Search API
  • Search Scroll API
  • Clear Scroll API
Miscellaneous APIs
  • Info API
