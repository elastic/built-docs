Security APIsedit

The Java High Level REST Client supports the following Security APIs:

  • Put User API
  • Enable User API
  • Disable User API
  • Change Password API
  • Delete Role API
  • SSL Certificate API
  • Put Role Mapping API
  • Delete Role Mapping API
