Watcher APIsedit

The Java High Level REST Client supports the following Watcher APIs:

  • Start Watch Service API
  • Stop Watch Service API
  • Put Watch API
  • Get Watch API
  • Delete Watch API
  • Deactivate Watch API
  • Ack Watch API
  • Activate Watch API
  • Execute Watch API
  • Watcher Stats API
