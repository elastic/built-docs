Indices APIsedit

The Java High Level REST Client supports the following Indices APIs:

Index Management
  • Analyze API
  • Create Index API
  • Delete Index API
  • Indices Exists API
  • Open Index API
  • Close Index API
  • Shrink Index API
  • Split Index API
  • Refresh API
  • Flush API
  • Flush Synced API
  • Clear Cache API
  • Force Merge API
  • Rollover Index API
  • Update Indices Settings API
  • Get Settings API
  • Validate Query API
  • Get Index API
Mapping Management
  • Put Mapping API
  • Get Mappings API
  • Get Field Mappings API
Alias Management
  • Index Aliases API
  • Exists Alias API
  • Get Alias API
Template Management
  • Get Templates API
  • Templates Exist API
  • Put Template API
