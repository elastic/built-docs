Rollup APIsedit

The Java High Level REST Client supports the following Rollup APIs:

  • Create or update rollup job API
  • Start Rollup Job API
  • Stop Rollup Job API
  • Delete Rollup Job API
  • Get Rollup Job API
  • Rollup Search API
  • Get Rollup Capabilities API
  • Get Rollup Index Capabilities API
