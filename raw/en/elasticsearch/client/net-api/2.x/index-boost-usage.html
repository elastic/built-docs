Index Boost Usageedit

Fluent DSL exampleedit

s => s
.IndicesBoost(b => b
    .Add("index1", 1.4)
    .Add("index2", 1.3)
)

Object Initializer syntax exampleedit

new SearchRequest<Project>
{
    IndicesBoost = new Dictionary<IndexName, double>
    {
            { "index1", 1.4 },
            { "index2", 1.3 }
    }
}
