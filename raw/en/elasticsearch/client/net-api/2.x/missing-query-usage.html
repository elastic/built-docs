Missing Query Usage

edit

Fluent DSL example

edit
q
.Missing(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
    .NullValue()
    .Existence()
)

Object Initializer syntax example

edit
new MissingQuery
{
    Name = "named_query",
    Boost = 1.1,
    Field = "description",
    NullValue = true,
    Existence = true
}

Example json output.

{
  "missing": {
    "_name": "named_query",
    "boost": 1.1,
    "existence": true,
    "field": "description",
    "null_value": true
  }
}
