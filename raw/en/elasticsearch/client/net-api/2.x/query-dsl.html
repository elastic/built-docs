Query DSLedit

NEST exposes all of the query DSL endpoints available in Elasticsearch

  • Full text queries
  • Term level queries
  • Compound queries
  • Joining queries
  • Geo queries
  • Specialized queries
  • Span queries
  • NEST specific queries

See the documentation on Writing queries for how to use them in NEST.
