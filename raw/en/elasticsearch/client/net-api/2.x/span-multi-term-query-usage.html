Span Multi Term Query Usageedit

Fluent DSL exampleedit

q
.SpanMultiTerm(c => c
    .Name("named_query")
    .Boost(1.1)
    .Match(sq=>sq
        .Prefix(pr=>pr.Field(p=>p.Name).Value("pre-*"))
    )
)

Object Initializer syntax exampleedit

new SpanMultiTermQuery
{
    Name = "named_query",
    Boost = 1.1,
    Match = new PrefixQuery { Field = "name", Value = "pre-*" }
}

Example json output.

{
  "span_multi": {
    "_name": "named_query",
    "boost": 1.1,
    "match": {
      "prefix": {
        "name": {
          "value": "pre-*"
        }
      }
    }
  }
}
