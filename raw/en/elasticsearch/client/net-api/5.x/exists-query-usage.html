Exists Query Usageedit

Fluent DSL exampleedit

q
.Exists(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
)

Object Initializer syntax exampleedit

new ExistsQuery
{
    Name = "named_query",
    Boost = 1.1,
    Field = "description",
}
