Ids Query Usageedit

Fluent DSL exampleedit

q
.Ids(c => c
    .Name("named_query")
    .Boost(1.1)
    .Values(1, 2, 3, 4)
    .Types(typeof(Project), typeof(Developer))
)

Object Initializer syntax exampleedit

new IdsQuery
{
    Name = "named_query",
    Boost = 1.1,
    Values = new List<Id> { 1, 2, 3, 4 },
    Types = Type<Project>().And<Developer>()
}
