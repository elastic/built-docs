Type Query Usageedit

Fluent DSL exampleedit

q
.Type(c => c
    .Name("named_query")
    .Boost(1.1)
    .Value<Developer>()
)

Object Initializer syntax exampleedit

new TypeQuery
{
    Name = "named_query",
    Boost = 1.1,
    Value = Type<Developer>()
}

Example json output.

{
  "type": {
    "_name": "named_query",
    "boost": 1.1,
    "value": "developer"
  }
}
