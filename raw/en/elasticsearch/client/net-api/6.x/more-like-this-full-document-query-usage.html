Fluent DSL example

edit
q
.MoreLikeThis(sn => sn
    .Like(l => l
        .Document(d => d
            .Document(Project.Instance)
            .Routing(Project.Instance.Name)
        )
        .Text("some long text")
    )
)

Object Initializer syntax example

edit
new MoreLikeThisQuery
{
    Like = new List<Like>
    {
        new LikeDocument<Project>(Project.Instance) { Routing = Project.Instance.Name },
        "some long text"
    }
}
