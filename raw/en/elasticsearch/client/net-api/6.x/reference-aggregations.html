Aggregationsedit

Aggregations are arguably one of the most powerful features of Elasticsearch and NEST exposes all of the available Aggregation types

  • Bucket Aggregations
  • Pipeline Aggregations
  • Matrix Aggregations

See the documentation on Writing aggregations for how to use them in NEST.
