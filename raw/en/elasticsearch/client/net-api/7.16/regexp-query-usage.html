Regexp Query Usage

edit

Fluent DSL example

edit
q
.Regexp(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
    .Value("s.*y")
    .Flags("INTERSECTION|COMPLEMENT|EMPTY")
    .MaximumDeterminizedStates(20000)
    .Rewrite(MultiTermQueryRewrite.TopTerms(10))
)

Object Initializer syntax example

edit
new RegexpQuery
{
    Name = "named_query",
    Boost = 1.1,
    Field = "description",
    Value = "s.*y",
    Flags = "INTERSECTION|COMPLEMENT|EMPTY",
    MaximumDeterminizedStates = 20000,
    Rewrite = MultiTermQueryRewrite.TopTerms(10)
}
