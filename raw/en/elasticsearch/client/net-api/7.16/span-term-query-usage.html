Span Term Query Usageedit

Fluent DSL exampleedit

q
.SpanTerm(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field("user")
    .Value("kimchy")
)

Object Initializer syntax exampleedit

new SpanTermQuery
{
    Name = "named_query",
    Boost = 1.1,
    Value = "kimchy",
    Field = "user"
}

Example json output.

{
  "span_term": {
    "user": {
      "_name": "named_query",
      "boost": 1.1,
      "value": "kimchy"
    }
  }
}
