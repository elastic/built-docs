Searchedit

The Search API allows you to execute a search query and get back search hits that match the query. The search capabilities of Elasticsearch is likely one of the reasons you’re using it, and NEST exposes all of the different types of search available, along with some smarts to make working with Elasticsearch from a strongly typed language easier.

  • Writing queries
  • Writing bool queries
  • Returning only certain fields
  • Scrolling documents
  • Covariant search results
