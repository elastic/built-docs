Span Multi Term Query Usage

edit

Fluent DSL example

edit
q
.SpanMultiTerm(c => c
    .Name("named_query")
    .Boost(1.1)
    .Match(sq => sq
        .Prefix(pr => pr.Field(p => p.Description).Value("pre-*"))
    )
)

Object Initializer syntax example

edit
new SpanMultiTermQuery
{
    Name = "named_query",
    Boost = 1.1,
    Match = new PrefixQuery { Field = Infer.Field<Project>(f => f.Description), Value = "pre-*" }
}

Example json output.

{
  "span_multi": {
    "_name": "named_query",
    "boost": 1.1,
    "match": {
      "prefix": {
        "description": {
          "value": "pre-*"
        }
      }
    }
  }
}
