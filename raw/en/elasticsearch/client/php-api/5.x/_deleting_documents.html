Deleting documentsedit

Finally, you can delete documents by specifying their full /index/type/id path:

$params = [
    'index' => 'my_index',
    'type' => 'my_type',
    'id' => 'my_id'
];

// Delete doc at /my_index/my_type/my_id
$response = $client->delete($params);
