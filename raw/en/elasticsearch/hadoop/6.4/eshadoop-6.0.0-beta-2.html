Elasticsearch for Apache Hadoop version 6.0.0-beta2edit

August 31, 2017

Please note that this is a beta release and that we do not recommend running this in production! See our Breaking Changes in 6.0 page for more information on what you might need to modify.

Bug Fixesedit

REST
  • Scroll Id regression on Elasticsearch 2.4.5 #1034
  • A new set of logging commands added for SSL debugging have been reduced in severity to avoid spamming streaming users.

Documentationedit

  • Spark Structured Streaming is no longer an experimental feature in the docs.
