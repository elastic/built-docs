Elasticsearch for Apache Hadoop version 6.6.0edit

January 29, 2019

Deprecationsedit

  • Cascading is now deprecated and will be removed in 7.0 #1222
  • Using SOCKS proxy to connect to Elasticsearch is new deprecated and will be removed in a future release #1223

Enhancementsedit

REST
  • Be able to specify preference query string option when querying Elasticsearch #1213 #1214
Serialization
  • Support for custom MetadataExtractor within Spark flows #1159

Bug Fixesedit

Core
  • Duplicated jars in classpath when submitting a spark job with Elasticsearch to YARN #1097[#1216]
