Identifiersedit

Use an identifier as a named token to specify a variable, type, field, method, or function.

Errors

If a keyword is used as an identifier.

Grammar

ID: [_a-zA-Z] [_a-zA-Z-0-9]*;

Examples

  • Variations of identifiers.

    a
Z
id
list
list0
MAP25
_map25
Map_25
