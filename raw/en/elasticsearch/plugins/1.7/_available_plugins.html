Available pluginsedit

This page is a place holder for the new plugins documentation coming with Elasticsearch 2.0. The officially supported open source plugins in 2.0 have moved to the main Elasticsearch GitHub repository

Elasticsearch provided pluginsedit

The open-source plugins provided for Elasticsearch 1.x are:

  • elasticsearch-analysis-icu - ICU analyzer
  • elasticsearch-analysis-kuromoji - Kurumoji Japanese Analyzer
  • elasticsearch-analysis-phonetic - Phonetic Analyzer
  • elasticsearch-analysis-smartcn - Smart Chinese Analyzer
  • elasticsearch-analysis-stempel - Stempel Polish Analyzer
  • elasticsearch-cloud-aws - AWS Cloud Discovery and Repository
  • elasticsearch-cloud-azure - Azure Cloud Discovery and Repository
  • elasticsearch-cloud-gce - Google Compute Engine Cloud Discovery
  • elasticsearch-lang-js - Javascript Support
  • elasticsearch-lang-js - Python Support
  • elasticsearch-mapper-attachments - Tika support
  • elasticsearch-transport-memcached - Memcached transport
  • elasticsearch-transport-thrift - Thrift Transport
  • elasticsearch-transport-wares - WAR-ES Servlet Transport
