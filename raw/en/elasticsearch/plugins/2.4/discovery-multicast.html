Multicast Discovery Plugin

edit

Deprecated in 2.2.0.

This plugin is deprecated and has been removed in 5.0.0

The Multicast Discovery plugin provides the ability to form a cluster using TCP/IP multicast messages.

Installation

edit

This plugin can be installed using the plugin manager:

sudo bin/plugin install discovery-multicast

The plugin must be installed on every node in the cluster, and each node must be restarted after installation.

This plugin can be downloaded for offline install from https://download.elastic.co/elasticsearch/release/org/elasticsearch/plugin/discovery-multicast/2.4.6/discovery-multicast-2.4.6.zip.

Removal

edit

The plugin can be removed with the following command:

sudo bin/plugin remove discovery-multicast

The node must be stopped before removing the plugin.
