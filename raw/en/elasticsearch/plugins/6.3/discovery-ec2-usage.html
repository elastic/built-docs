Getting started with AWSedit

The plugin provides a hosts provider for zen discovery named ec2. This hosts provider finds other Elasticsearch instances in EC2 through AWS metadata. Authentication is done using IAM Role credentials by default. The only necessary configuration change to enable the plugin is setting the unicast host provider for zen discovery:

discovery.zen.hosts_provider: ec2
