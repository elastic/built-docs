Mapper Size Pluginedit

The mapper-size plugin provides the _size metadata field which, when enabled, indexes the size in bytes of the original _source field.

Installationedit

This plugin can be installed using the plugin manager:

sudo bin/elasticsearch-plugin install mapper-size

The plugin must be installed on every node in the cluster, and each node must be restarted after installation.

You can download this plugin for offline install from https://artifacts.elastic.co/downloads/elasticsearch-plugins/mapper-size/mapper-size-7.10.2.zip. To verify the .zip file, use the SHA hash or ASC key.

Removaledit

The plugin can be removed with the following command:

sudo bin/elasticsearch-plugin remove mapper-size

The node must be stopped before removing the plugin.
