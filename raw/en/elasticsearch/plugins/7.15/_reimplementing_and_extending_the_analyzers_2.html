Reimplementing and extending the analyzersedit

The polish analyzer could be reimplemented as a custom analyzer that can then be extended and configured differently as follows:

PUT /stempel_example
{
  "settings": {
    "analysis": {
      "analyzer": {
        "rebuilt_stempel": {
          "tokenizer":  "standard",
          "filter": [
            "lowercase",
            "polish_stop",
            "polish_stem"
          ]
        }
      }
    }
  }
}
