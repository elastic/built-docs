Stempel Polish analysis plugin

edit

Stempel Polish analysis pluginedit

The Stempel analysis plugin integrates Lucene’s Stempel analysis module for Polish into elasticsearch.

Installationedit

Version 8.15.0 of the Elastic Stack has not yet been released.

This plugin can be installed using the plugin manager:

sudo bin/elasticsearch-plugin install analysis-stempel

The plugin must be installed on every node in the cluster, and each node must be restarted after installation.

You can download this plugin for offline install from https://artifacts.elastic.co/downloads/elasticsearch-plugins/analysis-stempel/analysis-stempel-8.15.0.zip. To verify the .zip file, use the SHA hash or ASC key.

Removaledit

The plugin can be removed with the following command:

sudo bin/elasticsearch-plugin remove analysis-stempel

The node must be stopped before removing the plugin.

stempel tokenizer and token filtersedit

The plugin provides the polish analyzer and the polish_stem and polish_stop token filters, which are not configurable.
