Quota-aware Filesystem Pluginedit

The Quota-aware Filesystem plugin adds an interface for telling Elasticsearch the disk-quota limits under which it is operating.

Installationedit

This plugin can be installed using the plugin manager:

sudo bin/elasticsearch-plugin install quota-aware-fs

The plugin must be installed on every node in the cluster, and each node must be restarted after installation.

You can download this plugin for offline install from https://artifacts.elastic.co/downloads/elasticsearch-plugins/quota-aware-fs/quota-aware-fs-7.11.1.zip. To verify the .zip file, use the SHA hash or ASC key.

Removaledit

The plugin can be removed with the following command:

sudo bin/elasticsearch-plugin remove quota-aware-fs

The node must be stopped before removing the plugin.
