Document APIsedit

This section describes the following CRUD APIs:

Single document APIs

  • Index API
  • Get API
  • Delete API
  • Update API

Multi-document APIs

  • Multi Get API
  • Bulk API

All CRUD APIs are single-index APIs. The index parameter accepts a single index name, or an alias which points to a single index.
