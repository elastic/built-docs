Limit Queryedit

Deprecated in 2.0.0-beta1.

Use the terminate_after parameter instead

A limit query limits the number of documents (per shard) to execute on. For example:

{
    "filtered" : {
        "filter" : {
             "limit" : {"value" : 100}
         },
         "query" : {
            "term" : { "name.first" : "shay" }
        }
    }
}
