Mapping Char Filteredit

A char filter of type mapping replacing characters of an analyzed text with given mapping.

mappings

A list of mappings to use.

mappings_path

A path, relative to the config directory, to a mappings file configuration.

Here is a sample configuration:

{
    "index" : {
        "analysis" : {
            "char_filter" : {
                "my_mapping" : {
                    "type" : "mapping",
                    "mappings" : [
                      "ph => f",
                      "qu => k"
                    ]
                }
            },
            "analyzer" : {
                "custom_with_char_filter" : {
                    "tokenizer" : "standard",
                    "char_filter" : ["my_mapping"]
                }
            }
        }
    }
}
