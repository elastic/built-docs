Start API

Start API

The start API starts the Watcher service if the service is not already running.

Request

POST _xpack/watcher/_start

Authorization

You must have manage_watcher cluster privileges to use this API. For more information, see Security Privileges.

Examples

POST _xpack/watcher/_start

Watcher returns the following response if the request is successful:

{
   "acknowledged": true
}
