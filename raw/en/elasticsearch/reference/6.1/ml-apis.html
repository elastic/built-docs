Machine Learning APIs

You can use the following APIs to perform machine learning activities. See Definitions for the resource definitions used by the machine learning APIs and in advanced job configuration options in Kibana.

Datafeeds

  • Create datafeed, Delete datafeed
  • Start datafeed, Stop datafeed
  • Get datafeed info, Get datafeed statistics
  • Preview datafeed
  • Update datafeed

Jobs

  • Create job, Delete job
  • Open job, Close job
  • Get job info, Get job statistics
  • Flush job
  • Post data to job
  • Update job
  • Forecast job behavior

Model Snapshots

  • Delete model snapshot
  • Get model snapshot info
  • Revert model snapshot
  • Update model snapshot

Results

  • Get buckets
  • Get overall buckets
  • Get categories
  • Get influencers
  • Get records
