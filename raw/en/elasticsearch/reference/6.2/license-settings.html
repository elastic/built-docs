X-Pack License Settings

You can configure this licensing setting in the elasticsearch.yml file. For more information, see X-Pack License Management.

xpack.license.self_generated.type

Set to trial (default) to enable all X-Pack features.

If set to basic, the self-generated license gives access only to the features of a standard basic license. You can later upgrade the cluster to a trial license.
