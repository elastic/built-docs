Delete Jobs from Calendar API

This API enables you to delete jobs from a calendar.

Request

DELETE _xpack/ml/calendars/<calendar_id>/jobs/<job_id>

Path Parameters

calendar_id(required)
(string) Identifier for the calendar.
job_id (required)
(string) Identifier for the job or job group.

Authorization

You must have manage_ml, or manage cluster privileges to use this API. For more information, see Security Privileges.

Examples

The following example removes the association between the planned-outages calendar and total-requests job:

DELETE _xpack/ml/calendars/planned-outages/jobs/total-requests

When the job is removed from the calendar, you receive the following results:

{
   "calendar_id": "planned-outages",
   "job_ids": []
}
