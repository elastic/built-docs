Machine Learning APIsedit

You can use the following APIs to perform machine learning activities. See Definitions for the resource definitions used by the machine learning APIs and in advanced job configuration options in Kibana.

Calendarsedit

  • Create calendar, Delete calendar
  • Add job to calendar, Delete job from calendar
  • Add scheduled events to calendar, Delete scheduled events from calendar
  • Get calendars, Get scheduled events

Filtersedit

  • Create filter, Delete filter
  • Update filters
  • Get filters

Datafeedsedit

  • Create datafeed, Delete datafeed
  • Start datafeed, Stop datafeed
  • Get datafeed info, Get datafeed statistics
  • Preview datafeed
  • Update datafeed

Jobsedit

  • Create job, Delete job
  • Add job to calendar, Delete job from calendar
  • Open job, Close job
  • Get job info, Get job statistics
  • Flush job
  • Post data to job
  • Update job
  • Forecast job behavior

Model Snapshotsedit

  • Delete model snapshot
  • Get model snapshot info
  • Revert model snapshot
  • Update model snapshot

Resultsedit

  • Get buckets
  • Get overall buckets
  • Get categories
  • Get influencers
  • Get records
