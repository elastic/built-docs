Delete Model Snapshots API

edit

Deletes an existing model snapshot.

Request

edit

DELETE _xpack/ml/anomaly_detectors/<job_id>/model_snapshots/<snapshot_id>

Description

edit

You cannot delete the active model snapshot. To delete that snapshot, first revert to a different one. To identify the active model snapshot, refer to the model_snapshot_id in the results from the get jobs API.

Path Parameters

edit
job_id (required)
(string) Identifier for the job
snapshot_id (required)
(string) Identifier for the model snapshot

Authorization

edit

You must have manage_ml, or manage cluster privileges to use this API. For more information, see Security privileges.

Examples

edit

The following example deletes the 1491948163 snapshot:

DELETE _xpack/ml/anomaly_detectors/farequote/model_snapshots/1491948163

When the snapshot is deleted, you receive the following results:

{
  "acknowledged": true
}
