API Quick Referenceedit

This functionality is experimental and may be changed or removed completely in a future release. Elastic will take a best effort approach to fix any issues, but experimental features are not subject to the support SLA of official GA features.

Most rollup endpoints have the following base:

/_xpack/rollup/

/job/edit

  • PUT /_xpack/rollup/job/<job_id>: Create a job
  • GET /_xpack/rollup/job: List jobs
  • GET /_xpack/rollup/job/<job_id>: Get job details
  • POST /_xpack/rollup/job/<job_id>/_start: Start a job
  • POST /_xpack/rollup/job/<job_id>/_stop: Stop a job
  • DELETE /_xpack/rollup/job/<job_id>: Delete a job

/data/edit

  • GET /_xpack/rollup/data/<index_pattern>/_rollup_caps: Get Rollup Capabilities
  • GET /<index_name>/_rollup/data/: Get Rollup Index Capabilities

/<index_name>/edit

  • GET /<index_name>/_rollup_search: Search rollup data
