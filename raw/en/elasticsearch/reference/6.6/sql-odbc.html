SQL ODBCedit

Overviewedit

This functionality is in technical preview and may be changed or removed in a future release. Elastic will apply best effort to fix any issues, but features in technical preview are not subject to the support SLA of official GA features.

Elasticsearch SQL ODBC Driver is a feature-rich 3.80 ODBC driver for Elasticsearch. It is a core level driver, exposing all of the functionality accessible through the Elasticsearch’s SQL ODBC API, converting ODBC calls into Elasticsearch SQL.
