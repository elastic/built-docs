Clear roles cache APIedit

Evicts roles from the native role cache.

Requestedit

POST /_xpack/security/role/<name>/_clear_cache

Descriptionedit

For more information about the native realm, see Realms and Configuring a native realm.

Path Parametersedit

name
(string) The name of the role.

Authorizationedit

To use this API, you must have at least the manage_security cluster privilege.

Examplesedit

The clear roles cache API evicts roles from the native role cache. For example, to clear the cache for my_admin_role:

POST /_xpack/security/role/my_admin_role/_clear_cache
