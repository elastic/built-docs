Where to go from hereedit

Now that you’ve set up a cluster, indexed some documents, and run some searches and aggregations, you might want to:

  • Dive in to the Elastic Stack Tutorial to install Kibana, Logstash, and Beats and set up a basic system monitoring solution.
  • Load one of the sample data sets into Kibana to see how you can use Elasticsearch and Kibana together to visualize your data.

  • Try out one of the Elastic search solutions:

    • Site Search
    • App Search
    • Enterprise Search
