Get autoscaling policy APIedit

This functionality is experimental and may be changed or removed completely in a future release. Elastic will take a best effort approach to fix any issues, but experimental features are not subject to the support SLA of official GA features.

Get autoscaling policy.

Requestedit

PUT /_autoscaling/policy/my_autoscaling_policy
{
  "roles" : [],
  "deciders": {
    "fixed": {
    }
  }
}
GET /_autoscaling/policy/<name>

Prerequisitesedit

  • If the Elasticsearch security features are enabled, you must have manage_autoscaling cluster privileges. For more information, see Security privileges.

Descriptionedit

This API gets an autoscaling policy with the provided name.

Examplesedit

This example gets an autoscaling policy named my_autosaling_policy.

GET /_autoscaling/policy/my_autoscaling_policy

The API returns the following result:

{
   "roles": <roles>,
   "deciders": <deciders>
}
