Type Query

edit

Deprecated in 7.0.0.

Types and the type query are deprecated and in the process of being removed. See Removal of mapping types.

Filters documents matching the provided document / mapping type.

GET /_search
{
  "query": {
    "type": {
      "value": "_doc"
    }
  }
}
