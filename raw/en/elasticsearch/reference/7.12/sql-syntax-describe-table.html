DESCRIBE TABLE

edit

Synopsis:

DESCRIBE
    [table identifier | 
    [LIKE pattern]]

single table identifier or double quoted es multi index

SQL LIKE pattern

or

DESC
    [table identifier | 
    [LIKE pattern]]

single table identifier or double quoted es multi index

SQL LIKE pattern

Description: DESC and DESCRIBE are aliases to SHOW COLUMNS.
