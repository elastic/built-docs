Migration APIsedit

The migration APIs power Kibana’s Upgrade Assistant feature.

These APIs are designed for indirect use by Kibana’s Upgrade Assistant. We strongly recommend you use the Upgrade Assistant to upgrade from 7.17 to 8.0 and later versions. For upgrade instructions, refer to Upgrading to Elastic 7.17.11.
