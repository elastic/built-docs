Index lifecycle management APIedit

You can use the following APIs to manage policies on indices. See Managing the index lifecycle for more information about Index Lifecycle Management.

Index management APIsedit

  • Retry policy on indices
  • Remove policy from index

Operation management APIsedit

  • Start ILM
  • Stop ILM
  • Explain API
