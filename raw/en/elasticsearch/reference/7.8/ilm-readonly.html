Read onlyedit

Phases allowed: warm.

Makes the index read-only.

Optionsedit

None.

Exampleedit

PUT _ilm/policy/my_policy
{
  "policy": {
    "phases": {
      "warm": {
        "actions": {
          "readonly" : { }
        }
      }
    }
  }
}
