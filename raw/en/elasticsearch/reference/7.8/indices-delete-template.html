Delete index template APIedit

Deletes an index template.

DELETE /_template/template_1

Requestedit

DELETE /_template/<index-template>

Descriptionedit

Use the delete index template API to delete one or more index templates. Index templates define settings, mappings, and aliases that can be applied automatically to new indices.

Path parametersedit

<index-template>
(Required, string) Comma-separated list or wildcard expression of index template names used to limit the request.

Query parametersedit

master_timeout
(Optional, time units) Specifies the period of time to wait for a connection to the master node. If no response is received before the timeout expires, the request fails and returns an error. Defaults to 30s.
timeout
(Optional, time units) Specifies the period of time to wait for a response. If no response is received before the timeout expires, the request fails and returns an error. Defaults to 30s.
