Get machine learning info APIedit

Returns defaults and limits used by machine learning.

Requestedit

GET _ml/info

Prerequisitesedit

  • If the Elasticsearch security features are enabled, you must have monitor_ml, monitor, manage_ml, or manage cluster privileges to use this API. The machine_learning_admin and machine_learning_user roles provide these privileges. See Security privileges, Built-in roles and Machine learning security privileges.

Descriptionedit

This endpoint is designed to be used by a user interface that needs to fully understand machine learning configurations where some options are not specified, meaning that the defaults should be used. This endpoint may be used to find out what those defaults are.

The endpoint takes no arguments:

GET _ml/info
