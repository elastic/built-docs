Deleteedit

Phases allowed: delete.

Permanently removes the index.

Optionsedit

delete_searchable_snapshot
(Optional, Boolean) Deletes the searchable snapshot created in a previous phase. Defaults to true. This option is applicable when the searchable snapshot action is used in the cold phase.

Exampleedit

PUT _ilm/policy/my_policy
{
  "policy": {
    "phases": {
      "delete": {
        "actions": {
          "delete" : { }
        }
      }
    }
  }
}
