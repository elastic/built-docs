Get data stream APIedit

Returns information about one or more data streams.

GET _data_stream/my-data-stream

Requestedit

GET _data_stream/<data-stream>

Path parametersedit

<data-stream>

(Required, string) Name or wildcard expression of the data stream(s) to retrieve.

Examplesedit

Basic exampleedit

GET _data_stream/my-data-stream*

The API returns the following response:

[
  {
    "name" : "my-data-stream", 
    "timestamp_field" : "@timestamp", 
    "indices" : [ 
      {
        "index_name" : "my-data-stream-000001",
        "index_uuid" : "DXAE-xcCQTKF93bMm9iawA"
      },
      {
        "index_name" : "my-data-stream-000002",
        "index_uuid" : "Wzxq0VhsQKyPxHhaK3WYAg"
      }
    ],
    "generation" : 2 
  }
]

Name of the data stream

The name of the timestamp field for the data stream

List of backing indices

Current generation for the data stream
