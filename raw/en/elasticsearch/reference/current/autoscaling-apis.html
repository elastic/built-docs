Autoscaling APIsedit

Autoscaling is designed for indirect use by Elasticsearch Service, Elastic Cloud Enterprise, and Elastic Cloud on Kubernetes. Direct use is not supported.

You can use the following APIs to perform autoscaling operations.

Top-Leveledit

  • Get autoscaling capacity
  • Create or update autoscaling policy
  • Delete autoscaling policy
  • Get autoscaling policy
