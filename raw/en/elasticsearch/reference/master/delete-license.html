Delete license API

edit

This API enables you to delete licensing information.

Request

edit

DELETE /_license

Description

edit

When your license expires, X-Pack operates in a degraded mode. For more information, see License expiration.

Prerequisites

edit
  • If the Elasticsearch security features are enabled, you must have the manage cluster privilege to use this API.
  • If the operator privileges feature is enabled, only operator users can use this API.

Examples

edit

The following example queries the info API:

resp = client.license.delete()
print(resp)
response = client.license.delete
puts response
const response = await client.license.delete();
console.log(response);
DELETE /_license

When the license is successfully deleted, the API returns the following response:

{
  "acknowledged": true
}
