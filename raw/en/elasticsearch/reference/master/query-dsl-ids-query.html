IDsedit

Returns documents based on their IDs. This query uses document IDs stored in the _id field.

Example requestedit

response = client.search(
  body: {
    query: {
      ids: {
        values: [
          '1',
          '4',
          '100'
        ]
      }
    }
  }
)
puts response
GET /_search
{
  "query": {
    "ids" : {
      "values" : ["1", "4", "100"]
    }
  }
}

Top-level parameters for idsedit

values
(Required, array of strings) An array of document IDs.
