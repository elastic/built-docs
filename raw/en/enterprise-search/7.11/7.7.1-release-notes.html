7.7.1 Release Notesedit

  • Enterprise Search
  • App Search
  • Workplace Search

Enterprise Searchedit

Bug Fixesedit

  • Honors the configured external_url in Elastic Cloud, enabling a custom URL for the deployment. This fix changes the value of the X-Forwarded-Host header, and therefore could affect deployments who are depending on that value.

App Searchedit

Bug Fixesedit

  • Prevents sporadic "engine named undefined" errors in the dashboard.
  • Honors values for app_search.engine.query.limit greater than 128.

Workplace Searchedit

Bug Fixesedit

  • Ensures clients authenticate before refreshing tokens. To refresh a token, a client must send client_id. A confidential client must also send client_secret.
  • Enables syncing Google Drive when document permissions are disabled.
  • Allows administrators to add private sources for configured connectors when no organization content sources exist.
  • Ensures content counts are consistent between administration screens and the search experience.
  • The search experience now works in Read-Only mode.
