7.10.2 Release notesedit

Upgrading to Enterprise Search 7.10.2? See Upgrading & migrating.

Workplace Searchedit

Bug fixesedit

  • Fixes pagination for Confluence Cloud results.
  • Fixes the UI for fixing a connector error.
  • Restores protection against rate limit errors.

  • Improves performance when including thumbnails with results.

    • Reduces memory requirements for thumbnailing image files (png, jpg, gif).
    • Reduces memory requirements for thumbnailing Microsoft Office documents (doc, docx, pptx, xls, xlsx).
    • Disables Excel thumbnails.
