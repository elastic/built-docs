7.13.2 release notesedit

Upgrading to Enterprise Search 7.13.2? See Upgrading & migrating.

  • App Search
  • Workplace Search

App Searchedit

Bug fixesedit

  • Fixes a bug that allowed App Search API keys with only read access to create engines if the key also had full engine access. Now, API keys must have write access to create engines.

Workplace Searchedit

Bug fixesedit

  • Fixes a bug that caused content source synchronization to fail sporadically, due to a race condition. The fix ensures proper recovery in these instances.
  • Fixes a bug that caused TCP connections to leak. Background threads now share a connection pool.
