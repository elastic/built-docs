7.17.2 release notesedit

Upgrading to Enterprise Search 7.17.2? See Upgrading & migrating.

Bug fixesedit

  • index.search.slowlog.level settings are now removed from all indices managed by Enterprise Search.

    These settings caused issues when upgrading from 7.17.0 or 7.17.1 to 8.x — for deployments that previously ran on versions pre-7.12.0.

Known issuesedit

  • Jira Cloud and Confluence Cloud content sources cannot be newly connected or reauthorized in 7.17.2. This is due to backwards incompatible API changes in these Atlassian products that were subsequently rolled back.
  • App Search users with dev and editor roles do not have access to the web crawler.
