7.17.3 release notesedit

Upgrading to Enterprise Search 7.17.3? See Upgrading & migrating.

Bug fixesedit

  • Adaptive Relevance suggestions are created correctly for Meta Engines.
  • Updates the Confluence Cloud connector and Jira Cloud connector to restore full compatibility with these Atlassian services. Prior Enterprise Search versions are incompatible due to a series of Atlassian API changes.

Known issuesedit

  • App Search users with dev and editor roles do not have access to the web crawler.
