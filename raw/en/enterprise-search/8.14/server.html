Elastic Enterprise Search server

edit

Elastic Enterprise Search serveredit

Enterprise Search server is an Elastic service this is required for the following features and functionalities:

  • The Elastic web crawler
  • Native connectors that run on Elastic Cloud
  • The App Search and Workplace Search standalone products

Connector servicesedit

The Enterprise Search server artifacts used in Elastic Cloud include the connector services required for native connectors. However, the downloadable artifacts for Enterprise Search server do not.

When running Enterprise Search on your own infrastructure, you must additionally run one or more connector services to use connectors. Refer to the connectors repository for more information.
