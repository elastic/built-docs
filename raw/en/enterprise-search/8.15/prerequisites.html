Prerequisitesedit

Enterprise Search tools are features of the Elastic search platform. To use these tools, you must first have an Elastic subscription, deployment, and user (or token).

To satisfy these requirements, create an Elastic Cloud account, and create your first deployment through the Elastic Cloud UI. This process provides a subscription, deployment, and superuser with access to all Enterprise Search features.

For each prerequisite, additional options are available. The following documents cover the details for each:

  • Elastic subscription
  • Elastic deployment
  • Elastic user (or token)
