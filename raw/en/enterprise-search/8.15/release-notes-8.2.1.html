8.2.1 release notesedit

Upgrading to Enterprise Search 8.2.1? See Upgrading and migrating.

Bug fixesedit

  • Fixed a bug where nested lists could cause the web crawler to make parsing errors. This impacted the application of data-elastic-exclude and data-elastic-include directives in a document.
  • Fixed some uncommon scenarios where the Salesforce connector would error when attempting to download Attachment objects.

Known issuesedit

  • App Search users with dev and editor roles do not have access to the web crawler.
