8.0.0-beta1 release notesedit

Upgrading to Enterprise Search 8.0.0-beta1? See Upgrading & migrating.

  • Enterprise Search
  • App Search

Enterprise Searchedit

Breaking changesedit

  • Renames the following configuration settings:

    • workplace_search.custom_api_source.document_size → workplace_search.content_source.document_size.limit
    • workplace_search.custom_api_source.total_fields → workplace_search.content_source.total_fields.limit

  • Removes the following deprecated configuration settings:

    • enable_ecs_logs
    • ent_search.monitoring_enabled
    • ent_search.monitoring_port
    • ent_search.monitoring_ssl

App Searchedit

Breaking changesedit

  • Enterprise Search crawler event log (crawl-status events) and Crawler Status API now use millisecond resolution for all duration fields.
