Elastic Search UIedit

Elastic Search UI is a collection of JavaScript and React tools for building search experiences, optimized for use with Elasticsearch, App Search, and Workplace Search.

Use Search UI to integrate Elastic with your existing UI, or quickly build a search UI for testing or for proof-of-concept use cases.
