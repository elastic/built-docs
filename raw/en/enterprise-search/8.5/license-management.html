Elastic subscriptionedit

Enterprise Search tools share several prerequisites, including an Elastic subscription.

Get started with a free trial on Elastic Cloud. An Elastic Cloud subscription provides access to all Enterprise Search features.

Alternatively, download artifacts to deploy Elastic services to production infrastructure or within local development environments. Run the software to automatically begin your subscription. To use all Enterprise Search features, you will need to upgrade to a paid subscription (or begin a free trial).

The following pages on the Elastic website list the subscription levels required for various Enterprise Search features:

  • Elastic Cloud managed service features
  • Elastic Stack subscriptions

The following sections within this document present various options to start your Elastic subscription:

  • Elastic Cloud
  • Elastic Cloud Enterprise (ECE)
  • Elastic Cloud on Kubernetes (ECK)
  • Elasticsearch (and Kibana)

Questions about subscriptions? Contact Elastic.

It may take up to 5 minutes for Enterprise Search to reflect changes to your subscription.

Elastic Cloudedit

Get started for free on Elastic Cloud.

Create your Elastic Cloud account and get started with your first deployment. Your free trial will begin automatically.

After your trial expires, complete the billing information prompts to continue your subscription. Or, convert your trial to a paid subscription at any time by providing your billing details. See Add your billing details in the Elastic Cloud documentation.

Multiple subscription levels are available. Your subscription level affects which features are available to your Elastic deployments. See Elastic Cloud managed service features on the Elastic website.

Alternatively, you can subscribe through a marketplace to manage billing in one place. See Subscribing from a marketplace in the Elastic Cloud documentation.

Elastic Cloud Enterprise (ECE)edit

Get started for free with Elastic Cloud Enterprise (ECE).

Download and run ECE on your chosen infrastructure. Your free trial will begin automatically.

To upgrade to a paid subscription, submit a request through the Enterprise subscriptions contact form.

After purchasing a subscription, you will receive a license key. Add this license to your ECE deployment to complete the subscription process. See Add a license in the ECE documentation.

Elastic Cloud on Kubernetes (ECK)edit

Get started for free with Elastic Cloud on Kubernetes (ECK).

Download and run ECK on your chosen infrastructure. Your free subscription will begin automatically.

To enable all Enterprise Search features, you will need to start a free trial or upgrade to a paid subscription.

To start a trial, see Start a trial in the ECK documentation.

To upgrade to a paid subscription, submit a request through the Enterprise subscriptions contact form.

After purchasing a subscription, you will receive a license key. Add this license to your ECK deployment to complete the subscription process. See Add a license in the ECE documentation.

Elasticsearch (and Kibana)edit

Get started for free with Elasticsearch.

Download and run Elasticsearch and any additional services you require. Your free subscription will begin automatically.

To enable all Enterprise Search features, you will need to start a free trial or upgrade to a paid subscription.

To start a trial, use the License management UI within Kibana or the Start trial API for Elasticsearch. See License management in the Kibana documentation, and Start trial API in the Elasticsearch documentation.

Alternatively, use an Elasticsearch configuration setting to start a trial. See License settings in the Elasticsearch documentation.

To upgrade to a paid subscription, submit a request through the Subscriptions contact form.

After purchasing a subscription, you will receive a license key. Add this license to your Elastic Stack deployment to complete the subscription process. See License management in the Kibana documentation, and Update license API in the Elasticsearch documentation.

Multiple subscription levels are available. Your subscription level affects which features are available to your Elastic deployments. See Elastic Stack subscriptions on the Elastic website.
