8.0.0-alpha1 release notesedit

Upgrading to Enterprise Search 8.0.0-alpha1? See Upgrading and migrating.

  • Enterprise Search
  • App Search
  • Workplace Search

Enterprise Searchedit

Breaking changesedit

  • Requires Elasticsearch security features.

    • You must configure security for the Elastic Stack before using Enterprise Search.
    • Security features are enabled by default on Elastic Cloud.

  • Removes the standalone Enterprise Search management interface.

    • Manage Enterprise Search in Kibana, the standard management interface for Elastic.

App Searchedit

Bug fixesedit

  • Ensures that search boost factors are greater or equal to zero.

    • The API will respond with a 400 error code when boost factors are negative.

Workplace Searchedit

Known issuesedit

  • Within the search application, selecting the thumbnail of a result may cause a redirect to the Kibana management interface.
