8.4.1 release notesedit

Upgrading to Enterprise Search 8.4.1? See Upgrading and migrating.

Bug fixesedit

  • Fixes a bug in the Github and Github Enterprise Server Workplace Search content sources where rate limit errors may cause syncs to fail instead of retrying later.
  • Fixes document-level permissions for GitHub Workplace Search content sources to prevent access by suspended GitHub users.

Known issuesedit

  • The Elastic web crawler does not currently respect a crawl schedule, when configured. Crawls must be manually triggered in the UI.
