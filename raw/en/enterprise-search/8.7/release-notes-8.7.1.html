8.7.1 release notesedit

Upgrading to Enterprise Search 8.7.1? See Upgrading and migrating.

New featuresedit

  • The Elastic Python connector framework has improved monitoring capabilities:

    • During sync job execution, a new fly-out displays ingestion statistics such as indexed document count, indexed document volume and deleted document count.
    • Added the ability to cancel a sync job.
    • Orphaned sync jobs and idle sync jobs are now cleaned up by the Python connector service.
