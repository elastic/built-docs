App Search and Workplace Search product documentation

edit

Using Elastic Stack tools

ℹ️ For new users, we recommend using our native Elasticsearch tools, rather than the standalone App Search or Workplace Search products. We are actively developing new features and capabilities in the Elastic Stack to help you build powerful search applications.

Refer to the following for more information:

  • App Search
  • Workplace Search

While the Enterprise Search documentation does contain some documentation for App Search and Workplace Search (specifically, users and compatibility), these standalone products are primarily documented separately.
