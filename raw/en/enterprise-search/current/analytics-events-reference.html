Analytics events referenceedit

Behavioral analytics logs events using the Elastic Common Schema, including a custom field set for analytics events. Refer to examples of the full data objects that are logged.

Fields overviewedit

  • Base fields
  • Event fields
  • Labels fields
  • User agent fields
  • URL fields

Base fieldsedit

@timestamp
UTC timestamp of the event.
data_stream.dataset
Datastream dataset type: elastic_analytics.events.
data_stream.namespace
Datastream namespace: corresponds to the events collection name.
data_stream.type
Datastream type: logs.
ecs.version
Elastic Common Schema version.

Event fieldsedit

event.action
Action: click, search, or pageview.
event.dataset
Dataset: elastic_analytics.events.
event.user_agent
User agent string.
event.customer_data.action
User action: for example add_to_cart.
event.customer_data.category
Category: for example product.
event.customer_data.field
Field: for example product_id.
event.customer_data.label
Label: for example product_name.
event.customer_data.query
Search query.
event.customer_data.totalResults
Total results for a search query.
event.customer_data.type
Event type: for example SearchQuery.
event.customer_data.value
Event value: for example product_price.

Labels fieldsedit

labels.index_date
Date the event was indexed.
labels.session_uuid
Session UUID.
labels.user_uuid
User UUID.

User agent fieldsedit

user_agent.device.name
Device name: for example Mac.
user_agent.name
User agent name: for example Chrome.
user_agent.original
User agent string.
user_agent.os.full
Full operating system details: for example Mac OS X 10.15.7.
user_agent.os.name
Operating system name: for example Mac OS X.
user_agent.os.version
Operating system version.
user_agent.version
User agent version.

URL fieldsedit

These fields are only present for pageview events.

url.domain
Domain component of the URL. For example www.elastic.co.
url.path
Path component of the URL. For example /search.
url.port
Port component of the URL. For example 80 or 443.
url.scheme
Scheme component of the URL. For example http or https.
