7.11.2 release notesedit

Upgrading to Enterprise Search 7.11.2? See Upgrading & migrating.

App Searchedit

Bug fixesedit

  • Enforces access control for API requests when Elasticsearch security is disabled.

    Prior to 7.11.0, App Search enforced access control for APIs, even when Elasticsearch security was disabled. App Search 7.11.0 unintentionally changed this behavior and no longer enforced access control when security was disabled. App Search 7.11.2 restores the previous behavior to provide a consistent experience across App Search releases.

    Outside of a few limited use cases, always enable security for your Enterprise Search deployments.

Bug fixesedit

  • Fixes an intermittent, broken search experience for some deployments connected to Slack.
