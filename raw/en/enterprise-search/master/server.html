Elastic Enterprise Search serveredit

Enterprise Search server is an Elastic service this is required when using App Search and Workplace Search and other Enterprise Search features, such as the web crawler.

Connector servicesedit

The Enterprise Search server artifacts used in Elastic Cloud include the connector services required for native connectors. However, the downloadable artifacts for Enterprise Search server do not.

When running Enterprise Search on your own infrastructure, you must additionally run one or more connector services to use connectors. Refer to the following repositories for documentation:

  • Python (branch main, compatible with Elastic 8.9)
  • Ruby (branch main, compatible with Elastic 8.9)
