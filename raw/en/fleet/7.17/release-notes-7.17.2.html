Fleet and Elastic Agent 7.17.2edit

Review important information about the Fleet and Elastic Agent 7.17.2 release.

Bug fixesedit

Fleet
  • Use validated fields for default_fields index setting. #128094
  • Fix links to Agent logs for APM, Endpoint, synthetics, and osquery. #127480
  • Make input IDs unique in agent policy yaml. #127343
Elastic Agent
  • Propagate input ID from the Agent policy into the Filebeat configuration. Note that no validation is performed on this field. #30386
  • Fix the start sequence of Beats that was non-deterministic making Beats missing their configuration from Agent and not sending events. #30694
