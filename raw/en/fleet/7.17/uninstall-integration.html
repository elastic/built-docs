Uninstall an Elastic Agent integration

edit

Uninstall an integration to remove all Kibana and Elasticsearch assets that were installed by the integration.

Some integrations, like the System integration, are installed by default and cannot be uninstalled.

  1. Before uninstalling an integration, delete the integration policy from any Elastic Agent policies that use it.

    Any Elastic Agents enrolled in the policy will stop using the deleted integration.

  2. After deleting all integration policies, click the Settings tab.
  3. Click Uninstall to remove all Kibana and Elasticsearch assets that were installed by the integration.
