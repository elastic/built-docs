Fleet and Elastic Agent 8.13.3edit

Review important information about Fleet Server and Elastic Agent for the 8.13.3 release.

Security updatesedit

Elastic Agent
  • Update Go version to 1.21.9. #4508

Bug fixesedit

Fleet
  • Fix managed agent policy preconfiguration update. (#181624)
  • Use lowercase dataset name in template and pipeline names. (#180887)
  • Fix KQL/kuery for getting Fleet Server agent count. (#180650)
Elastic Agent
  • Fix unconditionally logging all 400 HTTP responses from Fleet Server as API compatibility errors. #4481 #4477
  • Allow the elastic-agent upgrade command to succeed when the agent restarts before it has completely acknowledged the upgrade. #4519 #3890s
  • Only allow installing Elastic Defend on Windows kernels newer than 6, that is, Windows 10 / Server 2016 and newer.
