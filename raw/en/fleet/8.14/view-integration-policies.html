View Elastic Agent integration policies

edit

An integration policy is created when you add an integration to an Elastic Agent policy.

To view details about all the integration policies for a specific integration:

  1. In Kibana, go to the Integrations page and open the Installed integrations tab. Search for and select the integration you want to view. You can select a category to narrow your search.
  2. Click the Policies tab to see the list of integration policies.

Open the Actions menu to see other actions you can perform from this view.
