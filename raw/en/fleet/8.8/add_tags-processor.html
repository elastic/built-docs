Add tagsedit

The add_tags processor adds tags to a list of tags. If the target field already exists, the tags are appended to the existing list of tags.

Exampleedit

This configuration:

  - add_tags:
      tags: [web, production]
      target: "environment"

Adds the environment field to every event:

{
  "environment": ["web", "production"]
}

Configuration settingsedit

Elastic Agent processors execute before ingest pipelines, which means that they process the raw event data rather than the final event sent to Elasticsearch. For related limitations, refer to What are some limitations of using processors?

Name Required Default Description

tags

Yes

List of tags to add.

target

No

tags

Field the tags will be added to. Setting tags in @metadata is not supported.
