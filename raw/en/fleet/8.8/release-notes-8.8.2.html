Fleet and Elastic Agent 8.8.2

edit

Review important information about the Fleet and Elastic Agent 8.8.2 release.

Security updates

edit
Elastic Agent
  • Updated Go version to 1.19.10. #2846

Enhancements

edit
  • Log start and stop operations from service runtime at INFO rather than DEBUG level. #2879 #2864

Bug fixes

edit
Fleet
  • Fixes usage of AsyncLocalStorage for audit log. #159807
  • Fixing issue of returning output API key. #159179
Elastic Agent
  • Explicitly specify timeout units as seconds in the Endpoint spec file. #2870 #2863
  • Fix logs collection in diagnostics when Elastic Agent is running on Kubernetes. #2905 #2899
  • The known issue that caused an Elastic Defend and Elastic Agent CPU spike when connectivity to Elasticsearch and/or Logstash is lost has been resolved in this release.
