Fleet and Elastic Agent 8.14.3edit

Review important information about the Fleet and Elastic Agent 8.14.3 release.

Bug fixesedit

Elastic Agent
  • Update the elastic-agent install -f command to use paths associated with the installed agent binary instead of paths associated with the agent running the install command. #4965 #4506
  • Allow the Elastic Agent container to work with a read-only filesystem. #5010
  • Upgrade NodeJS to LTS version 18.20.3. #4995
  • Fix Elastic Agent to account for the timeout period when waiting for Fleet Server to start. #5034 #5033
