Infrastructure appedit

After you have set up infrastructure monitoring and data is streaming to Elasticsearch, you can use the Infrastructure app to monitor your infrastructure and identify problems in real time.

For more information about the Infrastructure app, see the Infrastructure app in the Kibana documentation.
