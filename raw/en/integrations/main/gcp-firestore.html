Metrics

edit

The firestore dataset fetches metrics from Firestore in Google Cloud Platform. It contains all metrics exported from the GCP Firestore Monitoring API.

You can specify a single region to fetch metrics like us-central1. Be aware that GCP Storage does not use zones so us-central1-a will return nothing. If no region is specified, it will return metrics from all buckets.

        "provider": "gcp",
        "availability_zone": "us-central1-c",
        "region": "us-central1"
    },
    "event": {
        "dataset": "gcp.firestore",
        "duration": 115000,

    "gcp": {
        "firestore": {
            "document": {

Users with GCP integration enabled will need to input values for these variables again when upgrading the policies to this version.
