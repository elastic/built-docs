Supported Serverless project types

edit

Elastic serverless products allow you to deploy and use Elastic for your use cases without managing the underlying Elastic cluster, such as nodes, data tiers, and scaling. Serverless instances are fully-managed, autoscaled, and automatically upgraded by Elastic so you can focus more on gaining value and insight from your data.

The Elastic serverless offering includes three different project types: Elasticsearch, Elastic Observability, and Elastic Security. Each project type comes with capabilities targeting different use cases. Elastic Integrations require some of these capabilities to work properly.
