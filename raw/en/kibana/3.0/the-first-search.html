The first search

edit

Kibana allows you to search Elasticsearch data via the Lucene Query String syntax. Queries can be run via the query input at the top of the page

Type this in the query bar. Then check out the first few rows of the table.

friends, romans, countrymen

For more information on queries see Queries and Filters
