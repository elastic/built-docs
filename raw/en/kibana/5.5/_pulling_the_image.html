Pulling the image

edit

Obtaining Kibana for Docker is as simple as issuing a docker pull command against the Elastic Docker registry.

The Docker image for Kibana 5.5.3 can be retrieved with the following command:

docker pull docker.elastic.co/kibana/kibana:5.5.3
