Monitoringedit

The X-Pack monitoring components enable you to easily monitor the Elastic Stack from Kibana. You can view health and performance data for Elasticsearch, Logstash, and Kibana in real time, as well as analyze past performance.

When you install X-Pack, a monitoring agent runs on each Elasticsearch, Kibana, and Logstash instance to collect and index metrics.

For more information, see Monitoring the Elastic Stack.
