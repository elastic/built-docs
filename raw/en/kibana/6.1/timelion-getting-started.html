Getting Startededit

Ready to experience all that is Timelion? This getting started tutorial shows you how to:

  • Create a time series chart with multiple expressions
  • Customize the visualization type and formatting options
  • Use mathematical functions to manipulate your data
  • Track trends and use conditional logic to format outliers
  • Add Timelion visualizations to a dashboard
  • Access inline help and documentation
