Kibana Role Management API

edit

This API is experimental and may be changed or removed completely in a future release. The underlying mechanism of enforcing role based access control is stable, but the APIs for managing the roles are currently experimental.

The role management API allows people to manage roles that grant Kibana privileges. It is not supported to do so using the Elasticsearch role management APIs, and doing so will likely cause Kibana’s authorization to behave unexpectedly.
