Delete spaceedit

This API is experimental and may be changed or removed completely in a future release. The underlying Spaces concepts are stable, but the APIs for managing Spaces are currently experimental.

Deleting a space will automatically delete all saved objects that belong to that space. This operation cannot be undone!

Requestedit

To delete a space, submit a DELETE request to the /api/spaces/space/<space_id> endpoint:

DELETE /api/spaces/space/marketing

Responseedit

If the space is successfully deleted, the response code is 204; otherwise, the response code is 404.
