Profiling your Queries and Aggregationsedit

Elasticsearch has a powerful profiler API which can be used to inspect and analyze your search queries. The response, however, is a very large JSON blob which is difficult to analyze by hand.

The Search Profiler tool can transform this JSON output into a visualization that is easy to navigate, allowing you to diagnose and debug poorly performing queries much faster.
