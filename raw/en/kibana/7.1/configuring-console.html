Configuring Consoleedit

You can configure Console to your preferences.

Configuring settingsedit

Settings allows you to modify the font size and set the fileds for autocomplete.

Disabling Consoleedit

If you don’t want to use Console, you can disable it by setting console.enabled to false in your kibana.yml configuration file. Changing this setting causes the server to regenerate assets on the next startup, which might cause a delay before pages start being served.
