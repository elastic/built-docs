Configure monitoring in Kibanaedit

If you enable the monitoring features in your cluster, there are two methods to collect metrics about Kibana:

  • Metricbeat collection methods
  • Legacy collection methods

You can also use Kibana to visualize monitoring data from across the Elastic Stack.

To learn about monitoring in general, see Monitor a cluster.
