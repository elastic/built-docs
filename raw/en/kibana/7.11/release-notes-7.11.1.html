Kibana 7.11.1edit

The 7.11.1 release includes the following enhancements and bug fixes. Before you upgrade, refer to the Breaking changes in 7.11.0 and Known issue in 7.11.0.

Enhancementsedit

Machine Learning
  • Improves the lazy ML node UI #90455
Security
  • Prevent autocompleting the username field #88682

Bug fixesedit

Dashboard
  • Fixes a 7.11.0 regression where pending dashboard searches aren’t canceled when navigating to another app #90306
  • Sync tooltip legend values #90036
Discover
  • Fixes auto collapsing of documents #89712
Elastic Security Solution
For the Elastic Security Solution 7.11.1 bug fixes, refer to Elastic Security Solution Release Notes.
Lens and visualizations
  • Lens visualizations now load faster on dashboards #88953
  • Advanced input now works for aggregation-based visualizations #88154
Maps
  • Always check license at plugin startup #87873
Monitoring
  • Fully control the in memory table pagination and sorting properties #85862
  • Tweak timeout for failing cloud test #86671
  • Stop using constructor.name for logstash pipelines #87386
  • Make sure we use the right duration for messaging on this alert #87579
  • Avoid spamming toast messages #89930
