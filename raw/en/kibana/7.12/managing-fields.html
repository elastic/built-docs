Field managementedit

Whenever possible, Kibana uses the same field type for display as Elasticsearch. However, a few field types Elasticsearch supports are not available in Kibana. Use field formatters to customize how your fields are displayed in Kibana, regardless of how they are stored in Elasticsearch.

Kibana provides these field formatters:

  • Strings
  • Dates
  • Geopoints
  • Numbers

To format a field:

  1. Open the main menu, and click Stack Management > Index Patterns.
  2. Click the index pattern that contains the field you want to format.
  3. Find the field you want to format and click the edit icon ().
  4. Enter a custom label for the field, if needed.

  5. Select a format and fill in the details.
