Kibana 7.13.4edit

For information about the Kibana 7.13.4 release, review the following information.

Breaking changesedit

Breaking changes can prevent your application from optimal operation and performance. Before you upgrade to 7.13.4, review the 7.13.0 breaking changes, then mitigate the impact to your application.

Bug Fixesedit

The 7.13.4 release includes the following bug fixes.

Maps
  • Fixes tracking threshold alerts improper handling of Elasticsearch epoch millis strings #105010
Operations
  • Fixes KBN_PATH_CONF expansion in Windows binaries. Paths are now treated as a string #104517
