Stack rule typesedit

Kibana provides two types of rules:

  • Stack rules, which are built into Kibana
  • Domain-specific rules, which are registered by Kibana apps.

Kibana provides two stack rules:

  • Index threshold
  • Elasticsearch query

Users require the all privilege to access the Stack Rules feature and create and edit rules. See feature privileges for more information.

Some rule types are subscription features, while others are free features. For a comparison of the Elastic subscription levels, see the subscription page.
