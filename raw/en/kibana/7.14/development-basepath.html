Considerations for basepath

edit

In dev mode, Kibana by default runs behind a proxy which adds a random path component to its URL.

You can set this explicitly using server.basePath. This setting cannot end in a slash (/).

Use server.rewriteBasePath to tell Kibana if it should remove the basePath from requests it receives, and to prevent a deprecation warning at startup.

If you want to turn off the basepath when in development mode, start Kibana with the --no-base-path flag

yarn start --no-base-path
