Development tools settings in Kibanaedit

You do not need to configure any settings to use the development tools in Kibana. They are enabled by default.

Grok Debugger settingsedit

xpack.grokdebugger.enabled
Set to true to enable the Grok Debugger. Defaults to true.

Search Profiler settingsedit

xpack.searchprofiler.enabled
Set to true to enable the Search Profiler. Defaults to true.

Painless Lab settingsedit

xpack.painless_lab.enabled
When set to true, enables the Painless Lab. Defaults to true.
