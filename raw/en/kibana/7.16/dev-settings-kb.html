Development tools settings in Kibanaedit

You do not need to configure any settings to use the development tools in Kibana. They are enabled by default.

Grok Debugger settingsedit

xpack.grokdebugger.enabled
[7.16.0] Deprecated in 7.16.0. In 8.0 and later, this setting will no longer be supported. Set to true to enable the Grok Debugger. Defaults to true.

Search Profiler settingsedit

xpack.searchprofiler.enabled
[7.16.0] Deprecated in 7.16.0. In 8.0 and later, this setting will no longer be supported. Set to true to enable the Search Profiler. Defaults to true.

Painless Lab settingsedit

xpack.painless_lab.enabled
[7.16.0] Deprecated in 7.16.0. In 8.0 and later, this setting will no longer be supported. When set to true, enables the Painless Lab. Defaults to true.
