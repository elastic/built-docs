Kibana 7.16.1edit

Review the following information about the 7.16.1 release.

Known issuesedit

Before you upgrade, review the known issues, then mitigate the impact to your application.

There are no known issues in 7.16.1.

For the known issues in the previous releases, refer to the known issues in 7.16.0.

Breaking changesedit

Breaking changes can prevent your application from optimal operation and performance. Before you upgrade to 7.16.1, review the 7.16.0 breaking changes.

To review the breaking changes in previous versions, refer to the following:

Bug Fixesedit

The 7.16.1 release includes the following bug fixes.

Dashboard
Fixes invalid filter blank screen #120530
Discover
  • Fixes CSV report for filtered Discover results on dashboards #119583
  • Fixes timefield sorting when switching similar index patterns #116145
Elastic Security
For the Elastic Security 7.16.1 release information, refer to Elastic Security Solution Release Notes.
Fleet
Fixes bug with duplicate Fleet Server inputs in Cloud deployments #119925
Lens & Visualizations
  • Fixes time shift metric lookup #119774
  • Enables normal mode for percentage charts #119582
  • Defaults the point size to 1 and corrects the vislib migrations #118994
  • Adds rtl label support to Elastic Charts #118606
Logs
Respect the Advanced Settings for queries #116485
Machine Learning
  • Ensures validation call is successful in outlier detection wizard #120816
  • Fixes time range charts in anomaly detection job selection #120343
Management
  • Restore support for exclusion in index pattern #120703
  • Fixes the Fix button for ML snapshots in need of upgrade #119745
  • Fixes the Painless script editor scrolling experience #119577
  • Adds support for the Mapper Size Plugin _size parameter in the index template mappings #119365
Metrics
Handle invalid KQL in filterQuery #119557
Monitoring
Separate out mb and legacy index patterns from requests #120841
Osquery
Fixes Pack uploader and View in Lens link #120845
Platform
  • Fixes expressions performance regression #117714
  • Fixes an issue where banners were unintentionally appearing in reports #116147
Reporting
Improves the performance when generating a large CSV export #120365 and #120309
Security
Fixes an issue where Kibana crashed when no authentication providers were enabled #118784
