Kibana 7.17.1edit

Review the following information about the 7.17.1 release.

Known issueedit

Lens & visualizations
A change in the Markdown library that Kibana uses to create TSVB Markdown visualizations and Text dashboard panels renders some tables differently. For more information, check out #767.

For the known issues in the previous releases, refer to the <<known-issue-7.17.0, known issues in 7.17.0>.

Breaking changesedit

There are no breaking changes in the 7.17.1 release.

To review the breaking changes in previous versions, refer to the following:
