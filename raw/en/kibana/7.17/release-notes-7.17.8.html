Kibana 7.17.8edit

Review the following information about the 7.17.8 release.

Breaking changesedit

There are no breaking changes in the 7.17.8 release.

To review the breaking changes in previous versions, refer to the following:

Security updateedit

The 7.17.8 release contains a fix to a potential security vulnerability. For more information, check Security announcements.

Enhancementsedit

Reporting
Upgraded the bundled Chromium browser to version 107, which can cause a minor increase in time to generate PNG and PDF reports #143485
Security
Adds a Content-Security-Policy header to all /api/* responses, so now any Kibana HTTP endpoints that start with /api/* share the same header as the regular Kibana HTTP routes #144902

Bug fixesedit

APM
Fixes the environment filter in the metadata API #144474
Discover
Improves the performance for getFieldsToShow #144672
Machine Learning
Adds maxRetries:0 to relevant API calls for transforms #144603
