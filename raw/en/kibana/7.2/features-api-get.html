Get Featuresedit

This API is experimental and may be changed or removed completely in a future release.

Retrieves all Kibana features.

Note: You cannot access this endpoint via the Console in Kibana.

Get all Kibana featuresedit

Requestedit

To retrieve all features, issue a GET request to the /api/features/v1 endpoint.

GET /api/features/v1

Responseedit

A successful call returns a response code of 200 and a response body containing a JSON representation of the features.
