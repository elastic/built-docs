Config for multiple Kibana instancesedit

If you are using multiple instances of Kibana, you must manually assign at least one Kibana instance as a Code node. Add the following line of code to your kibana.yml file for each Kibana instance you wish to use and restart the instances:

xpack.code.codeNodeUrl: 'http://$YourCodeNodeAddress'

$YourCodeNoteAddress is the URL of your assigned Code node accessible by other Kibana instances.
