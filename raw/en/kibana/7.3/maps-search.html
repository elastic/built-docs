Searching your dataedit

Elastic Maps embeds the query bar for real-time ad hoc search. Only layers requesting data from Elasticsearch are filtered when you submit a search request.

You can create a layer that requests data from Elasticsearch from the following:

  • Vector layer with:

    • Documents source
    • Grid aggregation source
    • Term join
  • Heat map layer with Grid aggregation source
