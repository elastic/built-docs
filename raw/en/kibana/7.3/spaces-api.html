Kibana Spaces APIedit

This API is experimental and may be changed or removed completely in a future release. The underlying Spaces concepts are stable, but the APIs for managing Spaces are currently experimental.

The spaces API allows people to manage their spaces within Kibana.

You cannot access these endpoints via the Console in Kibana.

  • Create Space
  • Update Space
  • Get Space
  • Delete space
