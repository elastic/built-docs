REST APIedit

Some Kibana features are provided via a REST API, which is ideal for creating an integration with Kibana, or automating certain aspects of configuring and deploying Kibana.

Each API is experimental and can include breaking changes in any version of Kibana, or might have been entirely removed from Kibana.

You cannot access the APIs via the Console in Kibana.

APIsedit

  • Features API
  • Kibana Spaces APIs
  • Kibana role management APIs
  • Saved objects APIs
  • Dashboard import and export APIs
  • Logstash configuration management APIs
  • URL shortening API
  • Upgrade assistant APIs
