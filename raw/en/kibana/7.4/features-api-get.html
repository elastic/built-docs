Get features APIedit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will apply best effort to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieves all Kibana features.

Requestedit

GET /api/features/v1

Response codeedit

200
Indicates a successful call.
