Searching your dataedit

Elastic Maps embeds the search bar for real-time search. Only layers requesting data from Elasticsearch are filtered when you submit a search request. Layers narrowed by the search context contain the filter icon to the right of layer name in the legend.

You can create a layer that requests data from Elasticsearch from the following:

  • Vector layer with:

    • Documents source
    • Grid aggregation source
    • Term join. The search context is applied to both the terms join and the vector source when the vector source is provided by Elasticsearch documents.
  • Heat map layer with Grid aggregation source
