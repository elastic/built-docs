Retrieve pipeline APIedit

[experimental] This functionality is experimental and may be changed or removed completely in a future release. Elastic will take a best effort approach to fix any issues, but experimental features are not subject to the support SLA of official GA features. Retrieve a centrally-managed Logstash pipeline.

Requestedit

GET <kibana host>:<port>/api/logstash/pipeline/<id>

Path parametersedit

id
(Required, string) The pipeline ID.

Exampleedit

The API returns the following:

{
  "id": "hello-world",
  "description": "Just a simple pipeline",
  "username": "elastic",
  "pipeline": "input { stdin {} } output { stdout {} }",
  "settings": {
    "queue.type": "persistent"
  }
}
