Delete object APIedit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will apply best effort to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Remove Kibana saved objects.

Once you delete a saved object, it cannot be recovered.

Requestedit

DELETE <kibana host>:<port>/api/saved_objects/<type>/<id>

DELETE <kibana host>:<port>/s/<space_id>/api/saved_objects/<type>/<id>

Path parametersedit

space_id
(Optional, string) An identifier for the space. If space_id is not provided in the URL, the default space is used.
type
(Required, string) Valid options include visualization, dashboard, search, index-pattern, config, and timelion-sheet.
id
(Required, string) The object ID that you want to remove.

Response codeedit

200
Indicates a successful call.

Exampleedit

Delete an index pattern object with the my-pattern ID:

$ curl -X DELETE api/saved_objects/index-pattern/my-pattern
