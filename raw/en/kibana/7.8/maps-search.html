Search geographic dataedit

Maps embeds the search bar for real-time search. Only layers requesting data from Elasticsearch are filtered when you submit a search request. Layers narrowed by the search context contain the filter icon next to the layer name in the legend.

You can create a layer that requests data from Elasticsearch from the following:

  • Vector layer with:

    • Documents
    • Clusters and grid
    • Term join
  • Heat map layer
