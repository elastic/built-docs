Tile layeredit

Tile layers display image tiles served from a tile server.

To add a tile layer to your map, click Add layer, then select one of the following layers:

Configured Tile Map Service
Tile map service configured in kibana.yml. See map.tilemap.url in Configure Kibana for details.
EMS Basemaps
Tile map service from Elastic Maps Service.
Tile Map Service
Tile map service configured in interface.
Web Map Service
Maps from OGC Standard WMS.
