Exploring mobile sessions with Discover

edit

Elastic Mobile APM provides session tracking by attaching a session.id, a guid, to every span and event. This allows for the recall of the activities of a specific user during a specific period of time. The best way recall these data points is using the Discover document explorer. This guide will explain how to do that.
