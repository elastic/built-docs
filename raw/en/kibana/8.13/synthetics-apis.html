Synthetics APIs

edit

The following APIs are available for Synthetics.

  • Get Parameters API to get a parameter(s).
  • Create Parameter API to create a parameter.
  • Edit Parameter API to edit a parameter.
  • Delete Parameter API to delete a parameter.
  • Create Private Location API to create a private location
  • Delete Private Location API to delete a private location
  • Get Private Locations API to get a list of private locations
