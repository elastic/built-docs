Get settings APIedit

Retrieve uptime settings existing settings.

Requestedit

GET <kibana host>:<port>/api/uptime/settings

GET <kibana host>:<port>/s/<space_id>/api/uptime/settings

Prerequisitesedit

You must have read privileges for the uptime feature in Observability section of the Kibana feature privileges.

The API returns the following:
