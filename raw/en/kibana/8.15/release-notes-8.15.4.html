Kibana 8.15.4

edit

The 8.15.4 release includes the following bug fixes.

Bug fixes

edit
Dashboards and visualizations
  • Fixes incomplete string escaping and encoding in TSVB (#196248).
  • Adds scroll margin to panels (#193430).
  • Fixes an issue where label truncation in heat map legends was not working properly in Lens (#195928).
Discover
  • Fixes the width for saved object Type column (#194388).
Elastic Observability solution
  • Changes the slice outcome from bad to good whenever there is no data during the slice window (#196942).
Elastic Search solution
  • Fixes a bug with the Network Drive connector where advanced configuration fields were not displayed for CSV file role mappings with Drive Type: Linux selected (#195567).
Elastic Security solution
For the Elastic Security 8.15.4 release information, refer to Elastic Security Solution Release Notes.
Kibana platform
  • Fixes an issue causing the wrong date to show in the header of a report when generated from a relative date (#197027).
  • Fixes an issue with the export options for PNG/PDF reports in a dashboard (#192530).
Machine Learning
  • Fixes an issue preventing Anomaly swim lane panels from updating on query changes (#195090).
Management
  • Fixes the pagination of the source documents data grid in Transforms (#196119).
