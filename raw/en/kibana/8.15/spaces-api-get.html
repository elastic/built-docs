Get a space APIedit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieve a specified Kibana space.

Requestedit

GET <kibana host>:<port>/api/spaces/space/marketing

Response codeedit

200
Indicates a successful call.

Exampleedit

The API returns the following:

{
  "id": "marketing",
  "name": "Marketing",
  "description" : "This is the Marketing Space",
  "color": "#aabbcc",
  "initials": "MK",
  "disabledFeatures": [],
  "imageUrl": "",
  "solution": "es"
}
