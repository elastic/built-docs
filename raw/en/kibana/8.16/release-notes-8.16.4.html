Kibana 8.16.4

edit

The 8.16.4 release includes the following bug fixes.

Bug fixes

edit
Dashboards and visualizations
  • Fixes an issue with all embeddables getting rebuilt on refresh (#209677).
  • Sets options list control to fetch a maximum of 1,000 terms upon scrolling (#207901).
  • Fixes scroll-jumping when interacting with the Lens editor flyout (#207429).
  • Fixes an issue with Kibana Chrome appearing in generated reports (#206988).
  • Fixes missing highlight theme for HJSON in Vega (#208858).
  • Adds an accessible label to range slider in Lens (#205308).
Discover
  • Fixes document comparison table padding (#205984).
Osquery
  • Increases maximum Osquery timeout to 24 hours (#207276).
Elastic Observability solution
  • Displays No Data in Threshold breached component (#209561).
  • Fixes the preview chart in the Custom threshold rule creation form when the field name has slashes (#209263).
  • Fixes multiple AI assistant flyouts (#209158).
  • Fixes using data view runtime fields during rule execution for the custom threshold rule (#209133).
  • Fixes an issue with running processes missing from the processes table (#209076).
  • Fixes heatmap regression when Inspect flag is turned off (#208726).
  • Fixes missing exception stack trace (#208577).
  • Fixes leading wildcard issue on the Custom threshold alert details page (#206615).
Elastic Security solution
For the Elastic Security 8.16.4 release information, refer to Elastic Security Solution Release Notes.
Kibana security
  • Fixes breaks in YAML parsing by using single quotes in Structured log template (#209736).
  • Adds missing fields to input manifest templates (#208768).
  • Adds missing fields into AWS S3 manifest (#208080).
