Search

edit

The Search space in the Kibana UI contains the following GUI features:

  • Connectors
  • Web crawler
  • Playground
  • Search Applications
  • Behavioral Analytics
  • Inference Endpoints UI
  • AI Assistant for Search
  • Dev Tools Console
