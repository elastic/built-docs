Inference endpoints UI

edit

Inference endpoints streamline the deployment and management of machine learning models in Elasticsearch. The Inference endpoints page in Kibana provides an interface for managing inference endpoints. You can create and manage endpoints either from the Kibana UI or the Inference API.
