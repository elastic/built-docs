Kibana 8.17.9

edit

The 8.17.9 release includes the following fixes.

Fixes

edit
Alerting
  • Fixes an issue causing reports to fail with an invalid header error (#225919).
Dashboards and Visualizations
  • Fixes an issue with dashboard sharing links where copied links were not shortened and some users were unable to copy links in new spaces (#227625).
Data ingestion and Fleet
  • Fixes error on the Agents tab when a tag is added to an agent while the No Tags filter is selected (#225413).
Discover
  • Prevents selected document from changing when resizing the Document flyout with a keyboard (#225594).
Elastic Observability solution
  • Sets environment name length when creating an ML job (#225973).
  • Fixes SLO federated view issue when remote clusters and index name listed exceed 4096 bytes (#224478).
Machine Learning
  • Fixes the handling of time range in embedded anomaly swim lane (#225803).
  • Resets to the default value when the lookbackInterval field is empty (#225249).
