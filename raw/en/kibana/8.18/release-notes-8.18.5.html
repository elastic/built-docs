Kibana 8.18.5

edit

The 8.18.5 release includes the following fixes.

Fixes

edit
Dashboards and Visualizations
  • Fixes an issue with the plus sign + not being properly encoded when it’s part of a date math expression (#230469).
  • Fixes an issue in Firefox preventing you from scrolling in the Lens editor flyout (#228625).
  • Fixes an issue preventing you from saving linked TSVB visualizations when changing data views (#228685).
  • Fixes a console error occurring when adding a Region map visualization to a dashboard (#228669).
Data ingestion and Fleet
  • Fixes an issue that could cause some SSL configurations set using the UI to be overridden (#230758).
  • Fixes an issue where the SSL configuration isn’t properly added to agent policies with pre-configured output types (#230211).
  • Fixes agentless integrations using organization, division, or team data fields being overwritten by package agentless metadata on the agent policy (#230479).
Discover
  • Improves performance of breakdown field search (#229335).
  • Enables the Save query button after making changes (#229053).
  • Fixes the field being picked for fetching stats in the sidebar popover (#228969).
  • Fixes an issue where some Discover Copy to clipboard actions only worked when accessing Kibana in a secure browser context (#229531).
Elastic Observability solution
  • Prevents from performing destructive actions using Elastic AI Assistant (#229497).
  • Fixes an issue with column sorting in the Services Errors table (#229199).
  • Fixes an issue with dependencies infinitely loading when no data exists (#228094).
  • Fixes an error in Uptime occurring when filters are missing in the parameters of a rule created using the API (#229422).
Elastic Security solution
For the Elastic Security 8.18.5 release information, refer to Elastic Security Solution Release Notes.
Kibana platform
  • Fixes an issue with overlapping action icons in the Data views menu when on mobile (#229771).
  • Fixes a display issue with counter indicators on toast messages (#229300).
  • Fixes an issue leading to some features not being visible when updating a space solution view to Classic (#230385).
Machine Learning
  • Removes removeIfExists from the synchronization task scheduler (#228783).
