Delete Logstash pipeline API

edit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Delete a centrally-managed Logstash pipeline.

Request

edit

DELETE <kibana host>:<port>/api/logstash/pipeline/<id>

Path parameters

edit
id
(Required, string) The pipeline ID.

Response code

edit
204 No Content
Indicates a successful call.

Example

edit
$ curl -X DELETE api/logstash/pipeline/hello-world
