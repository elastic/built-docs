Kibana 8.15.2

edit

The 8.15.2 release includes the following enhancements and bug fixes.

Enhancements

edit
Elastic Security solution
For the Elastic Security 8.15.2 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Bumps maximum supported package spec version to 3.2 (#193574).
Kibana security
  • Adds a feature to add support for handling syslogs with unsupported message body (#192817).
  • Automatic Import now performs reproducible sampling from the list of log entries instead of just truncating them (#191598).
  • Adds support for Google Gemini, OpenAI, and Azure OpenAI connectors to Automatic Import (#191577).
  • Displays better error messages for issues with logs sample file upload in Automatic Import (#191310).

Bug fixes

edit
Alerting
  • Fixes error when saving a rule after toggling alerts filter properties on and off (#192522).
Dashboards
  • Fixes map layers that disappear from map panel when using session storage to continue editing a dashboard (#193629).
Discover
  • Fixes "View conflicts" button when a data view ID has special characters (#192374).
Elastic Observability solution
  • Fixes OpenTelemetry agent names (#193134).
  • Resolves an issue for multi-step browser journeys where timings for cached resources within the same step were inaccurate within the waterfall chart (#193089).
  • Updates parsing to skip replacing missing values in Synthetics monitors (#192662).
Elastic Security solution
For the Elastic Security 8.15.2 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Prevents extra agent_status call with empty policyId (#192549).
  • Sets correct title for Restart upgrade agent modal under agent list (#192536).
Kibana security
  • Stops removing message field for unstructured logs when using Automatic Import (#193678).
  • Integrations created using Automatic Import now indicate that they have been developed by the Community instead of Elastic (#193002).
  • Fixes issues with rendering the package manifest in Automatic Import (#192316).
Machine Learning
  • Fixes deletion in Check interval input for anomaly detection rule (#193420).
  • Fixes unnecessary ML services initialization during plugin setup (#193153).
  • Fixes link to anomaly detection wizard from pattern analysis in Discover (#192375).
  • Fixes an issue with the http_endpoint input config loading incorrectly in an Automatic Import workflow (#191964).
