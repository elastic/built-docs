Kibana 8.19.1

edit

The 8.19.1 release includes the following known issues and fixes.

Known issues

edit
Issues with rules occur when xpack.alerting.rules.run.alerts.max is set to a value greater than 5000.

Details

If you’ve set xpack.alerting.rules.run.alerts.max to a value greater than 5000, you will encounter Result window is too large error messages when a maintenance window is active.

Workaround
To mitigate the issue, set xpack.alerting.rules.run.alerts.max to a value equal to or less than 5000.

Reports created in non-default Kibana spaces aren’t shown in the Reporting UI.

Details

After creating a report in a non-default Kibana space, the document exists in the .kibana-reporting index but isn’t shown in the Reporting UI.

Fixes

edit
Dashboards and Visualizations
  • Fixes incorrect panel rendering when the Defer loading panels below "the fold" advanced setting is on (#229662).
  • Fixes an issue in reports where a PNG or PDF export would offset and clip part of the visualization in Lens (#228603).
  • Fixes ES|QL loading button state for long running queries in Lens (#226565).
  • The system color mode now correctly applies the dark theme to charts and their legends when the system is in dark mode (#218417).
  • Fixes a console error occurring when adding a Region map visualization to a dashboard (#228669).
Discover
  • Improves performance of breakdown field search (#229335).
  • Fixes the field being picked for fetching stats in the sidebar popover (#228969).
  • Fixes an issue where HTML search highlighting markup is incorrectly carried over into filter values when using the Filter for value field action in the Log overview tab (#227652).
  • Fixes an issue where some Discover Copy to clipboard actions only worked when accessing Kibana in a secure browser context (#229531).
Elastic Observability solution
  • AI Assistant now works offline, allowing to use the knowledge base UI for cases like air-gapped environments (#229874).
  • Fixes global parameters sync for non-default private locations (#230157).
  • Fixes an issue with column sorting in the Services Errors table (#229199).
  • Fixes an error in Uptime occurring when filters are missing in the parameters of a rule created using the API (#229422).
Elastic Security solution
For the Elastic Security 8.19.1 release information, refer to Elastic Security Solution Release Notes.
Kibana platform
  • Fixes a display issue with counter indicators on toast messages (#229300).
