Kibana 8.19.2

edit

The 8.19.2 release includes the following known issues, enhancements, and fixes.

Known issues

edit
Alerts aren’t generated for rules with alert flapping off and an alert delay higher than 1.

Details

Alerts aren’t generated for rules that have Alert flapping detection turned off and the alert delay set to a value higher than 1.

Workaround
Set the alert delay value to 1 or turn on Alert flapping detection.

Enhancements

edit
Search
  • Bumps default Gemini model for the Gemini Connector in Playground from Gemini 1.5 Pro to Gemini 2.5 Pro (#230457).

Fixes

edit
Alerting
  • Fixes an issue that prevents reports from showing in the Reporting UI when they’re created in a non-default Kibana space (#230876).
Dashboards and Visualizations
  • Fixes screen reader-only header in the Dashboards list (#230470).
Elastic Observability solution
  • Fixes an issue with setupLockManagerIndex where if the first call failed, all subsequent calls would fail as well (#230519).
  • Adds loading state to the Install and Uninstall buttons for Elastic documentation on the AI Assistant Settings page (#229579).
