Get short URL APIedit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will apply best effort to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieve a single Kibana short URL.

Requestedit

GET <kibana host>:<port>/api/short_url/<id>

Path parametersedit

id
(Required, string) The ID of the short URL.

Response codeedit

200
Indicates a successful call.

Exampleedit

Retrieve the short URL with the 12345 ID:

$ curl -X GET api/short_url/12345

The API returns the following:

{
  "id": "12345",
  "slug": "adjective-adjective-noun",
  "locator": {
    "id": "LOCATOR_ID",
    "version": "x.x.x",
    "state": {}
  },
  "accessCount": 0,
  "accessDate": 1632680100000,
  "createDate": 1632680100000
}
