Get all data views APIedit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will apply best effort to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieve a list of all data views.

Requestedit

GET <kibana host>:<port>/api/data_views

GET <kibana host>:<port>/s/<space_id>/api/data_views

Response codeedit

200
Indicates a successful call.

Exampleedit

Retrieve the list of data views:

$ curl -X GET api/data_views

The API returns a list of data views:
