Error reports with Watcheredit

Deprecated in 7.9.0.

Watcher error reports have been removed and replaced with Kibana’s alerting and actions feature. To create error alerts with new tool, select Alerts - Create threshold alert - Error rate.

More information on this new feature is available in Create an alert.
