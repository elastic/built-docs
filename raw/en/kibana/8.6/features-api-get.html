Get features API

edit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieves all Kibana features. Features are used by spaces and security to refine and secure access to Kibana.

Request

edit

GET <kibana host>:<port>/api/features

Response code

edit
200
Indicates a successful call.
