Retrieve pipeline APIedit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will apply best effort to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieve a centrally-managed Logstash pipeline.

Requestedit

GET <kibana host>:<port>/api/logstash/pipeline/<id>

Path parametersedit

id
(Required, string) The pipeline ID.

Exampleedit

The API returns the following:

{
  "id": "hello-world",
  "description": "Just a simple pipeline",
  "username": "elastic",
  "pipeline": "input { stdin {} } output { stdout {} }",
  "settings": {
    "queue.type": "persistent"
  }
}
