Kibana 8.16.3

edit

The 8.16.3 release includes the following bug fixes.

Bug fixes

edit
Alerting
  • Updates handling of invalid RRule params and prevents infinite looping (#205650).
  • Fixes an issue with saving ECS group by fields at the root level for query DSL rules (#203769).
  • Fixes race condition in alerting rules data view selector (#203654).
Data ingestion and Fleet
  • Fixes an issue with showing all agent tags when agent list is filtered (#205163).
Elastic Observability solution
  • Preserves kuery filters when switching between Universal Profiling pages in new solution navigation (#203545).
Elastic Search solution
  • Adds ML as required plugin to Search Assistant (#204009).
  • Fixes web crawler name inconsistencies (#202738).
Elastic Security solution
For the Elastic Security 8.16.3 release information, refer to Elastic Security Solution Release Notes.
Kibana security
  • Fixes Integration and Datastream name validation (#204943).
  • Adds RBAC to the Automatic Import APIs (#203882).
