Kibana 8.17.3

edit

The 8.17.3 release includes the following bug fixes.

Bug fixes

edit
Elastic Observability solution
  • Allows editing of non-resolving log views (#210633).
  • Removes error.id in getErrorGroupMainStatistics queries (#210613).
Elastic Security solution
For the Elastic Security 8.17.3 release information, refer to Elastic Security Solution Release Notes.
Kibana security
  • Fixes the generated name for imported package titles (#210916).
  • Fixes package name validation on the Data streams page (#210770).
  • Handles empty categorization results from LLMs (#210420).
Platform
  • Adds versioning to inventory_view_saved_object (#207007).
Sharing
  • Reinstates switch to support generating public URLs for Embed when supported (#207383).
