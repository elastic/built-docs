Kibana 8.17.4

edit

The 8.17.4 release includes the following fixes.

Fixes

edit
Dashboards and Visualizations
  • Prevents unnecessary re-render when switching between View and Edit modes (#213902).
  • Fixes an issue with the handling of the viewMode check in Lens (#213887).
  • Adds event-annotation-group to saved object privileges for dashboards (#212926).
Elastic Observability Solution
  • Makes Kibana retrieval namespace-specific (#213505).
  • Fixes an error with the retrieve-elastic-doc function being unavailable (#212676).
Elastic Security solution
For the Elastic Security 8.17.4 release information, refer to Elastic Security Solution Release Notes.
Management
  • Adds support for rollup data views that reference aliases (#212592).
