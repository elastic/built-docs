Kibana 8.17.7

edit

The 8.17.7 release includes the following enhancements and fixes.

Enhancements

edit
Alerting
  • Ensures alerts created within Maintenance Windows trigger actions after the window expires (#219797).

Fixes

edit
Dashboards and Visualizations
  • Correctly applies default timezone in TSVB when the timezone parameter is set to default in Advanced Settings (#220658).
  • Removes extra icon from map visualization tooltips (#220134).
Discover
  • Fixes drill-down state not saving in by-value Discover sessions (#219857).
Elastic Observability Solution
  • Considers locations only if the array is not empty in a monitor status rule (#220983).
  • Uses update-by-query for semantic_text migration (#220255).
  • Prevents undefined errors in Transaction flyout (#220224).
  • Fixes broken Span Links flyout in Trace Explorer (#219763).
Elastic Security solution
For the Elastic Security 8.17.7 release information, refer to Elastic Security Solution Release Notes.
Kibana security
  • Reworks cookie and session storage to prevent unexpected logouts for certain users (#220430).
