Delete Logstash pipeline APIedit

[experimental] This functionality is experimental and may be changed or removed completely in a future release. Elastic will take a best effort approach to fix any issues, but experimental features are not subject to the support SLA of official GA features. Delete a centrally-managed Logstash pipeline.

Requestedit

DELETE <kibana host>:<port>/api/logstash/pipeline/<id>

Path parametersedit

id
(Required, string) The pipeline ID.

Response codeedit

204 No Content
Indicates a successful call.

Exampleedit

$ curl -X DELETE api/logstash/pipeline/hello-world
