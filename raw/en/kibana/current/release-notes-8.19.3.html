Kibana 8.19.3

edit

The 8.19.3 release includes the following fixes.

Fixes

edit
Alerting
  • Fixes an issue that prevents reports from showing in the Reporting UI when they’re created in a non-default Kibana space (#230876).
Canvas
  • Fixes a rendering issue that affected progress elements in Canvas (#232432).
Dashboards and Visualizations
  • Ensures passing of the allowLeadingWildcards advanced setting for ES|QL charts in Lens (#232609).
  • Removes the list of availableOptions from the ES|QL VALUES_FROM_QUERY saved object (#231690).
  • Fixes older color mapping configuration in Lens (#231563).
  • Fixes lost references when returning to unsaved dashboards with reference panels (#231517).
  • Fixes screen reader-only header in the Dashboards list (#230470).
Discover
  • Disables sorting for JSON-like fields in ES|QL mode (#231289).
Elastic Security solution
For the Elastic Security 8.19.3 release information, refer to Elastic Security Solution Release Notes.
Kibana platform
  • Fixes an issue where the unusedUrlsCLeanupTask run interval did not update correctly when changed (#231883).
