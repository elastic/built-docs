Spaces settings in Kibanaedit

By default, Spaces is enabled in Kibana, and you can secure Spaces using roles when Security is enabled.

Spaces settingsedit

xpack.spaces.enabled

Set to true (default) to enable Spaces in Kibana. This setting is deprecated. Starting in 8.0, it will not be possible to disable this plugin.

xpack.spaces.maxSpaces

The maximum amount of Spaces that can be used with this instance of Kibana. Some operations in Kibana return all spaces using a single _search from Elasticsearch, so this must be set lower than the index.max_result_window in Elasticsearch. Defaults to 1000.
