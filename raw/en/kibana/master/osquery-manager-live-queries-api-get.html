Get live query API

edit

Get live query APIedit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieves a single live query by ID.

Requestedit

GET <kibana host>:<port>/api/osquery/live_queries/<id>

GET <kibana host>:<port>/s/<space_id>/api/osquery/live_queries/<id>

Path parametersedit

space_id
(Optional, string) The space identifier. When space_id is not provided in the URL, the default space is used.
id
(Required, string) The ID of the live query you want to retrieve.

Response codeedit

200
Indicates a successful call.
404
The specified live query and ID doesn’t exist.

Exampleedit

Retrieve the live query object with the bbe5b070-0c51-11ed-b0f8-ad31b008e832 ID:

$ curl -X GET api/osquery/live_queries/bbe5b070-0c51-11ed-b0f8-ad31b008e832
