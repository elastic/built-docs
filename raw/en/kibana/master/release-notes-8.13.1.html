Kibana 8.13.1edit

The 8.13.1 release includes the following bug fixes.

Enhancementsedit

Fleet
  • Remove index.query.default_field setting from managed component template settings (#178020).

Bug Fixesedit

Alerting
  • Limit useEffect to calculate view in app URL (#179197).
Data Discovery
  • Hide "Save"/"Save as" actions from "Unsaved changes" badge for read-only users (#179132).
Elastic Security
For the Elastic Security 8.13.1 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Use index exists check in fleet-metrics-task (#179404).
Lens & Visualizations
  • Fixes wilcard complex scenarios (#178938).
Machine Learning
  • AIOps: Fix text field candidate selection for log rate analysis (#179699).
Observability
  • Fall back to top 5 docs on scoring error (#179615).
  • Fixing APM data view id (#179257).
