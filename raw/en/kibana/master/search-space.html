Search

edit

The Search space in Kibana comprises the following features:

  • Connectors
  • Web crawler
  • Playground
  • Search Applications
  • Behavioral Analytics
  • Inference Endpoints UI
  • AI Assistant for Search
  • Persistent Dev Tools Console

Docs and release notes

edit

The Search solution and use case is made up of many tools and features across the Elastic Stack. As a result, the release notes for your features of interest might live in different Elastic docs.
