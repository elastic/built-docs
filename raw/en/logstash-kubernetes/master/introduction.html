Introductionedit

This documentation is still in development and may be changed or removed in a future release.

This guide helps you to run and work with Logstash in a Kubernetes environment.

Are you trying out Logstash for the first time? We recommend beginning with our guide Getting Started with Logstash.

If you’re already familiar with Logstash, then it’s time to try it out in Kubernetes. The Getting started with Logstash and Kubernetes demo guides you through the steps of configuring Logstash inside a running Kubernetes cluster.
