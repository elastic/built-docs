compress_leveledit

  • Value type is number
  • Default value is 6

min_flush_timeedit

  • Value type is number
  • Default value is 0

The amount of time in seconds since last flush before a flush is forced, on the next event. Values smaller than 0 disables time based flushing.

spool_sizeedit

  • Value type is number
  • Default value is 50
