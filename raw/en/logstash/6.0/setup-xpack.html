Setting Up X-Pack

X-Pack is an Elastic Stack extension that bundles security, alerting, monitoring, reporting, machine learning, and graph capabilities into one easy-to-install package.

X-Pack also provides a monitoring UI for Logstash. To access this functionality, you must install X-Pack in Logstash.

X-Pack security enables you to secure your Elasticsearch cluster. If other products in the stack are connected to the cluster, they need to be secured as well, or at least communicate with the cluster in a secured way. For more information, see Configuring Security.
