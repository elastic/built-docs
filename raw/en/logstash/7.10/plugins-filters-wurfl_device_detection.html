Wurfl_device_detection filter plugin

  • This plugin was created and is maintained by a contributor.
  • Change log

Installation

You can easily install plugins that are not bundled by default. Run bin/logstash-plugin install logstash-filter-wurfl_device_detection. See Working with plugins for more details.

Description

This plugin uses WURFL Microservice by ScientiaMobile to enrich logs with device information, such as brand and model name, OS name and version, and so forth. You can integrate WURFL Device Detection capabilities into applications for use-cases such as mobile optimization, targeted advertising, Event Streams analysis, and device analytics.
