List your pluginedit

The Logstash Reference is the first place Logstash users look for plugins and documentation. If your plugin meets the quality and acceptance guidelines, we may be able to list it in the guide.

The plugin source and readme will continue to live in your repo, and we will direct users there.

If you would like to have your plugin included in the Logstash Reference:

  • verify that it meets our quality and acceptance guidelines

  • create a new issue in the Logstash repository.

    • Use PluginListing: <yourpluginname> as the title for the issue.
    • Apply the docs label.
    • In the body of the issue, explain the purpose and value your plugin offers, and describe how this plugin adheres to the guidelines.
