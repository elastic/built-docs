Logstash 7.17.4 Release Notesedit

Notable issues fixededit

  • Fixed a bug in retrieving pipeline configuration from central management and license validation which make Logstash process crash if a configured node of the Elasticsearch cluster is down. #12776
  • Fix support for reload of pipelines via SIGHUP #13993
  • Fix stopped pipeline unable to be deleted in registry #14017
  • Fix geoip database download does not respect http_proxy setting #14047
  • Fixed no metrics update issue when PQ is draining #13935
  • Fix a seek problem in the dead letter queue reader when asked to restart consuming from specific point #14093

  • Clarify the definition of matches that depend on previous captures #169
