Descriptionedit

The Kafka Integration Plugin provides integrated plugins for working with the Kafka distributed streaming platform.

  • Kafka Input Plugin
  • Kafka Output Plugin

This plugin uses Kafka Client 2.4. For broker compatibility, see the official Kafka compatibility reference. If the linked compatibility wiki is not up-to-date, please contact Kafka support/community to confirm compatibility.
