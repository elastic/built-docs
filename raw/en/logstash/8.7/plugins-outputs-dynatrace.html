Dynatrace plugin

  • This plugin was created and is maintained by a contributor.
  • Change log

Installation

For plugins not bundled by default, it is easy to install by running bin/logstash-plugin install logstash-output-dynatrace. See Working with plugins for more details.

Description

This plugin sends Logstash events to the Dynatrace Generic log ingest API v2.

Documentation

Documentation for the logstash-output-dynatrace plugin is maintained by the creator.

Getting Help

This is a third-party plugin. For bugs or feature requests, open an issue in the plugins-outputs-dynatrace Github repo.
