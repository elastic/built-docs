Descriptionedit

The AWS Integration Plugin provides integrated plugins for working with Amazon Web Services:

  • Cloudfront Codec Plugin
  • Cloudtrail Codec Plugin
  • Cloudwatch Input Plugin
  • S3 Input Plugin
  • Sqs Input Plugin
  • Cloudwatch Output Plugin
  • S3 Output Plugin
  • Sns Output Plugin
  • Sqs Output Plugin
