API quick referenceedit

All data frame analytics endpoints have the following base:

/_ml/data_frame/analytics

The evaluation API endpoint has the following base:

/_ml/data_frame/_evaluate

All the inference endpoints have the following base:

/_ml/inference/
  • Create data frame analytics jobs
  • Delete data frame analytics jobs
  • Get data frame analytics jobs info
  • Get data frame analytics jobs statistics
  • Start data frame analytics jobs
  • Stop data frame analytics jobs
  • Evaluate data frame analytics
  • Create inference trained model
  • Get inference trained model
  • Get inference trained model statistics
  • Delete inference trained model
