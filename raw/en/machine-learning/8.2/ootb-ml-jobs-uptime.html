Appendix I: Uptime anomaly detection configurationsedit

If you have appropriate Heartbeat data in Elasticsearch, you can enable this anomaly detection job in the Uptime app in Kibana. For more usage information, refer to Inspect uptime duration anomalies.

Uptime: Heartbeatedit

Detect latency issues in heartbeat monitors.

These configurations are available in Kibana only if data exists that matches the recognizer query specified in the manifest file.

Name Description Job Datafeed

high_latency_by_geo
