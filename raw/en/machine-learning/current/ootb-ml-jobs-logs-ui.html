Appendix D: Logs anomaly detection configurationsedit

These anomaly detection jobs appear by default in the Logs app in Kibana. For more information about their usage, refer to Categorize log entries and Inspect log anomalies.

Log analysisedit

Detect anomalies in log entries via the Logs UI.

Name Description Job Datafeed

log_entry_rate

Detects anomalies in the log entry ingestion rate

Log entry categoriesedit

Detect anomalies in count of log entries by category.

Name Description Job Datafeed

log_entry_categories_count

Detects anomalies in count of log entries by category
