AWS ECS Instance metricsedit

CPU Usage
Average of aws.ec2.cpu.total.pct
Inbound Traffic
Average of aws.ec2.network.in.bytes_per_sec
Outbound Traffic
Average of aws.ec2.network.out.bytes_per_sec
Disk Reads (Bytes)
Average of aws.ec2.diskio.read.bytes_per_sec
Disk Writes (Bytes)
Average of aws.ec2.diskio.write.bytes_per_sec

For information about which required fields the Metrics app uses to display EC2 instance metrics, see the metrics field reference.
