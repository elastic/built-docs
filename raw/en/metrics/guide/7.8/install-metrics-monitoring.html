Install Metricsedit

The easiest way to get started with Elastic Logs is by using our hosted Elasticsearch Service on Elastic Cloud. The Elasticsearch Service is available on both AWS and GCP, and automatically configures Elasticsearch and Kibana.

If your data uses nonstandard fields, you may need to modify some default configuration settings.

Hosted Elasticsearch Serviceedit

Skip installing and managing your own Elasticsearch and Kibana instance by using our hosted Elasticsearch Service. Try out the Elasticsearch Service for free.

Install the stack yourselfedit

If you’d rather install the stack yourself, first see the Elastic Support Matrix for information about supported operating systems and product compatibility.

  • Step 1: Install Elasticsearch
  • Step 2: Install Kibana (version 6.5 or later) with a basic license
  • Step 3: Set up and run Metricbeat (version 6.5 or later) on each of the systems you want to monitor
