Amazon Web Services (AWS) monitoring

edit

Monitoring the performance and health of your AWS infrastructure is crucial for maintaining reliable and efficient services. Elastic Observability offers a powerful solution for monitoring and troubleshooting AWS environments, providing deep insights into the performance of your applications, services, and infrastructure components.

Learn how to use the Elastic Observability solution to observe and monitor a broad range of AWS resources and applications. If the service or resource you want to monitor isn’t shown here, go to the Elastic Integrations docs for a full list of supported AWS integrations.
