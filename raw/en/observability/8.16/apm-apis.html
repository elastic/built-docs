APM APIs

edit

There are two kinds of APIs related to Elastic APM:

APM UI API

Kibana APIs specific to working with the Applications UI including updating configuration options, uploading real user monitoring (RUM) source maps, adding annotations, and more.

APM Server API

APIs for working with APM Server. These are mainly intake APIs that accept data from APM agents and are used primarily by APM agent developers.
