Deploy Elastic Agent to send dataedit

Elastic Agent is a single, unified way to add monitoring for logs, metrics, and other types of data to a host. It can also protect hosts from security threats, query data from operating systems, forward data from remote services or hardware, and more.

Prefer to use our legacy Beats for your use case? Refer to Deploy Beats to send data.

To get started with Elastic Agent, refer to:

  • Ingest logs, metrics, and uptime data
  • Ingest traces
  • Ingest data from Splunk (Experimental)
