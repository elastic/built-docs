Step 4: Next stepsedit

If you haven’t already, you can now install APM Agents in your services!

  • Android agent
  • Go agent
  • iOS agent
  • Java agent
  • .NET agent
  • Node.js agent
  • PHP agent
  • Python agent
  • Ruby agent
  • JavaScript Real User Monitoring (RUM) agent

Once you have at least one APM agent sending data to APM Server, you can start visualizing your data in the Kibana APM UI.

If you’re migrating from Jaeger, see Integrate with Jaeger.
