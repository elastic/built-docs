Tutorialsedit

Using Elastic Observability, learn how to analyze log data, monitor system and service metrics, instrument code and collect performance data, and monitor host availability and endpoints.

The Elastic Stack provides a variety of ways to add monitoring data to Elasticsearch. The tutorials in this section focus on Elastic Agent and Beats.

It’s recommended that you use Elastic Agent, when possible, to achieve your monitoring goals. However, if Elastic Agent doesn’t work for your use case, you can use Beats instead.

Not sure which agent to use? Refer to Beats and Elastic Agent capabilities.
