Install the backend

edit

Install the backendedit

To install the Universal Profiling backend, complete the following steps:

  1. Update the Elastic stack to the minimum supported version.
  2. Enable Universal Profiling in Kibana.
  3. Set up Universal Profiling in Kibana.
  4. Run the backend applications.
  5. Next steps.

If you face any issues during installation, refer to Troubleshooting Universal Profiling backend.

After the Universal Profiling installation is complete, refer to Operating the Universal Profiling backend for more on monitoring and scaling the backend.
