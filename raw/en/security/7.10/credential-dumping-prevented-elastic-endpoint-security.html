Credential Dumping - Prevented - Elastic Endpoint Securityedit

Elastic Endpoint Security prevented Credential Dumping. Click the Elastic Endpoint Security icon in the event.module column or the link in the rule.reference column for additional information.

Rule type: query

Rule indices:

Severity: medium

Risk score: 47

Runs every: 10 minutes

Searches indices from: now-15m (Date Math format, see also Additional look-back time)
