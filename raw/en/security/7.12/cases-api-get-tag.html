Get tagsedit

Aggregates and returns all unique tags from all cases.

The Kibana Console supports only Elasticsearch APIs. You cannot interact with the Kibana APIs with the Console and must use curl or another HTTP tool instead. For more information, refer to Console.

Request URLedit

GET <kibana host>:<port>/api/cases/tags

Example requestedit

Gets all tags for all cases:

GET api/cases/tags

Response codeedit

200
Indicates a successful call.

Example responseedit

[
  "windows",
  "phishing",
  "social engineering",
  "bubblegum"
]
