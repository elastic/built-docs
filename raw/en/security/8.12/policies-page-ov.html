Policies

edit

The Policies page lists all of the integration policies configured for Elastic Defend.

Requirements

You must have the Elastic Defend Policy Management privilege to access this feature.

Click on an integration policy’s name to configure its settings. For more information on configuring an integration policy, refer to Configure an integration policy for Elastic Defend.
