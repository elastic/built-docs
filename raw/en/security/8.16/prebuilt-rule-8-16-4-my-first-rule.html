My First Rule

edit

This rule helps you test and practice using alerts with Elastic Security as you get set up. It’s not a sign of threat activity.

Rule type: threshold

Rule indices:

  • auditbeat-*
  • filebeat-*
  • logs-*
  • winlogbeat-*

Severity: low

Risk score: 21

Runs every: 30m

Searches indices from: now-35m (Date Math format, see also Additional look-back time)

Maximum alerts per execution: 1
