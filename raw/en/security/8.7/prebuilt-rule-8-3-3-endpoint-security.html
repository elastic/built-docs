Endpoint Securityedit

Generates a detection alert each time an Elastic Endpoint Security alert is received. Enabling this rule allows you to immediately begin investigating your Endpoint alerts.

Rule type: query

Rule indices:

  • logs-endpoint.alerts-*

Severity: medium

Risk score: 47

Runs every: 5m

Searches indices from: now-10m (Date Math format, see also Additional look-back time)

Maximum alerts per execution: 10000

References: None

Tags:

  • Elastic
  • Endpoint Security

Version: 101

Rule authors:

  • Elastic

Rule license: Elastic License v2

Rule queryedit

event.kind:alert and event.module:(endpoint and not endgame)
