Get started with Elastic Security

edit

This section describes how to set up Elastic Security, install Elastic Agent and the Elastic Defend integration on your hosts, and use the Elastic Security UI in Kibana. To get started, click on one of the following tutorials, depending on your use case:

  • Detect threats in my data with SIEM
  • Secure my hosts with endpoint security
  • Secure my cloud assets with cloud posture management (CSPM)
