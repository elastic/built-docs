Set up connectors for large language models (LLM)edit

This section contains instructions for setting up connectors for LLMs so you can use Elastic AI Assistant and Attack discovery.

Setup guides are available for the following LLM providers:

  • Azure OpenAI
  • Amazon Bedrock
  • OpenAI
  • Google Vertex
  • LM Studio (custom local LLM)
