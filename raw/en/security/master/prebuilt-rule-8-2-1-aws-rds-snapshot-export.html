AWS RDS Snapshot Exportedit

Identifies the export of an Amazon Relational Database Service (RDS) Aurora database snapshot.

Rule type: query

Rule indices:

  • logs-aws*

Severity: low

Risk score: 21

Runs every: 10m

Searches indices from: now-60m (Date Math format, see also Additional look-back time)
