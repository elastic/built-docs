Developer tools

edit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features.

A number of developer tools are available in your project’s UI under the Dev Tools section.

  • Console: Make API calls to your Elasticsearch instance using the Query DSL and view the responses.
  • Search Profiler: Inspect and analyze your search queries to identify performance bottlenecks.
  • Grok Debugger: Build and debug grok patterns before you use them in your data processing pipelines.
