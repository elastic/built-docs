Management settings

edit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features.

Go to Project settings, then Management to manage your indices, data views, saved objects, settings, and more. You can also open Management by using the global search field.

Access to individual features is governed by Elastic user roles. Consult your administrator if you do not have the appropriate access. To learn more about roles, refer to Assign user roles and privileges.

Feature Description Available in

API keys

Create and manage keys that can send requests on behalf of users.

Asset criticality

Bulk assign asset criticality to multiple entities by importing a text file.

Connectors

Create and manage reusable connectors for triggering actions.

Data views

Manage the fields in the data views that retrieve your data from Elasticsearch Serverless.

Entity Risk Score

Manage entity risk scoring, and preview risky entities.

Files

Manage files that are stored in Kibana.

Index management

View index settings, mappings, and statistics and perform operations on indices.

Ingest pipelines

Create and manage ingest pipelines that parse, transform, and enrich your data.

Logstash pipelines

Create and manage Logstash pipelines that parse, transform, and enrich your data.

Machine learning

View, export, and import your anomaly detection and data frame analytics jobs and trained models.

Maintenance windows

Suppress rule notifications for scheduled periods of time.

Maps

Create maps from your geographical data.

Reports

Manage and download reports such as CSV files generated from saved searches.

Rules

Create and manage rules that generate alerts.

Saved objects

Copy, edit, delete, import, and export your saved objects. These include dashboards, visualizations, maps, data views, and more.

Spaces

Organize your project and objects into multiple spaces.

Tags

Create, manage, and assign tags to your saved objects.

Transforms

Use transforms to pivot existing Elasticsearch indices into summarized or entity-centric indices.
