[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features.

This content applies to:

Several Elastic Cloud Serverless features let you upload files. For example, you can add files to cases or upload a logo to an Image panel in a dashboard.

You can access these uploaded files in Project settings → Management → Files.
