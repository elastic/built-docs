Enable cloud security features

edit

To use cloud security features in your Elastic Security project, you must have the Cloud Protection Essentials or Cloud Protection Complete options enabled for your project.

To enable these options or check their current status:

  1. Click your project name in the upper-left corner of Kibana. Select Manage project.

  2. To the right of Project features, select Edit.

  3. Enable the necessary options, then click Save.

Continue with cloud security setup.
