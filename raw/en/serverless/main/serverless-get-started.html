Get started

edit

There are two options to create serverless projects:

  • If you are a new user, sign up for a free 14-day trial to create a serverless project. For more information about the Elastic Cloud trials, check Trial features.
  • If you are an existing customer, log in to Elastic Cloud. On the home page, you will see a new option to create serverless projects.

Choose the type of project that matches your needs and we’ll help you get started with our solution guides.
