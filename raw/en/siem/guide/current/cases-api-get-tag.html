Get tags

edit

Aggregates and returns all unique tags from all cases.

Request URL

edit

GET <kibana host>:<port>/api/cases/tags

Example request

edit

Gets all tags for all cases:

GET api/cases/tags

Response code

edit
200
Indicates a successful call.

Example response

edit
[
  "windows",
  "phishing",
  "social engineering",
  "bubblegum"
]
