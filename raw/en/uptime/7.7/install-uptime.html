Install Uptime

edit

Install Uptimeedit

The easiest way to get started with Elastic Uptime is by using our hosted Elasticsearch Service on Elastic Cloud. The Elasticsearch Service is available on both AWS and GCP, and automatically configures Elasticsearch and Kibana.

Hosted Elasticsearch Serviceedit

Skip managing your own Elasticsearch and Kibana instance by using our hosted Elasticsearch Service on Elastic Cloud.

Try out the Elasticsearch Service for free, then jump straight to Step 3: Install and configure Heartbeat.

Install the stack yourselfedit

If you’d rather install the stack yourself, first see the Elastic Support Matrix for information about supported operating systems and product compatibility.
