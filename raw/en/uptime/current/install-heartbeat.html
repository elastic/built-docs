Step 3: Install and configure Heartbeat

edit

Uptime requires the setup of monitors in Heartbeat. These monitors provide the data you’ll be visualizing in the Uptime app.

For instructions on installing and configuring Heartbeat, see the Setup Instructions in Kibana. Additional information is available in Heartbeat quick start.
