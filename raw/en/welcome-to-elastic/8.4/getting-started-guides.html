Getting started guidesedit

These guides help you to get up and running quickly with Elastic products and solutions. In just a few steps, you can ingest and analyze your data in a new Elastic Cloud deployment.

  • Monitor applications and systems with Elastic Observability
  • Build a custom search engine experience with Elastic Enterprise Search
  • Use Elastic Security for SIEM
  • Protect hosts with endpoint threat intelligence from Elastic Security
  • Deploy your own platform to store, search, and visualize any data
