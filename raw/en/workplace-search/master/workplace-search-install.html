Installation

edit

Workplace Search requires three Elastic services: Elasticsearch, Kibana, and Enterprise Search.

Choose one of the following options to deploy these services:

Elastic Cloud
Visit https://cloud.elastic.co, log in or create an account, and then create a new deployment. By default, a new deployment includes the required services. (A free trial is available for new accounts.)
Docker
See Run Enterprise Search server using Docker images.
Downloads
See Run Enterprise Search server using downloads (packages).

After deploying the required services, continue to Getting started.
