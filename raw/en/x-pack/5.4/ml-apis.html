Machine Learning APIsedit

You can use APIs to perform the following machine learning activities:

Datafeedsedit

  • Create datafeed, Delete datafeed
  • Start datafeed, Stop datafeed
  • Get datafeed info, Get datafeed statistics
  • Preview datafeed
  • Update datafeed

Jobsedit

  • Validate detectors, Validate job
  • Create job, Delete job
  • Open job, Close job
  • Get job info, Get job statistics
  • Flush job
  • Post data to job
  • Update job

Model Snapshotsedit

  • Delete model snapshot
  • Get model snapshot info
  • Revert model snapshot
  • Update model snapshot

Resultsedit

  • Get buckets
  • Get categories
  • Get influencers
  • Get records
