License Managementedit

When you initially install X-Pack, a 30 day license is generated that allows access to X-Pack features. By default, this is a trial license that allows access to all features — the equivalent of a platinum license. If you prefer, you can configure the cluster to generate a basic license at startup. If you choose the basic license, you can later activate a trial license by using an API. At the end of the 30 day period, you can purchase a subscription to keep using the full functionality of the X-Pack components.

When your license expires, X-Pack operates in a degraded mode. For more information, see License Expiration.
