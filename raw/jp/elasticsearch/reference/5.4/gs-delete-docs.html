ドキュメントの削除

edit

ドキュメントの削除はとても簡単です。次の例は、IDが2の前のcustomerを削除する方法を示しています。

DELETE /customer/external/2?pretty

特定のクエリに一致するドキュメントをすべて削除するには、​_delete_by_query API​を参照してください。 Delete By Query APIを使用してすべてのドキュメントを削除するよりも、インデックス全体を削除する方がはるかに効率的であることに注意してください。
