문서 삭제edit

문서 삭제는 매우 간단합니다. 이 예는 앞서 만든 ID가 2인 문서를 삭제하는 방법을 보여줍니다.

DELETE /customer/external/2?pretty

특정 쿼리와 일치하는 모든 문서를 삭제하려면 _delete_by_query API의 내용을 참조하십시오. Delete By Query API를 사용하여 모든 문서를 삭제하기보다는 아예 색인을 삭제하는 것이 훨씬 더 효율적입니다.
