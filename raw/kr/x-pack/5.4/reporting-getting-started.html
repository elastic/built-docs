시작하기edit

Kibana에 install X-Pack할 때 X-Pack reporting이 자동으로 활성화됩니다.

수동으로 보고서를 생성하려면 다음을 수행합니다.

  1. 웹 브라우저에서 Kibana를 열고 로그인합니다. 로컬에서 Kibana를 실행하는 경우 http://localhost:5601`로 이동합니다. Kibana에 액세스하고 보고서를 생성하려면 `kibana_user 및 reporting_user 역할이 필요합니다.
  2. 보고서에 포함시킬 대시보드, 시각화 또는 저장된 검색을 엽니다.

  3. Kibana 툴바에서 보고 를 클릭합니다.

  4. *인쇄 가능 PDF*를 클릭합니다.

스크립트에서 또는 Watcher를 사용하여 자동으로 보고서를 생성하려면 표시된 생성 URL을 사용합니다. 자세한 내용은 보고서 생성 자동화를 참조하십시오.
