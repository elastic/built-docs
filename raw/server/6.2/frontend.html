Frontend supportedit

Frontend support is an experimental feature not intended for production usage at the moment.

This section describes features specifically designed to support frontend application monitoring and a guide on how to enable experimental frontend support.

Enable Frontend Supportedit

To try out experimental frontend support, set the apm-server.frontend.enabled to true. See apm-server.reference.yml for more configuration options.
