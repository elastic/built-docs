Dashboards

edit

APM Server comes packaged with example Kibana dashboards, visualizations, and searches for visualizing APM Server data in Kibana. Before you can use the dashboards, you need to create the index pattern, apm-*, and load the dashboards into Kibana. To do this, you can either run the setup command (as described here) or configure dashboard loading in the apm-server.yml config file.

This requires a Kibana endpoint configuration. If you didn’t already configure a Kibana endpoint, see configured APM Server

Make sure Kibana is running before you perform this step. If you are accessing a secured Kibana instance, make sure you’ve configured credentials as described in Configuration file.

To set up the Kibana dashboards for APM Server, use the appropriate command for your system.

Open a PowerShell prompt as an Administrator (right-click the PowerShell icon and select Run As Administrator). If you are running Windows XP, you may need to download and install PowerShell.

From the PowerShell prompt, change to the directory where you installed APM Server, and run:

PS > apm-server setup --dashboards
