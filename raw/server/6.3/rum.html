Real User Monitoring (RUM)edit

Real user monitoring is an experimental feature not intended for production usage at the moment.

By supporting real user monitoring (RUM). Elastic APM allows to collect performance data about your frontend JavaScript applications by setting up the JavaScript Frontend Agent.

You can make use of source maps when using Elastic RUM.
